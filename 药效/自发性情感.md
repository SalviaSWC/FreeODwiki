

# 自发性情感 (Simultaneous emotions)

[◀返回](./home.md)

|   |   |
|---|---|
| [![](https://psychonautwiki.org/w/thumb.php?f=Plutchik-wheel.svg&width=300)](/文件/Plutchik-wheel.png) |  [罗伯特·普拉奇克的情绪轮](http://en.wikipedia.org/wiki/Contrasting_and_categorization_of_emotions#Plutchik.27s_wheel_of_emotions) |

**自发性情感 (Simultaneous emotions)** 定义为在没有明显外部诱因的情况下同时感觉到多种情绪的体验。例如，在这种状态下，用户可能会突然感到强烈的冲突情绪，如同时感到快乐、悲伤、爱、恨等。这可能导致一种精神状态，即用户可能感受到任何可能的冲突情绪组合。

自发性情感通常伴随着其他并发效应，如[记忆抑制](/药效/记忆抑制.md "Memory suppression")和[情绪增强](/药效/情绪强化.md "Emotion intensification")。它们最常在[严重](/文档/药物剂量分类.md#heavy "Dosage") [剂量](/文档/给药剂量.md "Dosage")的[迷幻](/文档/药物分类/迷幻剂.md "Psychedelic")化合物影响下被诱发，例如[LSD](/药物/LSD.md "LSD")、[赛洛西宾](/药物/赛洛西宾蘑菇.md "Psilocybin")和[麦斯卡林](/药物/麦斯卡林.md "Mescaline")。
