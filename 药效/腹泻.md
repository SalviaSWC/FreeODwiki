# 腹泻

[◀返回](./home.md)

| [![Nhs-stool-chart.gif](../文件/Nhs-stool-chart.gif)](../文件/Nhs-stool-chart.gif) |
| :---: |
| 类型 6 - 7 的便便可以归类为腹泻哦。 |

**腹泻**（拉肚子）就是每天至少拉三次稀粑粑或者水状的粑粑哦。它通常会持续几天，并且因为体液流失可能导致[脱水](./脱水.md)。随着情况变严重，可能会发展成排尿减少、肤色苍白、心率加快和反应能力下降呢。在[精神活性物质](../文档/精神探索.md)使用的语境下，已知某些化合物会引起腹泻，或者至少会增加发生腹泻的可能性喵。[\[1\]](#cite_note-1)[\[2\]](#cite_note-2)[\[3\]](#cite_note-3) 这种情况不像因感染引起的腹泻那么危险，因为它只会在人还处于药物影响下的时候持续存在呢。

腹泻通常伴随着其他并发效应，比如[恶心](./恶心.md)和[脱水](./脱水.md)。它最常在使用了[严重](../文档/给药剂量.md#heavy) [剂量](../文档/给药剂量.md) 的某些[致幻剂](../文档/药物分类/致幻剂.md)化合物后出现，比如[死藤水](../药物/死藤水.md)、[麦斯卡林](../药物/麦斯卡林.md)和[赛洛西宾蘑菇](../药物/赛洛西宾蘑菇.md)。不过嘛，在使用某些[兴奋剂](../文档/药物分类/兴奋剂.md)、[莫达菲尼](../药物/莫达菲尼.md)和[咖啡因](../药物/咖啡因.md)时也可能发生哦。
