# 谵妄

**谵妄** (Delirium) (也称为**急性[困惑](/药效/困惑.md "Confusion")**)[\[1\]](#cite_note-SendelbachGuthrie2009-1) 在医学上被认定为一种意识的生理紊乱，伴随着基准认知的改变，而这种改变不能用先前存在的或正在演变的神经认知障碍来更好地解释哦。[\[2\]](#cite_note-ICD-11-Delirium-2) 意识的紊乱表现为[引导、集中、维持和转移注意力的能力降低](/药效/专注力抑制.md "Focus suppression")，并且伴随的认知改变可能涉及至少另一个领域，包括[记忆和学习 (特别是近期记忆)](/药效/记忆抑制.md "Memory suppression")、定向障碍 (特别是对[时间](/药效/时间扭曲.md "Time distortion") 和 [地点](/药效/空间定向障碍.md "Spatial disorientation") 的定向障碍)、[语言能力的改变](/药效/语言能力抑制.md "Language depression")，或者[感知扭曲](/药效/视觉扭曲.md "Visual distortions") 或 [知觉\-运动障碍](/药效/身体能力改变.md "Physical alterations")。伴随谵妄的感知障碍包括[误解](/药效/周边信息误判.md "Peripheral information misinterpretation")、[错觉](/药效/内部幻觉.md "Internal hallucination")或[幻觉](/药效/外部幻觉.md "External hallucination")；这些障碍通常是视觉上的，但也可能出现在其他感官模式中，范围从简单单一到高度复杂。患有谵妄的人可能还会表现出情绪障碍，比如[焦虑](/药效/焦虑.md "Anxiety")、恐惧、[抑郁](/药效/抑郁.md "Depression")、[易怒](/药效/易怒.md "Irritability")、愤怒、[欣快](/药效/认知欣快.md "Cognitive euphoria")和冷漠，并且会在一种情绪状态和另一种之间快速且不可预测地切换呢。[\[3\]](#cite_note-DSM5Delirium-3)

这种紊乱在很短的时间内（通常是几小时到几天）发展起来，并且往往在一天中波动，通常在晚上和夜间外部定向刺激减少时恶化。有人提出，睡眠\-觉醒周期的紊乱是谵妄的一个核心标准。正常的注意力/唤醒、谵妄和昏迷处于一个连续体上，昏迷被定义为对语言刺激缺乏任何反应。[\[3\]](#cite_note-DSM5Delirium-3)

谵妄可能以三种不同的形式出现。在科学文献中，这些被称为活跃型、低活跃型或混合型。[\[4\]](#cite_note-FongTulebaev2009-4) 在活跃型中，表现为严重的困惑和定向障碍，发作突然且强度波动。在低活跃型（即活跃不足）中，表现为同样突然地从与外界的互动中退缩，伴随嗜睡和普遍的不活动等症状。[\[5\]](#cite_note-HoskerWard2017-5) 谵妄也可能以混合型出现，患者可能在过度活跃和低活跃时期之间波动。

谵妄最常见于服用了[严重](/文档/药物剂量分类.md#heavy "Dosage") [剂量](/文档/给药剂量.md "Dosage")的[谵妄剂](/文档/药物分类/谵妄剂.md "Deliriant")化合物的时候，比如[苯海拉明](/药物/苯海拉明.md "DPH")[\[6\]](#cite_note-Serio2004-6)、[曼陀罗](/药物/曼陀罗.md "Datura")[\[7\]](#cite_note-7) 和 [苄达明](/药物/苄达明.md "Benzydamine")。不过呢，它也可能由极其广泛的健康问题引起，比如尿路感染、[\[8\]](#cite_note-BalogunPhilbrick2014-8) 流感、[\[9\]](#cite_note-ManjunathaMath2011-9) 和阿尔茨海默病。[\[10\]](#cite_note-10)
