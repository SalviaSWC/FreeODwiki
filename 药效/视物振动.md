# 视物振动

| | |
|---|---|
| ![](https://psychonautwiki.org/w/thumb.php?f=Vibrating_vision_2.gif&width=200) | 不由自主的眼球运动 |

**视物振动**（Vibrating vision），也称为**眼球震颤**（nystagmus），可以描述为体验到持续、快速且不由自主的眼球运动，眼睛会快速地左右移动，以至于视线开始振动和模糊。这会严重损害视力，导致无法正常行动或执行需要视力的基本任务，看东西都在抖呢。

视物振动通常伴随着其他并发效应，并被其增强，比如[刺激](/药效/刺激.md) 和[思维加速](/药效/思维加速.md)。它最常发生在使用[严重](/文档/给药剂量.md) [剂量](/文档/给药剂量.md) 的[兴奋剂](/文档/药物分类/兴奋剂.md) 化合物时，比如[MDMA](/药物/MDMA.md)、[苯丙胺](/药物/苯丙胺.md) 和[4-FA](/药物/4-FA.md)。
