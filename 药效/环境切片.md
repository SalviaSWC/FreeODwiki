# 环境切片

[//gfycat.com/ifr/FondEnlightenedKitty](//gfycat.com/ifr/FondEnlightenedKitty)

**环境切片** 作者 **[StingrayZ](https://www.reddit.com/user/StingrayZ)** - 这张图片精准地描绘了被称为环境切片的常见体验效果哦。

**环境切片** 的定义是体验者的视野似乎分裂成独立且切口整齐的部分，这些单独的切片随后会慢慢飘离原来的位置，然后消失并重置为正常状态。这种效应通常是自发发生的，很少持续超过几秒钟呢。

这些切片的组织方式可能非常多样；它们可以简单到只有三个分开的部分，也可以非常复杂，形成诸如多个互锁的移动螺旋切片，或者无限多样的其他几何设计。

环境切片经常伴随着其他并发效应，比如 [环境立体主义](./环境立体主义.md) 和 [视觉断连](./视觉断连.md)。它最常见于使用 [中等](./药物剂量分类.md#common) [剂量](./药物剂量分类.md) 的 [解离剂](./药物分类/解离剂.md) 类化合物，比如 [氯胺酮](./药物/氯胺酮.md)、[PCP](./药物/PCP.md) 和 [右美沙芬](./药物/右美沙芬.md) 时发生。不过呢，在使用 [迷幻剂](./药物分类/迷幻剂.md)[\[1\]](#cite_note-PapoutsisNikolaou2014-1) 如 [LSD](./药物/LSD.md)、[赛洛西宾](./药物/赛洛西宾蘑菇.md) 和 [麦斯卡林](./药物/麦斯卡林.md) 时，这种现象也会以较轻的程度出现。
