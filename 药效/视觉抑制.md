# 视觉抑制

[◀返回](./home.md)

**视觉抑制**指的是任何会降低一个人通过视觉感知外部环境能力的[主观效应](/文档/药物分类/主观效应索引.md "主观效应索引")哦。

这个页面列出并描述了在某些[精神活性物质](/文档/药物分类/药物全索引.md "药物全索引")影响下可能出现的各种视觉抑制现象呢。

### 颜色抑制

主条目：[颜色抑制](/药效/颜色抑制.md "颜色抑制")

**颜色抑制**（也就是大家说的**后天性色盲**[\[1\]](#cite_note-APADictionary-acquired-color-blindness-1)）指的是颜色变得暗淡、饱和度降低，而且彼此之间变得难以区分的体验。在这个过程中，红色可能看起来“没那么红”，绿色“没那么绿”，比起平时清醒的时候，所有的颜色都会变得更灰暗、更模糊不清呢。在程度比较严重的时候，这种效应甚至会让外面的世界变成黑白的、单色的，完全没有色彩了哦。

颜色抑制通常会伴随着其他并发效应，比如[视觉锐度抑制](/药效/视觉锐度抑制.md "视觉锐度抑制")和[复视](/药效/复视.md "复视")。这种情况最常发生在使用[严重](/文档/药物剂量分类.md#heavy "严重") [剂量](/文档/给药剂量.md "给药剂量")的[抗精神病药](/文档/抗精神病药.md "抗精神病药")[\[2\]](#cite_note-RichaYazbek2010-2)类药物时，比如[喹硫平](/药物/喹硫平.md "喹硫平")、[氟哌啶醇](/药物/氟哌啶醇.md "氟哌啶醇")和[利培酮](/药物/利培酮.md "利培酮")。

### 复视

主条目：[复视](/药效/复视.md "复视")

**复视**（医学上叫**diplopia**）定义为每只眼睛都把同一个物体看成是分开的图像的体验[\[3\]](#cite_note-APADictionary-double-vision-3)[\[4\]](#cite_note-Marsh1979-4)，就像大家斗鸡眼时看到的那样。根据强度的不同，这种效应可能会让大家看不清东西，没法做一些基本的需要用眼的事情呢。

只要闭上一只眼睛，这个效应通常就会很容易消失哦。这说明复视可能是因为大脑把两只眼睛接收到的数据错误地叠加在了一起，而没有像平时那样把它们整合成一个单一的三维图像呢。

这种效应通常有以下三个不同的强度等级：

1. **轻微** - 在最低等级，复视是很轻微的，虽然能感觉到它的存在，但大部分时候可以忽略不计。这种程度还不至于让人看不清细节，大家还是可以读读书或者过马路的。
2. **明显** - 到了这个等级，复视就变得比较强烈了，想要看清细节（比如阅读）会变得非常困难。不过，如果是看大的环境轮廓，睁着两只眼睛还是能勉强看清的。
3. **强烈** - 在最高等级，复视变得超级严重，大家已经没法准确地看清周围环境的大大小小的细节了。这时候想要像平时一样正常活动，就必须得一直闭着一只眼睛才行呢。

复视经常会伴随着其他并发效应，比如[视觉锐度抑制](/药效/视觉锐度抑制.md "视觉锐度抑制")和[模式识别抑制](/药效/模式识别抑制.md "模式识别抑制")。这种效应最常发生在使用[中等](/文档/药物剂量分类.md#common "中等") [剂量](/文档/给药剂量.md "给药剂量")的[抑制剂](/文档/药物分类/抑制剂.md "抑制剂")和[解离剂](/文档/药物分类/解离剂.md "解离剂")类药物时，比如[酒精](/药物/酒精.md "酒精")、[喹硫平](/药物/喹硫平.md "喹硫平")、[氯胺酮](/药物/氯胺酮.md "氯胺酮")和[右美沙芬](/药物/右美沙芬.md "右美沙芬")。[\[5\]](#cite_note-B.Sc.Buncic2018-5) 不过嘛，它也可能在其他很多种类的药物影响下偶尔出现，比如[致幻剂](/文档/药物分类/致幻剂.md "致幻剂")、[兴奋剂](/文档/药物分类/兴奋剂.md "兴奋剂")、[抗胆碱能药物](/文档/药物分类/谵妄剂.md "谵妄剂")、[SSRI类药物](/文档/药物分类/SSRI.md "SSRI类药物")、[阿片类药物](/文档/药物分类/阿片类药物.md "阿片类药物")、[GABA类药物](/文档/药物分类/GABA类药物.md "GABA类药物")和[大麻类](/文档/药物分类/大麻类.md "大麻类")物质。

### 模式识别抑制

主条目：[模式识别抑制](/药效/模式识别抑制.md "模式识别抑制")

| | |
|---|---|
| [![Pattern_recognition_suppression.jpg](/文件/Pattern_recognition_suppression.jpg)](/文件/Pattern_recognition_suppression.jpg) | 中风伴随的视觉扭曲模拟图。图中的一切都感觉很熟悉，但却很难叫出名字或定义它。药物引起的模式识别抑制感觉起来可能很像哦。 |

**模式识别抑制**指的是在心理上处理和解释视觉信息的能力部分或完全丧失，无论你看得有多清楚。比如说，如果一个人看着面前的一个物体，即使能清晰地看到细节，识别这个物体的能力也会下降。这会让最普通的日常物品都变得认不出来，尤其是人脸哦。经历这种效应的人看着一张脸时，可能看得清五官，也能描述出来，但就是没法把这些视觉信息组合起来认出这张脸是谁呢。值得注意的是，这种效应和被称为视觉统觉性失认症的视觉障碍是可以相提并论的，而且可能有关联哦。[\[6\]](#cite_note-BehrmannNishimura2010-6)

模式识别抑制经常会伴随着其他并发效应，比如[分析抑制](/药效/分析抑制.md "分析抑制")和[思维减慢](/药效/思维减慢.md "思维减慢")。这种情况最常发生在使用[严重](/文档/药物剂量分类.md#heavy "严重") [剂量](/文档/给药剂量.md "给药剂量")的[解离剂](/文档/药物分类/解离剂.md "解离剂")或[抗精神病药](/文档/抗精神病药.md "抗精神病药")类药物时，比如[氯胺酮](/药物/氯胺酮.md "氯胺酮")、[喹硫平](/药物/喹硫平.md "喹硫平")、[PCP](/药物/PCP.md "PCP")和[右美沙芬](/药物/右美沙芬.md "右美沙芬")。不过，在使用极高剂量的[迷幻剂](/文档/药物分类/迷幻剂.md "迷幻剂")类药物（比如[LSD](/药物/LSD.md "LSD")、[赛洛西宾](/药物/赛洛西宾蘑菇.md "赛洛西宾")和[麦斯卡林](/药物/麦斯卡林.md "麦斯卡林")）时，也可能出现程度较轻的这种现象呢。

### 周边信息误判

主条目：[周边信息误判](/药效/周边信息误判.md "周边信息误判")

**周边信息误判**指的是一种短暂的体验，你会错误地解读和看到视野边缘（余光）里的物体或细节。在这种状态下，一个人可能会短暂地在余光中看到很复杂的细节，但仔细一看，其实根本不存在。举个栗子，一个人可能会瞬间觉得余光里有物体、人或者事情发生了，但那些其实都不在那儿。一旦意识到这个细节是错的，误判就会被正确的感知覆盖掉。通常来说，只要大家有意识地注意到那个被误判的视觉细节，它就会立刻变回正确的视觉解释啦。

周边信息误判通常会伴随着其他并发效应，比如[模式识别增强](/药效/模式识别增强.md "模式识别增强")和[外部幻觉](/药效/外部幻觉.md "外部幻觉")，而且会被它们加强哦。这种情况最常发生在使用[中等](/文档/药物剂量分类.md#common "中等") [剂量](/文档/给药剂量.md "给药剂量")的[谵妄剂](/文档/药物分类/谵妄剂.md "谵妄剂")类药物时，比如[苯海拉明](/药物/苯海拉明.md "苯海拉明")、[曼陀罗](/药物/曼陀罗.md "曼陀罗")和[苄达明](/药物/苄达明.md "苄达明")。不过，在[兴奋剂精神病](/文档/兴奋剂精神病.md "兴奋剂精神病")和[睡眠剥夺](/文档/睡眠剥夺.md "睡眠剥夺")的影响下也可能发生呢。

### 视觉锐度抑制

主条目：[视觉锐度抑制](/药效/视觉锐度抑制.md "视觉锐度抑制")

| | |
|---|---|
| [![London_acuity_suppression.jpg](/文件/London_acuity_suppression.jpg)](/文件/London_acuity_suppression.jpg) | *模糊的伦敦 作者：[Josikins](/关于本站/关于本站.md "User:Josikins")* - 这张图片准确地模拟了在伦敦市中心五十周年纪念步行桥上看到的视觉锐度抑制效果哦。 |

**视觉锐度抑制**定义为视觉清晰度和锐度的下降，导致视野部分或完全变得模糊不清。[\[5\]](#cite_note-B.Sc.Buncic2018-5) 这种效应可能会影响整个视野，也可能只影响一部分。体验这种锐度抑制就像透过一个失焦的镜头看世界，会让大家看不清外面环境的细节呢。根据强度的不同，这通常会导致大家没法做一些基本的需要用眼的事情哦。

视觉锐度抑制经常会伴随着其他并发效应，比如[复视](/药效/复视.md "复视")和[模式识别抑制](/药效/模式识别抑制.md "模式识别抑制")。这种效应最常发生在使用[中等](/文档/药物剂量分类.md#common "中等") [剂量](/文档/给药剂量.md "给药剂量")的[抑制剂](/文档/药物分类/抑制剂.md "抑制剂")和[解离剂](/文档/药物分类/解离剂.md "解离剂")类药物时，比如[酒精](/药物/酒精.md "酒精")[\[7\]](#cite_note-KunchuliaPilz2012-7)、[喹硫平](/药物/喹硫平.md "喹硫平")、[氯胺酮](/药物/氯胺酮.md "氯胺酮")和[右美沙芬](/药物/右美沙芬.md "右美沙芬")。

### 视觉加工减慢

主条目：[视觉加工减慢](/药效/视觉加工减慢.md "视觉加工减慢")

**视觉加工减慢**指的是感觉视觉信息处理的速度变慢了。[\[8\]](#cite_note-8) 在这种效应的影响下，大家可能会觉得自己的视觉像是在卡顿，表现得像正在缓冲的视频、定格动画、胶片条、电脑显示器刷新或者频闪灯一样。在强度更高的时候，这可能会导致一个人的视觉暂时完全停止移动，就像冻结了一样呢。值得一提的是，这种效应和被称为运动盲或运动失认症（akinetopsia）的视觉障碍是可以比较的，但不一定有直接关系哦。[\[9\]](#cite_note-Zeki1991-9) 这种效应体验起来可能是平滑且一致的减速，也可能是粗糙且不规则的呢。

帧率抑制经常会伴随着其他并发效应，比如[视觉锐度抑制](/药效/视觉锐度抑制.md "视觉锐度抑制")和[复视](/药效/复视.md "复视")。这种效应最常发生在使用[中等](/文档/药物剂量分类.md#common "中等") [剂量](/文档/给药剂量.md "给药剂量")的[解离剂](/文档/药物分类/解离剂.md "解离剂")类药物时，比如[氯胺酮](/药物/氯胺酮.md "氯胺酮")、[MXE](/药物/MXE.md "MXE")、[PCP](/药物/PCP.md "PCP")和[右美沙芬](/药物/右美沙芬.md "右美沙芬")。
