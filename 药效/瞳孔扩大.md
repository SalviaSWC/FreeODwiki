# 瞳孔扩大

| ![Diapupils](/文件/Diapupils.jpg) |
| :---: |
| 扩大的瞳孔图片示例 |

**瞳孔扩大**（也称为**瞳孔散大**）可以被描述为在正常光照条件下，一个人的瞳孔尺寸变大。通常，瞳孔尺寸在黑暗中会增大，在光亮处会缩小，然而，扩张的瞳孔即使在明亮的环境中也会保持过大呢。

瞳孔扩大最常见于服用[中等](/文档/给药剂量.md#中等)[剂量](/文档/给药剂量.md)的各种[血清素能](/文档/血清素.md)化合物时，如[迷幻剂](/文档/药物分类/迷幻剂.md)、[解离剂](/文档/药物分类/解离剂.md)、[谵妄剂](/文档/药物分类/谵妄剂.md)、[共情剂](/文档/药物分类/共情剂.md)、各种[兴奋剂](/文档/药物分类/兴奋剂.md)[\[1\]](#cite_note-1) 以及一些[抗抑郁药](/文档/药物分类/抑制剂.md)。
