# 流涕

| | |
|---|---|
| ![](https://psychonautwiki.org/w/thumb.php?f=Runny_nose.jpg&width=200) | 这是一个流鼻涕的孩子哦 |

**流涕**（Runny nose）可以描述为鼻腔充满大量粘液（通常称为“鼻涕”）的情况。这在健康人中相对经常发生，是过敏或某些疾病（如普通感冒或花粉热）的常见症状。它也可能是哭泣和暴露在寒冷温度下的副作用呢。

流涕通常伴随着其他并发效应，比如[过度打哈欠](/药效/过度打哈欠.md)、[唾液分泌增加](/药效/唾液分泌增加.md)、[痰液增多](/药效/痰液增多.md) 和[流泪](/药效/流泪.md)。它最常发生在使用[严重](/文档/给药剂量.md) [剂量](/文档/给药剂量.md) 的[色胺类物质](/文档/药物分类/色胺类物质.md) [迷幻剂](/文档/药物分类/迷幻剂.md) 化合物时，比如[赛洛西宾](/药物/赛洛西宾蘑菇.md)、[4-AcO-DMT](/药物/4-AcO-DMT.md) 和[4-HO-MET](/药物/4-HO-MET.md)。不过嘛，在使用[阿片类药物](/文档/药物分类/阿片类药物.md) [戒断反应](/文档/药物戒断反应.md)[\[1\]](#cite_note-1) 以及作为[可卡因](/药物/可卡因.md) 滥用的症状时，也可能出现这种情况[\[2\]](#cite_note-2)。
