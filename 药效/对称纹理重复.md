# 对称纹理重复

[![](/文件/Grass_by_Chelsea_Morgan.jpg)](/文件/Grass_by_Chelsea_Morgan.jpg)

**草地** 作者 **[Chelsea Morgan](/wiki/User:TheReplicator "User:TheReplicator")** - 这张图片精准地描绘了这种被称为对称纹理重复的常见 [迷幻](./药物分类/迷幻剂.md) 效应呢。

**对称纹理重复** 的定义是感知到纹理在其自身的表面上以复杂且对称的方式反复镜像，并且在整个过程中保持一致。无论观察者试图多近距离地观察这种扭曲，它都保持着相同的细节水平，而且往往在周边视觉中最为明显。它通常出现在粗糙的纹理中，比如草地、地毯、树皮和沥青哦。

在这种状态下，如果盯着一个固定的点看，对称纹理重复可能会逐渐增强，并进一步镶嵌成更复杂的形状。但是嘛，一旦视线转移或重新审视，这种复杂性的增加通常会重置回基线。

对称纹理重复经常伴随着其他并发效应，比如 [模式识别增强](./模式识别增强.md)[\[1\]](#cite_note-PapoutsisNikolaou2014-1)[\[2\]](#cite_note-BersaniCorazza2014-2) 和 [变形](./变形.md)，这种方式可能导致在对称性中嵌入大量的抽象形式和图像。它最常见于使用 [轻微](./药物剂量分类.md#mild) [剂量](./药物剂量分类.md) 的 [迷幻剂](./药物分类/迷幻剂.md) 类化合物，比如 [LSD](./药物/LSD.md)、[赛洛西宾](./药物/赛洛西宾蘑菇.md) 和 [麦斯卡林](./药物/麦斯卡林.md) 时发生。不过呢，在某些 [兴奋剂](./药物分类/兴奋剂.md) 和 [解离剂](./药物分类/解离剂.md) 的影响下，比如 [MDMA](./药物/MDMA.md) 或 [3-MeO-PCP](./药物/3-MeO-PCP.md)，也可能比较少见地发生。

### 分析

[//gfycat.com/ifr/DizzyInexperiencedEuropeanpolecat](//gfycat.com/ifr/DizzyInexperiencedEuropeanpolecat)

这个动画演示了如何通过简单地将粗糙纹理对称地镜像来准确地重现对称纹理重复哦。

值得注意的是，要使用图像编辑软件准确复制这种效果的外观，只需要简单地将纹理在自身上进行对称镜像就可以了。这可能表明大脑内部也发生着类似的过程，也许是通过从中心视线获取视觉数据，并将其以导致纹理对称镶嵌的方式镜像到周边视觉，来填充缺失的细节呢。
