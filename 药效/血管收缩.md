
# 血管收缩

[◀返回](./home.md)

| ![](/文件/Vasodilation_and_vasoconstriction.png) | 该图展示了血管舒张、血管收缩和正常状态下静脉结构的对比差异喵。 |
| :--- | :--- |

**血管收缩**可以被描述为静脉和血管的变窄，这是由于它们的肌壁收缩造成的呢。这种情况在大动脉和小动脉中尤为常见哦。

这种效应通常会导致人的手臂和腿部产生紧绷感、酸痛感和麻木感。它的影响程度可以从轻微到极其不适不等呢。

血管收缩通常伴随着其他同时发生的效应，比如[刺激](/药效/刺激.md)。它最常见于在[中等](/文档/药物剂量分类.md) [剂量](/文档/给药剂量.md) 的[致幻](/文档/药物分类/致幻剂.md) [兴奋](/文档/药物分类/兴奋剂.md)类物质影响下诱发的，例如 [LSD](/药物/LSD.md)、[2C-E](/药物/2C-E.md) 和 [DOC](/药物/DOC.md) 喵。不过，在传统 [兴奋剂](/文档/药物分类/兴奋剂.md)（如 [甲基苯丙胺](/药物/甲基苯丙胺.md)、[咖啡因](/药物/咖啡因.md) 和 [MDMA](/药物/MDMA.md)）的影响下也可能发生哦。
