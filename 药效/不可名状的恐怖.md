# 不可名状的恐怖 (Unspeakable horrors)

|  | [说明](/wiki/Property:Caption "Property:Caption") |
| --- | --- |
| [![National Park by HR Giger.jpeg](/文件/National_Park_by_HR_Giger.png)](/文件/National_Park_by_HR_Giger.png) | **国家公园** 作者：**[H.R Giger](https://www.erowid.org/culture/characters/giger_hr/giger_hr.shtml)** |

**不可名状的恐怖**被定义为长时间暴露在那种无法描述的场景和可怕、令人不安的幻觉内容中的体验，而且这些内容通常直接受个人恐惧的影响呢。这可能发生在大剂量使用[致幻剂](/文档/药物分类/致幻剂.md)期间，特别是当使用者正经历着负面情绪压力和个人内心的纠结时哦。

虽然构成这些状态的内容通常是难以言喻的，并且很大程度上取决于体验者自身的恐惧，但某些主题和原型经常会出现。这些包括但不限于以下几种喵：

* **本质上邪恶且具威胁性的[自主实体](/药效/自主实体.md)** - 这可能包括恶魔、畸形的怪物、戴着兜帽的身影、嘲讽的实体，或者是因重伤或疾病而变得畸形的普通人类。


* **本质上邪恶且具威胁性的[几何](/药效/几何.md)** - 上面提到的幻觉状态通常由标准的致幻[几何](/药效/几何.md)构成，或者嵌在其中。这通常包含着那种纯粹的概念性恐惧、恐怖和痛苦的几何表现形式，让人看一眼就能从所有感官上真切地感受到，是不是很可怕？


* **本质上邪恶且具威胁性的[机械景观](/药效/机械景观.md)** - 这可以被描述为与[机械景观](/药效/机械景观.md)在功能上完全一样的体验。不同的是，它的构成部分通常几乎完全由难以描述的复杂、连锁、不断变化的生物机械血肉、器官、内脏和身体部位组成。


* **本质上邪恶且具威胁性的[场景和情节](/药效/场景和情节.md)** - 这可能包括对自己或他人施加痛苦的场景（比如被折磨），或者自己作为猎物被“邪恶”生物或力量追捕的场景，以及直接体验个人的恐惧。


* **本质上邪恶且具威胁性的[设置、布景和风景](/药效/设置、布景和风景.md)** - 这可能包括造访地狱般的风景、古老的巨大的洛夫克拉夫特式建筑和毁灭的文明遗迹。


* **担心自己的理智崩塌** - 这可以描述为一种感觉，觉得当前的感知太可怕了，以至于自己肯定会彻底疯掉，留下严重的心理创伤。这种效应会让人们立刻产生一种感觉，觉得日常生活只是覆盖在一个相对可怕的现实之上的薄薄外壳，而且无法忽视或逃避。不过值得注意的是，虽然压力山大，但这种效应很少真的给人留下持久的心理问题哦，所以不要太担心啦。


对这种成分的体验以及经历者如何解读它，似乎因人而异。虽然大多数人如果在毫无准备的情况下遇到这种状态会受到惊吓，但也有很多人发现，虽然很恐怖，但它同样令人兴奋，而且能锻炼性格呢。为了确保不会在不知不觉中陷入这种状态，如果没有事先练习，应该避免使用[严重](/文档/药物剂量分类.md)剂量的[致幻剂](/文档/药物分类/致幻剂.md)，而且应该随着自己感觉舒适的程度，慢慢地、小幅度地从低剂量往上加量哦。

不可名状的恐怖通常会伴随其他效应一起出现，比如[焦虑](/药效/焦虑.md)、[精神病](/药效/精神病发作.md)和[记忆抑制](/药效/记忆抑制.md)。它们最常在使用了[严重](/文档/药物剂量分类.md)剂量的[迷幻](/文档/药物分类/迷幻剂.md)化合物后被诱发，比如[LSD](/药物/LSD.md)、[赛洛西宾](/药物/赛洛西宾蘑菇.md)和[2C-P](/药物/2C-P.md)。使用[谵妄剂](/文档/药物分类/谵妄剂.md)如[苯海拉明](/药物/苯海拉明.md)、[曼陀罗](/药物/曼陀罗.md)和[苯扎明](/药物/苯扎明.md)时也可能发生这种情况。在使用[解离剂](/文档/药物分类/解离剂.md)如[右美沙芬](/药物/右美沙芬.md)和[PCP](/药物/PCP.md)时，也有可能体验到不可名状的恐怖，特别是当它们与其他[迷幻](/文档/药物分类/迷幻剂.md)物质（如[LSD](/药物/LSD.md)）或[谵妄剂](/文档/药物分类/谵妄剂.md)物质（如[苯海拉明](/药物/苯海拉明.md)）[药物联用](/文档/危险药物联用.md)时更是如此呢。
