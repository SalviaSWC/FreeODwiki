# 流泪

| | |
|---|---|
| ![](https://psychonautwiki.org/w/thumb.php?f=Water_eyes.jpg&width=200) | 这是一个流泪的图片示例哦 |

**流泪**（Watery eyes）可以描述为一种身体效应，结果就是眼睛里的泪管持续、不由自主地流出泪水、哭泣。体验到这种效应的人常常会觉得自己在毫无理由地哭泣，尽管当时并没有平时这种状态下该有的相关情绪，有点奇怪呢。

流泪通常伴随着其他并发效应，比如[过度打哈欠](/药效/过度打哈欠.md) 和[流涕](/药效/流涕.md)。它最常发生在使用[中等](/文档/给药剂量.md) [剂量](/文档/给药剂量.md) 的[迷幻](/文档/药物分类/迷幻剂.md) [色胺类物质](/文档/药物分类/色胺类物质.md) 化合物时，比如[赛洛西宾](/药物/赛洛西宾蘑菇.md)、[4-AcO-DMT](/药物/4-AcO-DMT.md) 和[4-HO-MET](/药物/4-HO-MET.md)。
