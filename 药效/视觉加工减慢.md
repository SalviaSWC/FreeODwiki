# 视觉加工减慢 (Visual processing deceleration)

**视觉加工减慢**定义为感知到的视觉信息处理速度降低哦。[1] 在这种效应的影响下，人们可能会觉得视觉有延迟，表现得就像缓冲中的视频、定格动画、胶卷、电脑显示器或频闪灯一样呢。在强度较高的情况下，它会导致一个人的视觉暂时完全停止移动，就像被冻结了一样。值得注意的是，这种效应与被称为运动盲（akinetopsia）的视觉障碍相当，但不一定相关哦。[2] 这种效应可以被体验为平滑且一致的视觉处理减速，也可能是粗糙且不规则的嘛。

帧率抑制通常伴随着其他并发效应，例如[视觉锐度抑制](/药效/视觉锐度抑制.md)和[复视](/药效/复视.md)。它最常在[中等](/文档/给药剂量.md#中等)剂量的[解离剂](/文档/药物分类/解离剂.md)类物质影响下诱发，例如[氯胺酮](/药物/氯胺酮.md)、[MXE](/药物/MXE.md)、[PCP](/药物/PCP.md)和[右美沙芬](/药物/右美沙芬.md)呢。
