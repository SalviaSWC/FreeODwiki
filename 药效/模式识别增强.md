# 模式识别增强

**模式识别增强**（Pattern recognition enhancement）被定义为一个人在模糊刺激中识别模式的能力和倾向的增加哦。

| 图片示例 | 描述 |
| --- | --- |
| ![](/文件/Faceincloud.jpg) | **云中的脸**，作者：[Neil Usher](http://www.wired.co.uk/news/archive/2012-10/14/pareidolic-robot) - 这张图片是模式识别增强的一个典型例子呢。 |

看到像人脸一样的图案是人类与生俱来的能力，在日常生活中非常普遍，科学文献里管这叫“幻想性视错觉”（pareidolia）喵。常见的例子包括在车头看到“脸”，或者在云朵里看到各种奇奇怪怪的东西呢。

在这种效应下，图案会比平时清醒状态下要显著得多哦。比如，风景里可能会出现细节惊人的图像，日常物品看起来可能像张脸，云朵也可能变成梦幻的物体，而这一切都不需要真正的视觉改变就会发生呢。一旦在某个物体或景观中感知到了图像，大脑可能会通过一种叫[变形](/药效/变形.md)的[幻觉状态](/药效/幻觉状态.md)来进一步夸大这种识别，这就不只是幻想性视错觉了，而是变成了标准的视觉幻觉喵。

模式识别增强通常伴随着其他效应，比如[视觉锐度增强](/药效/视觉锐度增强.md)和[颜色增强](/药效/颜色增强.md)呢。它最常见于服用[轻微](/文档/给药剂量.md#轻微)[剂量](/文档/给药剂量.md)的[致幻剂](/文档/药物分类/致幻剂.md)化合物，比如[LSD](/药物/LSD.md)、[赛洛西宾](/药物/赛洛西宾蘑菇.md)和[麦斯卡林](/药物/麦斯卡林.md)哦。
