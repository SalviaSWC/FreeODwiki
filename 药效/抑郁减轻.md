# 抑郁减轻 (Depression reduction)

[◀返回](./home.md)

**抑郁减轻**是指将与[抑郁](/药效/抑郁.md)和低落情绪状态相关的症状降到最低的体验。它与像[认知欣快](/药效/认知欣快.md)这样的效应不同哦，因为它不仅仅是提升用户的情绪，而是让人产生一种稳定的情感幸福感呢。

抑郁减轻最常在摄入充足营养时出现。[\[1\]](#cite_note-BenderHagan2017-1)[\[2\]](#cite_note-Spedding2014-2)[\[3\]](#cite_note-SubletteEllis2011-3)[\[4\]](#cite_note-HuangWang2016-4)[\[5\]](#cite_note-NgKoh2017-5) 严重的抑郁症可以用传统的抗抑郁药有效缓解；虽然在轻度到中度抑郁症中，SSRI和三环类抗抑郁药（平均而言）似乎要么只有微小的帮助，要么完全无效。[\[6\]](#cite_note-FournierDeRubeis2010-6) 不过，在睾酮替代疗法[\[7\]](#cite_note-ZarroufArtz2009-7)[\[8\]](#cite_note-8)（年轻男性中雌激素增加与抑郁增加有关，无论BMI如何）[\[9\]](#cite_note-9)和莫达非尼的影响下，也可能出现抑郁减轻的情况哦。[\[10\]](#cite_note-RockRoiser2013-10)

### 情感正常 (Euthymia)

**情感正常**（语义上与[心境恶劣](/药效/抑郁.md)相对）是一种持久且自我维持的稳定情感幸福体验。[\[11\]](#cite_note-FavaBech2015-11) 这种状态的特点是：

* 在之前临床诊断为情绪障碍的患者中，没有出现情绪紊乱；如果经历了悲伤/[焦虑](/药效/焦虑.md)/[易怒](/药效/易怒.md)，它们也是短暂的，不会显著影响日常生活。
* 感到快乐、平静、活跃，并且对事物感兴趣。
* 拥有认知灵活性。
* 睡眠能让人精神焕发或恢复体力。
* 拥有一种统一的人生观，指导行动和感受来塑造未来。
* 能够抵抗压力（具有韧性以及对焦虑或挫折的耐受力）。

这不太可能是一个孤立的效应成分，而更可能是合适的环境与其他的效应共同作用的结果，比如[复苏](/药效/复苏.md)、[内省](/药效/内省增强.md)、[个人偏见抑制](/药效/个人偏见抑制.md)和[灵性增强](/药效/灵性增强.md)。它也可能源于物质药理作用导致的直接神经变化呢。

情感正常最常在各种不同物质的影响下以不同程度的功效出现，主要是致幻剂结合心理治疗，[\[12\]](#cite_note-Galv.C3.A3o-CoelhoMarx2021-12)[\[13\]](#cite_note-LuomaChwyl2020-13) 或者是解离剂。[\[14\]](#cite_note-RyanMarta2014-14) 不过，它也可能在规定的精神科药物治疗过程中以及在某些共情剂的影响下出现哦。

### 分析

#### 即时缓解

*安排活动*是一种针对抑郁症的行为治疗，它似乎与其他心理或药物治疗一样有效呢。[\[15\]](#cite_note-Cuijpersvan_Straten2007-15) 情绪与参与的愉快活动数量之间存在显著关系。抑郁的人觉得愉快的活动较少，参与愉快活动的频率较低，获得的积极强化也较少。

*运动*是一种基于证据的抑郁症治疗方法。[\[16\]](#cite_note-CraftLanders1998-16)[\[17\]](#cite_note-KvamKleppe2016-17)[\[18\]](#cite_note-SchuchVancampfort2016-18)[\[19\]](#cite_note-CramerLauche2013-19)[\[20\]](#cite_note-JosefssonLindwall2014-20) 它还有一个优点是可以防止药物相互作用风险的增加。任何类型的中度到重度运动（包括瑜伽）似乎都同样有效，并且与其他纯粹的抑郁症治疗形式（如心理治疗或药物治疗）没有显著差异。身体素质的大幅改变似乎也并非必要呢。

#### 长期

认知行为疗法（CBT）是治疗成人抑郁症的有效方法。[\[21\]](#cite_note-21) 对于轻度到中度抑郁症，主要的心理疗法之间在疗效上没有巨大差异。[\[22\]](#cite_note-Cuijpersvan_Straten2008-22) 也有大量证据支持计算机化护理和心理教育干预。[\[23\]](#cite_note-Cuijpers1997-23)[\[24\]](#cite_note-24)[\[25\]](#cite_note-RichardsRichardson2012-25)[\[26\]](#cite_note-SpekCuijpers2006-26)[\[27\]](#cite_note-AnderssonCuijpers2009-27)

对于自然环境中的轻度或中度抑郁症患者，抗抑郁药的复发率很高。行为疗法可能在长期内防止复发哦。[\[28\]](#cite_note-28)

#### 致幻剂心理治疗

2021年的一项荟萃分析检查了12项关于经典致幻剂治疗抑郁症状的双盲随机对照试验。[\[12\]](#cite_note-Galv.C3.A3o-CoelhoMarx2021-12) 单次剂量*结合心理治疗方案*，能中度到大幅度地减少负面情绪症状。这种减少在3小时到60天的检查时间内保持一致；只是没有足够的随机临床试验来延长它们的时间框架。这种效应在健康和患病的患者中都存在呢。

2020年的一项荟萃分析检查了9项关于致幻剂心理治疗的安慰剂对照研究。[\[13\]](#cite_note-LuomaChwyl2020-13) 这些研究使用了1到3次剂量*结合心理治疗方案*，并在6个月的过程中多次测量结果。作者的效应量表明，随机选择的接受致幻剂心理治疗的患者比接受安慰剂的患者表现更好的概率为80%。此外，没有发生严重的副作用事件。作者甚至将他们的结果与其他检查多种心理健康问题“金标准”治疗的荟萃分析进行了比较：致幻剂心理治疗的表现大大胜出呢。

使用1-3剂经典致幻剂的心理治疗在产生抑郁减轻方面大大优于目前的药理学标准，同时只有极少或没有副作用。相比于每日服用药物会产生更大的副作用且直到2-4周才产生预期效果，在6个月内只需要单次剂量的优势显而易见嘛。

#### 氯胺酮及其异构体

| ![Dialog-warning.png](/文件/Dialog-warning.png) |
| :---: |
| **氯胺酮和艾氯胺酮都可能增加膀胱损伤哦。** |

[氯胺酮](/药物/氯胺酮.md)在典型的抗抑郁剂量下具有很强的滥用潜力。[\[29\]](#cite_note-KokaneArmant2020-29)[\[30\]](#cite_note-BozymskiCrouse2019-30) 氯胺酮有严重的膀胱和肝脏损伤报告病例。艾氯胺酮（Esketamine），一种较新的氯胺酮鼻喷雾剂型，没有任何报告病例，据称具有更好的安全性。然而，在最近的短期临床试验中，艾氯胺酮与安慰剂相比，不良膀胱事件的数量仍然增加了一倍以上（6-10% vs 1-4%）。虽然2/3的艾氯胺酮事件要么无需干预自行解决，要么通过降低剂量解决，但任何生理损伤都是急性和直接的：在典型的剂量方案中，无法达到稳态浓度。

氯胺酮为30-50%的患者提供（剂量依赖性的）大量即时抑郁减轻；其效应量在第7天变为中等到小。[\[31\]](#cite_note-XuHackett2016-31)[\[32\]](#cite_note-LeeDella_Selva2015-32)[\[33\]](#cite_note-PennybakerNiciu2017-33) 每周到每两周的给药维持了在第28天测量的统计学上显著的抑郁减轻，并且重复给药已导致情感正常的案例。[\[14\]](#cite_note-RyanMarta2014-14)[\[30\]](#cite_note-BozymskiCrouse2019-30) 直系亲属中有酒精使用障碍家族史与抗抑郁反应的改善有关，并减少了不良精神效应，如[认知不快](/药效/认知不快.md)。它的抗抑郁特性也可能更广泛地源于解离剂的[新奇感增强](/药效/新奇感增强.md)和/或[沉浸感强化](/药效/沉浸感强化.md)。
