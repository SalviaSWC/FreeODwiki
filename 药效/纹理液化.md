# 纹理液化

[![](https://psychonautwiki.org/w/thumb.php?f=Textureliquidation.jpg&width=250)](/文件/Textureliquidation.jpg)

> **向日葵的液化** by reddit user **/u/sincitysaint** - 这张图展示了向日葵发生纹理液化时的样子哦。

**纹理液化**是指物体和景色的纹理、形状和整体结构变得越来越简化、模糊和风格化，就像整个外部环境开始变成一幅画或者卡通片一样呢，喵！

纹理液化通常会伴随着其他的效应一起出现，比如[视觉锐度增强](/药效/视觉锐度增强.md)和[漂移](/药效/漂移.md)。这种效果最常在使用了中等剂量的[致幻剂](/文档/药物分类/致幻剂.md)类化合物时被诱发，比如[LSD](/药物/LSD.md)、[赛洛西宾](/药物/赛洛西宾蘑菇.md)和[麦斯卡林](/药物/麦斯卡林.md)哦。
