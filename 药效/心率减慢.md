# 心率减慢

| ![Sinus_bradycardia_lead2](/文件/Sinus_bradycardia_lead2.png) |
| :---: |
| 这张图片显示了在导联 II 中看到的窦性心动过缓，心率约为 50 BPM。 |

**心率减慢**或**心动过缓**可以被描述为心率低于休息时的正常心率。健康的成年人心脏在休息时通常每分钟跳动 60 到 100 次。当心率波动到 60 BPM 以下的水平时，就被描述为心动过缓或心率异常缓慢哦。

值得注意的是，心率减慢通常是心理症状的结果，作为对放松、[焦虑抑制](/药效/焦虑抑制.md)、[镇静](/药效/镇静.md)和[正念](/药效/正念.md)的自然反应呢。

心率减慢最常见于服用[严重](/文档/给药剂量.md#严重)[剂量](/文档/给药剂量.md)的[抑制剂](/文档/药物分类/抑制剂.md)类化合物时，如[GABA类物质](/文档/药物分类/抑制剂.md)和[阿片类药物](/文档/药物分类/阿片类药物.md)。然而，在服用[大麻类](/文档/药物分类/大麻类.md)物质、[解离剂](/文档/药物分类/解离剂.md)和[兴奋剂](/文档/药物分类/兴奋剂.md)时，也可能发生这种情况呢。
