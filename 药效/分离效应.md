# 分离效应

**分离效应**是指任何让人感觉脱离外部环境、感官和意识的[主观效应](/药效/主观效应索引.md)哦。

这些效应通常与[解离剂](/文档/药物分类/解离剂.md)[致幻剂](/文档/药物分类/致幻剂.md)有关，这可能是因为这些化合物作为[NMDA受体拮抗剂](/文档/药物分类/NMDA受体拮抗剂类药物.md)起作用的方式呢。这意味着它们结合到受体上，但不激活它，并阻止其他[神经递质](/文档/神经递质.md)这样做。结果就是大脑中电信号传递的剂量依赖性减少，以及神经元的整体断连，这导致了大脑意识部分与其感觉器官之间的断连状态嘛。

本页面列出并描述了在某些[精神活性物质](/文档/药物分类/药物全索引.md)影响下可能发生的各种分离效应。

### 认知分离

主条目：[认知分离](/药效/认知分离.md)

**认知分离**是指感觉与自己的身份感、思维流和一般认知过程疏远和脱离的体验呢。这会导致诸如[人格解体](/药效/人格解体.md)、[现实感丧失](/药效/现实感丧失.md)、[记忆抑制](/药效/记忆抑制.md)、“[自我解体](/文档/药物分类/自我解体.md)”（Ego death）以及一系列一般的[认知抑制](/药效/认知抑制.md)状态。这种体验还会对一个人对自己意识的感知产生广泛的主观变化哦。下面的列表描述并记录了这些变化：

* 感觉自己的意识思维流和记忆不是自己的
* 感觉自己的意识思维过程遥远而模糊
* 感觉自己的意识思维过程在结构或行为上变得自主和机械化
* 感觉认知能力的整体速度、连通性和分析能力下降

认知分离通常伴随着其他同时发生的效应，如[视觉分离](/药效/视觉分离.md)和[触觉分离](/药效/触觉分离.md)呢。这导致了一种感觉，即一个人正在部分或完全脱离他们的感官输入和意识机能。这是在[中等](/文档/药物剂量分类.md)[剂量](/文档/给药剂量.md)的[解离剂](/文档/药物分类/解离剂.md)化合物，如[氯胺酮](/药物/氯胺酮.md)、[PCP](/药物/PCP.md)和[右美沙芬](/药物/右美沙芬.md)影响下几乎普遍存在的效应哦。

### 既视感

主条目：[既视感](/药效/既视感.md)

**既视感**（Déjà Vu）被定义为对当前体验与未定义的过去有任何突然的不恰当的熟悉感。[\[1\]](#cite_note-OConnor2021-1)[\[2\]](#cite_note-2)[\[3\]](#cite_note-3)[\[4\]](#cite_note-4) 它的两个关键组成部分是强烈的熟悉感，以及确定当前时刻是新奇的。[\[5\]](#cite_note-5) 这个术语来自法语，字面意思是“已经见过”。这是一种有据可查的现象，在清醒生活中和在[致幻剂](/文档/药物分类/致幻剂.md)影响下都可能发生呢。

在[精神活性物质](/文档/药物分类/药物全索引.md)使用的背景下，许多化合物通常能够诱发自发的、通常是长时间的轻微到强烈的既视感。这可以让一个人产生一种压倒性的“以前来过这里”的感觉嘛。这种感觉通常还伴随着对当前位置或环境、正在进行的身体动作、整个情况或物质本身效应的熟悉感。

尽管在体验过程中，人们理性上知道“以前”体验的情况（以前的体验何时、何地以及如何发生）是不确定的或被认为是不可能的，但这种效应还是经常被触发呢。

既视感通常伴随着其他同时发生的效应，如[嗅觉幻觉](/药效/嗅觉幻觉.md)和[现实感丧失](/药效/现实感丧失.md)。[\[6\]](#cite_note-6) 它最常在[中等](/文档/药物剂量分类.md)[剂量](/文档/给药剂量.md)的[致幻剂](/文档/药物分类/致幻剂.md)化合物影响下诱发，[\[7\]](#cite_note-7) 如[迷幻剂](/文档/药物分类/迷幻剂.md)、[\[8\]](#cite_note-8) [大麻类](/文档/药物分类/大麻类.md)、[\[9\]](#cite_note-9) 和[解离剂](/文档/药物分类/解离剂.md)。

### 分离层级

主条目：[分离层级](/药效/分离层级.md)

这篇文章并不代表单独的主观效应成分，而是作为一个通用的分级系统，可以应用于同时体验到的[分离效应](/药效/分离效应.md)，如[认知](/药效/认知分离.md)、[躯体](/药效/躯体分离.md)和[视觉分离](/药效/视觉分离.md)哦。这些效应最常在[轻微](/文档/药物剂量分类.md)到[严重](/文档/药物剂量分类.md)[剂量](/文档/给药剂量.md)的[解离剂](/文档/药物分类/解离剂.md)化合物，如[氯胺酮](/药物/氯胺酮.md)、[PCP](/药物/PCP.md)和[右美沙芬](/药物/右美沙芬.md)影响下诱发。

这种感官和认知分离的过程可以分解为以下5个基本的强度级别：

1.  **与环境的微妙分离** - 在最低级别，效应可以描述为感官的微妙迟钝和与外部环境的主观分离的模糊感。
2.  **与环境的部分分离** - 在这个级别，环境开始感觉在物理距离上更远，并且与人的感官知觉越来越断开连接。[视觉模糊](/药效/视觉锐度抑制.md)和[复视](/药效/复视.md)开始出现，同时[类似麻醉的效应和触觉麻木](/药效/触觉抑制.md)开始发生。此时，[运动控制、协调和平衡](/药效/运动控制丧失.md)受到与剂量成比例的抑制。在声音方面，[听觉](/药效/听觉锐度抑制.md)也似乎变得模糊、低沉和遥远呢。
3.  **与环境的完全分离** - 在这个级别，感官和认知的断连已经完全阻断了一个人对外部环境的感知。正是在这里，人们发现自己正在经历一种灵魂出窍的体验，因为他们进入了一个[解离洞](/药效/视觉分离.md#洞、空间和虚空)，这通常伴随着[内部幻觉](/药效/内部幻觉.md)。
4.  **与自我的分离** - 在最高级别，解离和分离变得足够强烈，以至于一个人的长期记忆停止运作。这导致了一种通常被称为“[自我解体](/文档/药物分类/自我解体.md)”的状态，因为人们失去了回忆和理解自己身份感的能力嘛。
5.  **与意识的分离** - 当一个人失去所有感官意识和正常认知功能时，就会发生最高级别的分离。这使人失去知觉，并始终在人的记忆中留下巨大的空白，伴随着大段的[失忆](/药效/失忆.md)，这在体验结束后通常是不可能回忆起来的。正是在这个级别，[解离剂](/文档/药物分类/解离剂.md)如[氯胺酮](/药物/氯胺酮.md)因其麻醉作用而被用于医疗程序哦。[\[10\]](#cite_note-10)

### 躯体分离

主条目：[躯体分离](/药效/躯体分离.md)

**躯体分离**是指感觉与自己的触觉以及对身体的所有权和控制感疏远和脱离的体验呢。这导致了诸如[触觉抑制](/药效/触觉抑制.md)、[躯体自主](/药效/躯体自主.md)、[镇痛](/药效/镇痛.md)、[躯体形态感改变](/药效/躯体形态感改变.md)、[躯体轻盈感](/药效/躯体轻盈感.md)以及一系列一般的躯体抑制状态。这种效应的体验还会对一个人对自己身体的感知产生广泛的主观变化。下面的列表描述并记录了这些变化：

* 感觉自己的身体不是自己的
* 感觉自己的身体在控制自己
* 感觉自己的身体遥远而在远方
* 感觉自己的身体运动是机械和机器人的
* 感觉使用精细运动控制的能力下降
* 感觉使用和感知触觉的能力下降

躯体分离通常伴随着其他同时发生的效应，如[认知分离](/药效/认知分离.md)和[视觉分离](/药效/视觉分离.md)，其方式导致感觉一个人正在部分或完全脱离他们的感官输入和认知机能。这种效应最常在[中等](/文档/药物剂量分类.md)[剂量](/文档/给药剂量.md)的[抑制剂](/文档/药物分类/抑制剂.md)和[解离剂](/文档/药物分类/解离剂.md)化合物，如[氯胺酮](/药物/氯胺酮.md)、[PCP](/药物/PCP.md)和[右美沙芬](/药物/右美沙芬.md)影响下诱发哦。

### 视觉分离

主条目：[视觉分离](/药效/视觉分离.md)

**视觉分离**是指变得与自己的视觉感官疏远和脱离的体验。在较低级别，这导致诸如[视觉锐度抑制](/药效/视觉锐度抑制.md)、[复视](/药效/复视.md)、[模式识别抑制](/药效/模式识别抑制.md)和[视觉加工减慢](/药效/视觉加工减慢.md)等状态。这种效应的体验还会对一个人对自己视觉的感知产生广泛的主观变化呢。下面的列表描述并记录了这些变化：

* 感觉好像通过屏幕观看世界
* 视力模糊，普遍难以感知细节
* 感觉视觉上可感知的世界更加遥远
* 感觉好像通过别人的眼睛看世界
* 一种[复视](/药效/复视.md)，在较高级别，如果使用者需要阅读或感知精细的视觉细节，这就迫使他们闭上一只眼睛

视觉分离通常伴随着其他同时发生的效应，如[认知分离](/药效/认知分离.md)和[躯体分离](/药效/躯体分离.md)，其方式导致感觉一个人正在部分或完全脱离他们的感官输入和认知机能。这种效应最常在[中等](/文档/药物剂量分类.md)[剂量](/文档/给药剂量.md)的[抑制剂](/文档/药物分类/抑制剂.md)和[解离剂](/文档/药物分类/解离剂.md)化合物，如[氯胺酮](/药物/氯胺酮.md)、[PCP](/药物/PCP.md)和[右美沙芬](/药物/右美沙芬.md)影响下诱发。

在较高级别，视觉分离的效应可能会变得包罗万象。这导致与视觉感官的完全知觉断连，这可以描述为由于完全缺乏感官输入而失明且无法分辨眼睛是睁开还是闭上的体验。在这种状态下，该效应通常会导致一个人发现自己漂浮在一个黑暗且大部分空旷的幻觉虚空中的体验哦。

#### 洞、空间和虚空

[![](https://psychonautwiki.org/w/thumb.php?f=K-HOLE.jpg&width=350)](/文件/K-HOLE.jpg)

[](/文件/K-HOLE.jpg "放大")

*K-Hole 作者：[Josikins](/w/index.php?title=User:Josikins&action=edit&redlink=1)* - 这张图片是对常见的、同时发生的[解离剂](/文档/药物分类/解离剂.md)诱发效应的艺术复制，即[视觉分离](/药效/视觉分离.md)、[洞、空间和虚空](/药效/视觉分离.md#洞、空间和虚空)以及[幻觉结构](/药效/视觉分离.md#结构)。

**洞、空间和虚空**是视觉分离的一个子成分，当它的强度变得包罗万象时就会显现出来。这种体验在文献中更常被称为“K-hole”（K洞），并且尽管在大多数传统[解离剂](/文档/药物分类/解离剂.md)中都存在，但通常被讨论为与[氯胺酮](/药物/氯胺酮.md)特别相关的东西。它可以描述为一个人一旦视觉分离变得足够强大，使人无法接收外部感官输入时所处的地方。这用一个主观感觉好像是在普通现实之外的空间取代了他们的视觉输入呢。

这个空间、洞或虚空的视觉外观可以描述为一个巨大的、大部分空旷和黑暗的房间，通常感觉和看起来大小是无限的。这个空间通常是漆黑的，但偶尔会在地平线上显示出大块缓慢移动的无定形彩色云彩或微妙的[几何图案](/药效/几何.md)。在较高级别，这些虚空通常充满了幻觉结构，这在下面的小节中有全面的描述和记录哦。

除了这种视觉体验外，通常还存在[重力感改变](/药效/重力感改变.md)和强烈的[触觉分离](/药效/触觉分离.md)感，这可能导致一个人感觉好像正在经历灵魂出窍的体验，同时以各种不同的速度、方向和方位在虚空中失重地漂浮。这是一种被许多人解释为漂浮在太空中或夜空中的感觉呢。

##### **结构**

**结构**是在原本空旷和无人居住的虚空中发现的唯一特征。这些表现为无限种类和大小的整体3维形状或结构的视觉体验，它们漂浮在一个人的上方、下方、周围或前方，随着它们逐渐缩放、旋转、变换或平移进入焦点，并以渐进的速度在人的视线前揭开面纱。

这些结构可以采取任何形状的形式，但常见的例子包括巨大的柱子、圆柱、隧道、块状物、建筑物、滑梯、纪念碑、轮子、金字塔、洞穴和各种抽象形状。它们通常是分形的，可以表现为各种颜色，但通常遵循较暗的主题和色调，具有有时被主观解释为本质上是“外星”的相关美学。

结构可以分解为以下4个基本的复杂性和视觉强度级别：

1.  **2维结构** - 在最低级别，结构将其形式严格限制在2维形状。这些形状通常是扁平的，颜色较暗。它们的存在也经常是被“感觉”到的，而不是被看到的。就其大小而言，这些结构通常占据一个人的整个视野，但似乎没有任何特定的大小感归因于它们。
2.  **部分定义的3维结构** - 在这个级别，结构在形状上变得更清晰，呈现3维，并且在其光照和阴影方面有一些基本细节。它们似乎由半透明的凝聚颜色组成，并且在其边缘周围被视为不清晰或失焦。就大小而言，这些结构似乎非常大，延伸至数百米。
3.  **完全定义的3维结构** - 在这个级别，结构在形状、边缘、光照、阴影和细节方面变得完全定义。它们通常看起来是由固体和致密的现实材料制成的，如石头和金属。就其大小而言，它们能够表现为绵延数千英里的样子，并且在形状和纹理方面通常极其复杂。
4.  **结构性机器宇宙** - 在最高级别，幻觉结构可以描述为看到主观感知为整个宇宙凝聚成一个无限巨大且复杂的、不断变化的机器结构的感觉。就其外观而言，这种状态很难描述，但在主观上与[8A级几何](/药效/几何.md)有许多相似之处。结构可以采取任何形式，但通常表现为形状复杂的机器状结构，其大小似乎是无限的，并且可以传达大量天生可理解的信息。这种体验不仅仅通过视觉感知，而且还在难以理解的细节层面上被身体感受到，表现为复杂的认知和触觉感觉。作为一个整体的结构及其传达的信息通常被天生解释为感知“宇宙”或“一切”的结构表征。

结构通常会显示30秒到几分钟，然后体验它们的人会滑回现实或另一个结构的存在中。就这些结构如何在彼此之间转换而言，它们的过渡过程可以分解为4个基本类别。这些描述和记录如下：

*   **结构变换** - 结构可以通过以静态和可理解的方式变换或变形在彼此之间切换。这通常是以渐进的逐步变形过程展开的。
*   **结构平移** - 结构可以通过保持其形状静止但仅仅平移出视野直到它们不再在视野内来在彼此之间切换。通常从这里开始，另一个结构会在几秒钟到几分钟内从周边视野外进入视野。
*   **长距离旅行** - 第三种过渡方法是在结构看起来静止的同时，一个人在它们之间无声地漂浮，感觉像是极端的物理距离。这种漂浮有时感觉是发生在穿越巨大且无限的解离洞的无形轨道上。
