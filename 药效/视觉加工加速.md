# 视觉加工加速 (Visual processing acceleration)

**视觉加工加速**定义为一个人感知和解释快速发生的事件的速度增加。它最常见于感觉到危险、激烈或高度新奇的事件中[\[1\]](#cite_note-WittmannLeland2007-1)[\[2\]](#cite_note-Arstila2012-2)[\[3\]](#cite_note-BurrStetson2007-3)。这种效应使外部环境看起来像是在以慢动作观看，并且短暂的时间片段被“拉长”了。例如，快速移动的物体，如鸟类、昆虫和汽车，可能开始呈现出清晰可见的样子，而不是像在日常清醒状态下那样呈现为转瞬即逝的模糊运动。

就其强度而言，这种效应通常表现得很微妙，只是稍微减慢了一个人对运动的感知。然而，在罕见的情况下，这种效应可以暂时将视觉时间的感知减慢到接近静止的状态，导致事件看起来显著变慢。

视觉加工加速通常伴随着其他同时发生的效应，如[视觉锐度增强](../药效/视觉锐度增强.md)和[思维加速](../药效/认知增强.md)（这里译者注：原文为Thought acceleration，根据链接指向认知增强/Cognitive enhancement的分类处理）。它最常见于服用[中等](../文档/药物剂量分类.md#中等)[剂量](../文档/给药剂量.md)的[致幻剂](../文档/药物分类/致幻剂.md)类化合物时，例如[LSD](../药物/LSD.md)、[赛洛西宾](../药物/赛洛西宾蘑菇.md)和[麦斯卡林](../药物/麦斯卡林.md)。
