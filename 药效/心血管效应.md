# 心血管效应

[◀返回](./home.md)

**心血管效应**被定义为任何与心脏和血管相关的[不适性躯体效应](/药效/不适性躯体效应.md)哦。

本页列出并描述了在某些[药物全索引](/文档/药物分类/药物全索引.md)的影响下可能出现的各种心血管效应呢。

### 心律异常

主条目：[心律异常](/药效/心律异常.md)

**心律异常**（也称为**心律失常**或**心律不齐**）被定义为心跳速率或节律的问题喵。[^1] 心跳过快（大于每分钟 100 次）被称为“心动过速”，而心跳过慢（小于每分钟 60 次）被称为“心动过缓”哦。心律失常是由心脏组织的变化引起的。心脏跳动源于连锁的电信号，而这些信号会受到压力激素、电解质和药物物质的影响呢。

心律异常最常见于使用[中等](/文档/给药剂量.md#中等)[剂量](/文档/给药剂量.md)的[兴奋剂](/文档/药物分类/兴奋剂.md)和[抑制剂](/文档/药物分类/抑制剂.md)化合物的情况下，例如[可卡因](/药物/可卡因.md)、[^2][^3][^4][^5][^6][^7][^8] [苯丙胺类物质](/文档/药物分类/苯丙胺类物质.md)、[^9][^10][^11][^12][^13] [酒精](/药物/酒精.md)[^13] 和 [阿片类药物](/文档/药物分类/阿片类药物.md)哦。[^13][^14][^15] 虽然[兴奋剂](/文档/药物分类/兴奋剂.md)往往会增加人的心率，但[抑制剂](/文档/药物分类/抑制剂.md)往往会降低心率。将两者联用通常会导致危险的不规则心跳呢。

### 血压降低

主条目：[血压降低](/药效/血压降低.md)

**血压降低**可以描述为体循环动脉压力降低到异常水平的状态喵。对于成年人来说，120/80 的血压被认为是正常的。90/60 或更低的血压被认为是**低血压**，而 120/80 到 90/60 之间的血压被认为是**前低血压**哦。[^16]

血压降低最常见于使用[中等](/文档/给药剂量.md#中等)[剂量](/文档/给药剂量.md)的 [GABA 能](/文档/药物分类/抑制剂.md) [抑制剂](/文档/药物分类/抑制剂.md)化合物，如[苯二氮卓类物质](/文档/药物分类/苯二氮卓类物质.md)和[巴比妥类物质](/文档/药物分类/巴比妥类物质.md)呢。然而，它也可能在[亚硝酸酯](/药物/亚硝酸酯.md)等血管扩张化合物，以及某些[致幻剂](/文档/药物分类/致幻剂.md)和[兴奋剂](/文档/药物分类/兴奋剂.md)的影响下以不可预测的方式发生欸。

### 心率减慢

主条目：[心率减慢](/药效/心率减慢.md)

| ![](/文件/Sinus_bradycardia_lead2.png) |
| :---: |
| 这张图显示了 II 导联中的窦性心动过缓，心率约为 50 BPM 喵。 |

**心率减慢**或**心动过缓**可以描述为低于静息时正常心率的心率哦。健康成年人的心脏在休息时通常每分钟跳动 60 到 100 次。当心率波动到低于 60 BPM 的较低水平时，就被描述为心动过缓或心率异常偏低呢。

值得注意的是，心率减慢通常是心理症状的结果，是放松、[焦虑抑制](/药效/认知效应.md)、[镇静](/药效/镇静.md)和[正念](/药效/认知效应.md)的一种自然反应喵。

心率减慢最常见于使用[严重](/文档/给药剂量.md#严重)[剂量](/文档/给药剂量.md)的[抑制剂](/文档/药物分类/抑制剂.md)化合物，如 [GABA 能药物](/文档/药物分类/抑制剂.md)和[阿片类药物](/文档/药物分类/阿片类药物.md)哦。然而，它也可能在[大麻类](/文档/药物分类/大麻类.md)、[解离剂](/文档/药物分类/解离剂.md)和[兴奋剂](/文档/药物分类/兴奋剂.md)的影响下发生呢。

### 血压升高

主条目：[血压升高](/药效/血压升高.md)

**血压升高**可以描述为体循环动脉压力升高到异常水平的状态喵。120/80 的血压被认为是成年人的正常血压。90/60 或更低的血压被认为是**低血压**，而 120/80 到 90/60 之间的血压被认为是**前低血压**哦。[^17] 相反，大于 120/80 且小于 139/89 的血压被认为是**前高血压**呢。

在医学文献中，如果一个人的血压非常高（例如 >180/>110 mmHg），且症状极少或没有症状，也没有迹象或症状表明急性器官受损，这种情况被称为“高血压急症”喵。[^18] 相比之下，如果严重的血压升高伴有进行性器官或系统损伤的证据，则被称为“高血压危象”哦。

血压升高最常见于使用[严重](/文档/给药剂量.md#严重)[剂量](/文档/给药剂量.md)的[血管收缩](/药效/血管收缩.md)化合物，如传统的[兴奋剂](/文档/药物分类/兴奋剂.md)和具有[刺激](/药效/刺激.md)作用的[致幻剂](/文档/药物分类/致幻剂.md)呢。

### 心率增快

主条目：[心率增快](/药效/心率增快.md)

| ![](/文件/Tachycardia_ECG_paper.png) |
| :---: |
| 心率超过 100 BPM 喵。 |

**心率增快**或**心动过速**被描述为比静息时正常心率更快的心率哦。健康成年人的心脏在休息时通常每分钟跳动 60 到 100 次。当心率波动到 100 BPM 以上的较高水平时，就被描述为心动过速或心率异常偏高呢。

值得注意的是，心率增快通常是心理症状的结果，是针对焦虑、偏执、震惊和恐惧的自然肾上腺素反应喵。

心率增快最常见于使用[严重](/文档/给药剂量.md#严重)[剂量](/文档/给药剂量.md)的[刺激](/药效/刺激.md)性化合物，如传统的[兴奋剂](/文档/药物分类/兴奋剂.md)、某些[致幻剂](/文档/药物分类/致幻剂.md)和某些[解离剂](/文档/药物分类/解离剂.md)哦。这被认为是[多巴胺](/文档/多巴胺.md)或[肾上腺素](/文档/肾上腺素.md)调节的直接结果呢。[^19][^20] 然而，它也可能在[谵妄剂](/文档/药物分类/谵妄剂.md)的影响下发生，因为它们会抑制[乙酰胆碱](/文档/乙酰胆碱.md)——这是外周神经系统中调节心率的主要物质之一喵。[^21][^22]

### 血管收缩

主条目：[血管收缩](/药效/血管收缩.md)

| ![](/文件/Vasodilation_and_vasoconstriction.png) |
| :---: |
| 该图展示了血管在扩张、收缩和正常状态下的结构差异哦。 |

**血管收缩**可以描述为由于血管壁肌肉收缩而导致的静脉和血管变窄喵。这在大动脉和小动脉中尤为普遍。

这种效应通常会导致一个人的手臂和腿部感到紧绷、酸痛和麻木哦。其影响范围可以从轻微到极度不适呢。

血管收缩通常伴随着其他同时发生的效应，如[刺激](/药效/刺激.md)喵。它最常见于使用[中等](/文档/给药剂量.md#中等)[剂量](/文档/给药剂量.md)的[刺激](/药效/刺激.md)性[致幻剂](/文档/药物分类/致幻剂.md)化合物，如 [LSD](/药物/LSD.md)、[2C-E](/药物/2C-E.md) 和 [DOC](/药物/DOC.md) 呢。然而，它也可以在传统的[兴奋剂](/文档/药物分类/兴奋剂.md)如[甲基苯丙胺](/药物/甲基苯丙胺.md)、[咖啡因](/药物/咖啡因.md)和 [MDMA](/药物/MDMA.md) 的影响下发生哦。

### 血管扩张

主条目：[血管扩张](/药效/血管扩张.md)

**血管扩张**可以描述为由于血管壁内平滑肌细胞松弛而导致的静脉和血管加宽喵。这在大动脉和小动脉中尤为普遍。血管扩张的主要功能是增加体内流向最需要它的组织的血流量。本质上，这个过程与[血管收缩](/药效/血管收缩.md)相反，后者是血管的变窄哦。

这种效应通常很难被有意识地察觉，但通常会导致眼睛充血发红的效果，并能缓解青光眼呢。

血管扩张通常伴随着其他同时发生的效应，如[血压降低](/药效/血压降低.md)喵。它最常见于使用[中等](/文档/给药剂量.md#中等)[剂量](/文档/给药剂量.md)的[大麻类](/文档/药物分类/大麻类.md)化合物，如[大麻](/药物/大麻.md)、[JWH-018](/药物/JWH-018.md) 和 [THJ-018](/药物/THJ-018.md) 哦。然而，它也可以在[亚硝酸酯](/药物/亚硝酸酯.md)和西地那非（伟哥）的影响下发生呢。
