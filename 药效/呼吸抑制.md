# 呼吸抑制

**呼吸抑制**（Respiratory depression，也称为**通气不足**）可以描述为呼吸冲动的减少，根据其强度，这可是会致命的哦。在相对安全的水平下，这种效应通常会导致“叹气”式的呼吸模式，表现为深呼吸之间有异常长的停顿。然而，在较高水平下，个体可能会完全停止呼吸，如果不立即治疗，会迅速致命。

这种效应可以在以下4种不同的强度水平上表现出来：

1.  **轻微呼吸抑制** - 在最低水平，呼吸抑制通常非常微妙，难以察觉，并伴有轻微的[镇静](/药效/镇静.md) 作用。
2.  **中度呼吸抑制** - 在这个水平，人会意识到自己每分钟的呼吸次数比平时少。这种程度的呼吸抑制并不让人难受，也不会导致呼吸短促、挣扎或受损。
3.  **严重呼吸抑制** - 在这个水平，人的呼吸频率会显著减慢，导致人感觉呼吸异常、气短，吸入的空气不足。这迫使人的注意力转向手动调节自己的呼吸，以免感到极度不适。此时通常也会出现极度的[镇静](/药效/镇静.md)，如果睡着了，人可能会因为挣扎和喘不过气来而醒来。在这个水平上经常出现困惑、恐慌和[焦虑](/药效/焦虑.md)，进一步增强了那种感觉自己会因为睡觉或注意力不集中而完全停止呼吸的强烈感觉，好可怕呢。
4.  **呼吸衰竭** - 在最高水平，人的氧气和二氧化碳水平受到严重损害。人会陷入半意识状态，完全失去知觉，陷入昏迷，或完全停止呼吸。皮肤、指甲或嘴唇也可能呈现蓝色。如果不立即就医，这种程度的呼吸衰竭可能会致命。[\[1\]](#cite_note-1) 阿片类药物和巴比妥类药物，在过量使用或与其他抑制剂联用时，是导致此类死亡的罪魁祸首。

呼吸抑制通常伴随着其他并发效应，比如[镇静](/药效/镇静.md) 和[困倦](/药效/困倦.md)。它最常发生在使用[严重](/文档/给药剂量.md) [剂量](/文档/给药剂量.md) 的[镇静剂](/文档/镇静剂.md) 化合物时，特别是[阿片类药物](/文档/药物分类/阿片类药物.md)，如[海洛因](/药物/海洛因.md) 和[芬太尼](/药物/芬太尼.md)，或[GABA能药物](/文档/药物分类/加巴喷丁类物质.md)，如[酒精](/药物/酒精.md) 和[GHB](/药物/GHB.md)。值得注意的是，当与其他类型的抑制剂即使是少量联用时，这些化合物原本安全的剂量也可能变得致命。例如，[苯二氮卓类物质](/文档/药物分类/苯二氮卓类物质.md) 与[阿片类药物](/文档/药物分类/阿片类药物.md) 联用是导致致命呼吸抑制的极常见原因。因此，强烈建议不要在任何剂量范围内联用这些抑制剂哦！⚠️

#### 治疗

为了防止死亡，如果出现严重呼吸抑制，建议立即联系紧急医疗服务。如果是由阿片类药物过量引起的，应使用阿片拮抗剂，如[纳洛酮](/药物/纳洛酮.md)。许多减少伤害组织免费向使用者提供纳洛酮，也可以在药房购买。除非因其他[镇静剂](/文档/镇静剂.md) 而变得复杂，否则纳洛酮将迅速逆转呼吸抑制。

对于其他药物引起的呼吸抑制，可能需要住院治疗和机械呼吸机的辅助。
