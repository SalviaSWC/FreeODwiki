# 眼球滑动

**眼球滑动**（Optical sliding）可以描述为一种通过抑制人保持眼球静止的能力来阻碍眼球协调和控制的身体效应。这会导致眼球不断向各个方向移动，同时感觉无法盯着任何特定的点看呢。

眼科疾病[眼球震颤](/文档/药物/Nystagmus.md)会让患者出现非常相似的症状，尽管眼球震颤的移动频率通常要高得多哦。

眼球滑动通常伴随着其他的效应，比如[视觉锐度抑制](/药效/视觉锐度抑制.md)和[复视](/药效/复视.md)。它最常见于在使用了[严重](/文档/药物剂量分类.md)[剂量](/文档/给药剂量.md)的[解离剂](/文档/药物分类/解离剂.md)之后，比如[氯胺酮](/药物/氯胺酮.md)、[PCP](/药物/PCP.md)和[右美沙芬](/药物/右美沙芬.md)。不过呢，在使用极高剂量的[GABA](/文档/GABA.md)能[抑制剂](/文档/药物分类/抑制剂.md)时，也可能在较小程度上发生这种情况嘛。
