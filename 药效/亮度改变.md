# 亮度改变

| | |
|---|---|
| ![](../文件/Brightness_alteration.jpg) | **清醒状态 vs 亮度改变** 由 **[Josikins](../文件/User_Josikins)** 创作 |

**亮度改变**定义为构成一个人视觉的感知亮度水平的扭曲或变化呢。这通常会导致视觉变得更暗[1][2][3]且柔和，但也可能导致视觉变得更亮且更生动[3][4]，具体取决于环境和所摄入的物质哦。

亮度改变可能伴随着[瞳孔扩大](/药效/瞳孔扩大.md)或[瞳孔缩小](/药效/瞳孔缩小.md)以及[畏光](/药效/畏光.md)等并发效应。它最常在[中等](/文档/给药剂量.md#中等)剂量的[致幻](/文档/药物分类/致幻剂.md)化合物影响下诱发，例如[迷幻剂](/文档/药物分类/迷幻剂.md)、[解离剂](/文档/药物分类/解离剂.md)和[大麻类](/文档/药物分类/大麻类.md)物质呢。
