

# 时间扭曲

### 时间膨胀

时间膨胀定义为时间变慢的感觉。[\[3\]](#cite_note-3) 这通常发生在强烈的致幻体验期间，似乎是因为在激烈的旅程（Trip）中，在非常短的时间内感受到了异常大量的体验。[\[4\]](#cite_note-HillFischer1969-4)[\[5\]](#cite_note-Fischer1971-5) 这会产生一种错觉，让人觉得过去的时间比实际上要多。例如，在某些体验结束时，一个人可能会主观地感觉他们已经经历了好几天、好几周、几个月、几年，甚至无限长的时间。

时间膨胀通常伴随着其他并发效应，如[灵性强化](/药效/灵性增强.md "Spirituality intensification")、[\[6\]](#cite_note-Buckley1981-6) [思维循环](/药效/思维循环.md "Thought loops")、[新奇感增强](/药效/新奇感增强.md "Novelty enhancement")和[内部幻觉](/药效/内部幻觉.md "Internal hallucinations")，这种方式可能会让人觉得考虑到现实世界中实际过去的时间，自己感知到了不成比例的大量事件。它最常在[严重](/文档/药物剂量分类.md#heavy "Dosage") [剂量](/文档/给药剂量.md "Dosage")的[致幻](/文档/药物分类/致幻剂.md "Hallucinogen")化合物影响下被诱发，例如[迷幻剂](/文档/药物分类/迷幻剂.md "Psychedelic")、[\[7\]](#cite_note-7)[\[8\]](#cite_note-RileyBlackman2009-8) [解离剂](/文档/药物分类/解离剂.md "Dissociative")、[共情剂](/文档/药物分类/共情剂.md "Entactogen")[\[9\]](#cite_note-NikolovaDanchev2014-9)[\[10\]](#cite_note-10)以及[大麻素](/文档/药物分类/大麻类.md "Cannabinoid")。

### 时间压缩

时间压缩定义为时间加速流逝的体验，比清醒时快得多。例如，在这种状态下，一个人可能会意识到整个晚上已经过去了，但感觉只过了几个小时。

这通常发生在某些[兴奋](/文档/药物分类/兴奋剂.md "Stimulant")化合物的影响下，似乎至少部分源于这样一个事实：在强烈的刺激水平下，人们通常会过度专注于活动和任务，这种方式会让时间在不知不觉中流逝。然而，这种体验也可能发生在引起[失忆](/药效/失忆.md "Amnesia")的[抑制剂](/文档/药物分类/抑制剂.md "Depressant")化合物上。这是因为一个人实际上可能会忘记在受到物质影响时发生的一切，从而给人一种突然跳到未来的印象。

时间压缩通常伴随着其他并发效应，如[记忆抑制](/药效/记忆抑制.md "Memory suppression")、[专注力强化](/药效/专注力强化.md "Focus intensification")、[刺激](/药效/刺激.md "Stimulation")和[失忆](/药效/失忆.md "Amnesia")，这种方式可能会让人觉得考虑到现实世界中实际过去的时间，自己感知到了不成比例的少量事件。它最常在[严重](/文档/药物剂量分类.md#heavy "Dosage") [剂量](/文档/给药剂量.md "Dosage")的[兴奋性](/文档/药物分类/兴奋剂.md "Stimulating")和/或[致健忘](/文档/药物分类/抑制剂.md "Amnesic")化合物影响下被诱发，[\[11\]](#cite_note-GriffithNutt1975-11) 例如[解离剂](/文档/药物分类/解离剂.md "Dissociatives")、[\[12\]](#cite_note-CorazzaAssi2013-12) [共情剂](/文档/药物分类/共情剂.md "Entactogen")、[苯丙胺类物质](/文档/药物分类/苯丙胺类物质.md "GABAergic")和[苯二氮卓类物质](/文档/药物分类/苯二氮卓类物质.md "Benzodiazepine")。

### 时间倒流

时间倒流被定义为这样一种感知：在过去几分钟到几小时内发生在自己周围的事件、幻觉和经历，正自发地向后播放，方式有点类似于倒带的VHS录像带。在这个倒流过程中，当事人的认知和思维流通常会继续以连贯线性的方式向前播放，而他们却看着周围的外部环境和身体的物理动作以相反的顺序播放。这既可能实时发生，即5分钟的时间倒流大约需要5分钟才能完全倒退，也可能以加速的方式发生，即5分钟的时间倒流只需不到一分钟。可以合理推测，时间倒流的体验可能是通过[内部幻觉](/药效/内部幻觉.md "Internal hallucinations")和记忆编码错误的结合而发生的。

时间倒流通常伴随着其他并发效应，如[内部幻觉](/药效/内部幻觉.md "Internal hallucination")、[思维循环](/药效/思维循环.md "Thought loops")和[既视感](/药效/既视感.md "Deja vu")。它最常在极[严重剂量](/文档/药物剂量分类.md#heavy "Dosage")的[致幻](/文档/药物分类/致幻剂.md "Hallucinogen")化合物影响下被诱发，例如[迷幻剂](/文档/药物分类/迷幻剂.md "Psychedelic")、[解离剂](/文档/药物分类/解离剂.md "Dissociative")和[谵妄剂](/文档/药物分类/谵妄剂.md "Deliriant")。
