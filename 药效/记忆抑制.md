
# 记忆抑制

[◀返回](./home.md)

**记忆抑制**（也被称为**自我抑制**、**自我消解**、**自我丧失**或**自我死亡**）被定义为抑制一个人维持功能性短期和长期记忆的能力哦。[\[1\]](#cite_note-LebedevL.C3.B6vd.C3.A9n2015-1)[\[2\]](#cite_note-Carhart-HarrisMuthukumaraswamy2016-2)[\[3\]](#cite_note-Carhart-HarrisLeech2014-3) 这种发生方式与服用的剂量成正比，并且通常始于短期记忆的退化呢。

记忆抑制是一个可以细分为以下4个基本层次的过程喵：

1. **部分短期记忆抑制** - 在最低层级，这种效应是一个人短期记忆的部分和潜在的不一致的失败。它会导致诸如普遍[难以保持专注](/药效/专注力抑制.md)、分心增加以及普遍倾向于忘记自己在想什么或说什么的效应呢。

2. **完全短期记忆抑制** - 在这个层级，这种效应是一个人短期记忆的完全失败哦。它可以被描述为完全无法记住关于当前情况以及导致当前情况的事件的具体细节超过几秒钟的体验。这种心理状态通常会导致[思维循环](/药效/思维循环.md)、[困惑](/药效/困惑.md)、迷失方向和失控，尤其是对于没有经验的人来说喵。在这个层级，同时也变得不可能跟上对话和大多数形式的媒体情节了呢。

3. **部分长期记忆抑制** - 在这个层级，除了短期记忆的完全失败之外，这种效应还是一个人长期记忆的部分、通常是间歇性的失败。它可以被描述为难以从长期记忆中回忆起基本概念和自传信息。加上短期记忆的完全抑制，它创造了一种改变的状态，即使是基本的任务也变得具有挑战性或不可能，因为人们无法在心理上访问过去关于如何完成这些任务的记忆哦。例如，一个人可能需要更长的时间来回忆亲密朋友的身份，或者暂时忘记如何执行基本任务。这种状态可能会产生第一次经历某事的感觉呢。在这个阶段，某些习得的人格特征、文化规范意识和语言回忆的减少可能会伴随着长期记忆的抑制喵。

4. **完全长期记忆抑制** - 在最高层级，这种效应是一个人长期和短期记忆的完全和持久的失败哦。它可以被描述为完全无法记住存储在一个人长期记忆中的哪怕是最基本的概念的体验。这包括从他们的名字、家乡、过去的记忆、意识到正在服药、药物是什么、人类是什么、生活是什么、时间是否存在、任何东西是什么，或者任何东西是否存在的一切呢。这个层级的记忆抑制阻断了所有的心理联想、附加意义、获得的偏好和一个人对外部世界可能拥有的价值判断。足够强烈的记忆丧失也与自我意识的丧失有关，在这种状态下，一个人不再意识到自己的存在。在这种状态下，使用者无法回忆起关于自己和外部世界的所有习得的概念性知识，也不再体验到作为外部世界中一个独立观察者的感觉喵。这种体验通常被称为“自我死亡”。

记忆抑制通常伴随着其他并发效应，如[思维循环](/药效/思维循环.md)、[个人偏见抑制](/药效/个人偏见抑制.md)、[健忘](/药效/健忘.md)和[妄想](/药效/妄想.md)。它最常在[中等](/文档/给药剂量.md#中等)[剂量](/文档/给药剂量.md)的[致幻](/文档/药物分类/致幻剂.md)化合物影响下被诱发，如[迷幻剂](/文档/药物分类/迷幻剂.md)、[解离剂](/文档/药物分类/解离剂.md)和[谵妄剂](/文档/药物分类/谵妄剂.md)呢。[\[4\]](#cite_note-VollenweiderGeyer2001-4)

值得注意的是，虽然记忆抑制在效果上与[健忘](/药效/健忘.md)模糊相似，但不同之处在于它直接抑制了一个人使用其长期或短期记忆的能力，而不会抑制事后回忆这段经历中发生的事情的能力哦。相比之下，健忘并不直接影响一个人在体验过程中使用其短期或长期记忆，而是让人在药效消退后无法回忆起事件。经历记忆抑制的人无法访问他们现有的记忆，而药物引起的健忘的人无法正确存储新的记忆呢。因此，一个经历健忘的人可能并不明显表现出这种状态，因为他们通常可以进行正常的对话并执行复杂的任务。但记忆抑制的情况并非如此喵。

### 自我消解

完全长期记忆抑制（第4级）最重要的方面是它抑制回忆和理解与一个人的自我感和身份相关的概念信息的能力的方式哦。这种体验俗称[自我死亡](http://en.wikipedia.org/wiki/Ego_death)，其发生在现代[精神探索](/文档/精神探索.md)亚文化中被详细记录。然而，值得注意的是，这种“自我死亡”或“自我消解”的非正式术语也通常用来指代高水平的[联合与互联](/药效/联合与互联.md)状态呢。[\[1\]](#cite_note-LebedevL.C3.B6vd.C3.A9n2015-1)[\[5\]](#cite_note-NourEvans2016-5)

完全的记忆抑制会导致一种深刻的体验，即尽管保持完全清醒，但不再有一个“我”在体验一个人的感官输入；只有感官输入本身，且只有它自己喵。这抑制了清醒生活中几乎持续不断的作为一个独立观察者与外部世界互动的感觉得以存在。虽然自我死亡并不一定会关闭所有心理过程的意识，但它确实消除了作为思维者或心理过程起因的感觉哦。它通常导致一种完全不受过去记忆、先前经验、背景和偏见污染的中立视角来处理概念的感觉呢。

自我死亡通常与其他并发效应协同作用，如[个人偏见抑制](/药效/个人偏见抑制.md)、[联合与互联](/药效/联合与互联.md)、[灵性增强](/药效/灵性增强.md)和[妄想](/药效/妄想.md)喵。[\[1\]](#cite_note-LebedevL.C3.B6vd.C3.A9n2015-1)[\[3\]](#cite_note-Carhart-HarrisLeech2014-3) 这些伴随效应进一步提升了自我死亡体验的主观强度和超个人意义哦。[\[2\]](#cite_note-Carhart-HarrisMuthukumaraswamy2016-2)
