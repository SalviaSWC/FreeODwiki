# 瞳孔缩小

| ![Pupilc](/文件/Pupilc.jpg) |
| :---: |
| 瞳孔缩小的图片示例 |

**瞳孔缩小**（也称为**针尖样瞳孔**）可以被描述为在正常光照条件下，一个人的瞳孔尺寸减小。这通常会降低一个人在低光条件下看东西的能力哦。

瞳孔缩小最常见于服用[中等](/文档/给药剂量.md#中等)[剂量](/文档/给药剂量.md)的[阿片类药物](/文档/药物分类/阿片类药物.md)时，如[海洛因](/药物/海洛因.md)、[卡痛叶](/药物/卡痛叶.md)（原文Kratom，假设已有条目，若无则为`卡痛.md`，这里根据习惯暂定）、[曲马多](/药物/曲马多.md)和[芬太尼](/药物/芬太尼.md)。
