# 谵妄剂（Deliriants）

**分类描述**：  
谵妄剂系一类独特的精神活性物质，其核心效应为诱发高度逼真的谵妄状态（delirium），使个体完全丧失对现实的辨识能力。受影响者常经历生动、交互式的幻觉（如与不存在之人对话、感知昆虫爬行），伴随时间感扭曲、定向力障碍及记忆缺失。该类物质不同于致幻剂（psychedelics）的感知增强，亦不同于解离剂的“脱离”感，而是以抗胆碱能机制为主导，产生一种“醒着做梦”的混沌状态，常被描述为“真实噩梦”。非医疗使用下，该类体验普遍被报告为极度不愉快且恐怖。

**药理机制**：  
谵妄剂的主要作用靶点为毒蕈碱型乙酰胆碱受体（muscarinic acetylcholine receptors, M1–M5）的竞争性拮抗。该拮抗作用广泛阻断中枢及外周胆碱能传导，导致大脑皮层与基底节功能失调，干扰觉醒-注意力网络及记忆巩固通路。同时，外周效应包括瞳孔散大、心率加快、口干及尿潴留。高剂量时可完全抑制快速眼动睡眠相关胆碱能激活，产生类似于自然谵妄的脑电模式。

**精神病理特征**：  
受影响个体常呈现急性器质性谵妄综合征：意识清晰度显著降低、注意力无法维持、思维支离破碎。幻觉以视觉为主，具高度真实性与交互性（例如与幻影人物进行完整对话，或试图“捡起”不存在的物体）；伴随严重定向障碍（不知身处何时、何地），短期记忆几乎完全缺失，导致事后对体验的回忆呈碎片化或完全黑障。行为层面可见无目的游走、刻板动作及攻击性冲动。恢复期可出现持久性认知缺损、焦虑障碍或创伤后应激症状。长期反复暴露可能诱发持续性感知障碍（HPPD样表现）或胆碱能系统永久性下调。

**注意事项**：  
该类物质剂量-效应曲线极陡，微小剂量差异即可导致从轻度混沌至致命中毒的转变。非医疗使用极度危险，常伴严重生理风险（高热、癫痫、心律失常）及不可预测的精神崩溃。强烈建议避免任何形式的自行摄入
