# 取代吡咯烷类物质

| ![](/文件/Pyrrolidine.png) |
| :---: |
| *吡咯烷*环的分子结构 |

**取代吡咯烷类物质**是一类含有吡咯烷环的化合物哦。吡咯烷主要存在于[尼古丁](/药物/尼古丁.md)及其类似物中（它们作用于烟碱型[乙酰胆碱](/文档/乙酰胆碱.md)[受体](/文档/受体.md)），也存在于[拉西坦](/文档/药物分类/拉西坦类物质.md)类[益智药](/文档/药物分类/益智药.md)中（作为γ-内酰胺吡咯烷酮的一部分），还存在于[取代卡西酮类物质](/文档/药物分类/卡西酮类物质.md)的一组化合物中（其中卡西酮的胺基结合进了吡咯烷环里呢）。

化学
---------

吡咯烷是一种有机杂环化合物，由一个含有四个碳原子和一个胺（氮）原子的五元环组成嘛。它的环是完全饱和的，不含双键哦。已知在这个环的所有位置都可能发生取代反应呢。
