    119.ME

    间艾司卡林；3,4-二甲氧基-5-乙氧基苯乙胺

    合成： 向剧烈搅拌的18.6 g 5-溴波旁醛在100 mL 二氯甲烷中的悬浮液中加入14.2 g 碘甲烷、1.0 g 癸基三乙基铵碘化物和120 mL 5% 氢氧化钠。颜色呈深琥珀色，在1分钟内顶层相凝固成固体。加入另外50 mL 水后，这大部分被分散。允许反应搅拌2天。下层相用水洗涤并保存。上层相用另外100 mL 二氯甲烷、50 mL 25% 氢氧化钠、另一克癸基三乙基铵碘化物和另外50 mL 碘甲烷处理。形成的固体在几小时内自行分散，产生两个相对清晰的层。继续搅拌另外3天。分离下层相，用水洗涤，并与之前的提取液合并。真空除去溶剂得到20.3 g 琥珀色油状物，在0.4 mm/Hg压力、120-133 °C下蒸馏，得到15.6 g 3-溴-4-甲氧基-5-乙氧基苯甲醛，为白色结晶固体，熔点52-53 °C。

    将15.6 g 3-溴-4-甲氧基-5-乙氧基苯甲醛和10 mL 环己胺的混合物用明火加热，直到看起来没有水。残留物置于真空（0.5 mm/Hg）下，在148-155 °C下蒸馏，得到19.2 g 3-溴-N-环己基-4-甲氧基-5-乙氧基苄亚胺，为米白色结晶固体，熔点66-68.5 °C。从100 mL 沸腾的甲醇中重结晶，熔点为67-68.5 °C。红外光谱中的C=N伸缩振动在1640 cm-1。分析 (C16H22BrNO2) C, H。

    将17 g 3-溴-N-环己基-4-甲氧基-5-乙氧基苄亚胺溶解在200 mL 无水乙醚中，置于氦气氛围下，磁力搅拌，并用外部干冰丙酮浴冷却。然后在2分钟内加入38 mL 1.55 M 的丁基锂己烷溶液，产生澄清的黄色溶液。然后一次性加入25 mL 硼酸丁酯，允许搅拌下的溶液恢复至室温。随后加入100 mL 饱和硫酸铵水溶液。分离乙醚层，用额外的饱和硫酸铵溶液洗涤，并在真空下蒸发。残留物溶解在200 mL 50% 甲醇中，并用12 mL 30% 过氧化氢处理。该反应轻微放热，允许搅拌15分钟，然后加入到50 g 硫酸铵的水溶液中。用2x100 mL 二氯甲烷萃取，合并萃取液用水洗涤一次，真空除去溶剂。残留物悬浮在稀盐酸中，在蒸汽浴上加热0.5小时。继续搅拌直到反应再次达到室温，然后用2x100 mL 二氯甲烷萃取。合并这些萃取液，依次用2x100 mL 稀氢氧化钠萃取。水相萃取液用盐酸重新酸化，并用2x100 mL 二氯甲烷重新萃取。合并后，真空除去溶剂得到油状残留物。在0.2 mm/Hg压力、118-130 °C下蒸馏，得到7.5 g 3-乙氧基-5-羟基-4-甲氧基苯甲醛馏分，该馏分固化为白色晶体。从环己烷中重结晶得到熔点为77-78 °C的产物。分析 (C10H12O4) C, H。

    将7.3 g 3-乙氧基-5-羟基-4-甲氧基苯甲醛在100 mL 丙酮中的溶液用5 mL 碘甲烷和8.0 g 细粉末状无水碳酸钾处理，并在蒸汽浴上回流6小时。真空除去溶剂，残留物悬浮在水中。使其呈强碱性后，用3x50 mL 二氯甲烷萃取，合并萃取液，真空除去溶剂。残留的琥珀色油状物在0.4 mm/Hg压力、110-120 °C下蒸馏，得到7.3 g 白色油状物。这自发固化为3,4-二甲氧基-5-乙氧基苯甲醛的白色晶体，熔点为49-49.5 °C。分析 (C11H14O4) C, H。同样的醛也可以通过在MP条目下描述的3,4-二甲氧基-5-羟基苯甲醛的乙基化获得，但产率不太令人满意。

    将7.2 g 3,4-二甲氧基-5-乙氧基苯甲醛在含有0.1 g 无水乙酸铵的100 mL 硝基甲烷中的溶液回流50分钟。真空除去过量的硝基甲烷，产生6.8 g 红色油状物，将其从一些不溶物质中倾析出来。向倾析物中加入10 mL 热甲醇，得到均匀溶液，冷却后自发结晶。通过过滤除去黄色晶体，用少量甲醇洗涤并空气干燥，得到3.5 g 3,4-二甲氧基-5-乙氧基-beta-硝基苯乙烯黄色晶体，从甲醇中重结晶后熔点为89.5-90 °C。分析 (C12H15NO5) C, H。

    将2.0 g 氢化铝锂在100 mL 无水四氢呋喃中的溶液在氦气下冷却至0 °C并剧烈搅拌。在10分钟内滴加1.3 mL 100% 硫酸，随后滴加3.1 g 3,4-二甲氧基-5-乙氧基-beta-硝基苯乙烯的50 mL 无水四氢呋喃溶液。混合物在0 °C下搅拌一会儿，然后在蒸汽浴上回流30分钟。再次冷却后，用四氢呋喃中的异丙醇破坏过量的氢化物，随后加入20 mL 10% 氢氧化钠，这足以将固体转化为白色粒状。通过过滤除去这些固体，用异丙醇洗涤滤饼，合并母液和滤液，真空除去溶剂。残留物加入到150 mL 稀硫酸中，混浊的悬浮液用2x75 mL 二氯甲烷洗涤，这除去了大部分颜色。水相用25% 氢氧化钠调至碱性，并用3x50 mL 二氯甲烷萃取。从这些合并的萃取液中除去溶剂，残留物在0.25 mm/Hg压力、103-116 °C下蒸馏，得到2.3 g 无色粘稠液体。将其溶解在10 mL 异丙醇中，用约25滴浓盐酸中和，产生了不溶的白色固体。用40 mL 无水乙醚稀释，在持续搅拌下缓慢加入。通过过滤分离出白色结晶状3,4-二甲氧基-5-乙氧基苯乙胺盐酸盐（ME），用乙醚洗涤，空气干燥，重2.4 g。其熔点为202-203 °C，从沸腾的异丙醇中重结晶后熔点增加了一度。分析 (C12H20ClNO3) C, H。

    给药剂量： 200 - 350 mg。

    药效时长： 8 - 12 h。

    定性评论： （200 mg）尝起来相当重。然而，味道很快就消失了，一种充满活力的感觉开始占据我的身体。它继续增长。这种感觉是一种强烈的同志情谊，和人交谈变得非常容易。每个人都在和别人交谈。我发现它非常愉快，充满活力，同时又令人放松，卸下了防备。这种物质似乎不会导致内省；然而，如果没有其他人在场，它可能会。视觉意识的提高是轻微的，但听觉意识相当高。与每个人在一起的感觉很强烈。

    （250 mg）最初我服用了200毫克的间艾司卡林，体验对我来说起初发展得非常缓慢，而且非常愉快。大约半小时后，我意识到有一堵墙似乎把我关在里面，但并非不愉快。墙慢慢溶解了，但我担心我可能会陷入消极的体验。在服用额外的50毫克（实验开始2:23后）后，我立刻感到了缓解（从这种孤立感中），好像对这个决定感到高兴。我躺在外面的毯子上。内心有一种美妙的感觉，尽管没有意象。我感觉到那堵墙完全溶解了，我渴望加入小组。从这一点开始，体验最令人愉悦，处于欣快状态。虽然不像某些致幻剂那样富有戏剧性，但对我个人来说是非常有回报的。我感到与在场的每个人都有美妙的纽带，头脑清醒，思维出色，沟通顺畅。总而言之，这是一次最有回报且令人愉快的经历。事后我感到更有力量，精力充沛，洞察力强。我有一种强烈的感觉，即小组定制了这种体验的性质，我和其他人也非常渴望小组互动。我觉得如果一个人把注意力转向它，可以用它做很多其他事情。

    （275 mg）相对于其他心理化学物质，身体和心理变化的发生都很缓慢。在大约一个半小时的时间点感受到了非常缓慢的内部搅动。这些更多是感受性的而非认知性的，并且相当令人愉悦。在大约两个半小时的时间点，我变得非常口渴，喝了一品脱啤酒。几乎立即，且完全出乎意料地，我tomso到了一个高得多的水平，并在那里停留了另外三个小时，直到整个体验消退。[动词“tomso”意思是由于少量的酒精而突然重新点燃药物诱导的改变状态。这在TOMSO的配方中有所解释。] 在体验高峰期间，事实上在达到高峰之前，交谈是容易且通畅的。MDMA特有的那种移情感觉基本上不存在。但出于心理治疗的目的，它有一些优势：流畅的联想、不设防的立场和普遍的愉悦感。

    （400 mg）在下午1:30左右摄入了300毫克。非常安静的攀升。偶尔打哈欠。对世界采取实事求是的看法。没有玫瑰色的光辉。在第二个小时结束时，我似乎停留在 ++ 状态。下午3:45再服用100毫克。尝起来依然糟糕。服用补充剂十五分钟后感到轻微的冲顶感（head-rush），半小时内我完全达到了 +3。有一段时间，这是一种更严厉的麦司卡林。看到了永恒、持续的选择，所有的对立面都在相互运动中。阴阳无处不在，赋予每个分子生命。宇宙本身通过作用力-反作用力、是-否、黑-白、男-女、正-负的行动保持活力。所有的生命都是在各个层面持续不断地做出选择。然后我闭上眼睛，发现自己漂浮到寺庙的最顶端，那里有灿烂的光芒和一种回家的感觉。做爱是流过岩石和峡谷的清澈溪流——大地和天空在做爱，岩石向其他岩石示爱，而水则是爱之行动中调情、抚摸、生存和移动。要意识到，在某种程度上，所有的存在都在向所有其他存在示爱。日本花园：一种结构化的方式，展示了宇宙做爱的一瞥，以便它可以被其他人类灵魂解读。所有的爱，当直接、自由且不苛求时，都是对源头的触摸。当然，最难的一课是如何以同样的方式爱你自己。它既是幼儿园的第一课，也是博士学位的终期考试。我能够在凌晨4:00左右入睡。

    延伸和评论： 需要更多的评论。必须指出，艾司卡林（E）的单个乙基重新定位到间位（meta-position）产生了间艾司卡林（ME）。在50多年前对猫的研究中，这两种化合物在类似剂量下产生了类似的效果。在人类中，ME在药效上也似乎与麦司卡林相似。然而，ME与麦司卡林的天然来源乌羽玉（Peyote）之间存在微妙的差异。对于乌羽玉本身，原始仙人掌的初始味道不仅仅是恶心；更准确地描述应该是难以置信的恶心。但在乌羽玉体验的中途，仙人掌的味道确实变得友好了。当在体验中途再次品尝ME时，味道仍然很恶心。

    与麦司卡林还有其他区别。与麦司卡林或乌羽玉不同，在中毒的早期阶段很少有任何身体不适，没有恶心，只有偶尔的评论暗示反射亢进。而且，同样与麦司卡林不同，大多数关于ME的主观报告声称音乐产生的意象很少，对色彩感知的夸大也更加保守。食欲正常，食物的味道和质地异常诱人。没有受试者曾表示不愿重复这种体验。睡眠容易且令人清爽，第二天似乎没有残留感。
