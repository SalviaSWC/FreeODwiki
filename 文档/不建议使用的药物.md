# 不建议使用的药物清单

<mark> 不错啊</mark>

对于需要谨慎使用但不一定完全避免的精神活性物质（本页面列出的物质除外），请查阅[分类:面板](./home.md)页面。

本页面旨在提供基本信息并在事故或伤害发生前提高警惕，同时也为那些已经进行不负责任药物使用的人敲响警钟，促使他们重新考虑自己的选择并在需要时寻求帮助。有趣的是，值得注意的是，有些人认为精神探索的最终目标是达到一种不再需要精神活性物质的心智状态，因为一个人已经培养了持久的觉察和洞察力。按照这种观点，真正的精神探索者超越了对外在物质的需求，将教训和体验整合到了他们的日常意识中。

此外，这些信息对于不使用药物的人也很有价值，因为他们在教育他人和预防社区内的危险情况方面可以发挥关键作用。

## 按物质分类

### 单一物质使用

本资源不仅涵盖药物，还包括最初并非用于人类消费的危险精神活性物质，例如“黄蜂毒品”（wasp dope）——一种通过摄入常见家用杀虫剂以达到药效（high）的做法。

| 物质 | 避免原因 |
| --- | --- |
| 所有非法生产的药丸，例如作为摇头丸出售的那些 | 例如，纯粉末或晶体 [MDMA](/药物/MDMA.md) 通常被认为更安全。然而，“三菱”（Mistubishi）品牌在摇头丸使用者中曾经拥有的良好声誉因致命批次的发现而遭到打击（白色三菱中发现了 [PMA](/药物/PMA.md) [\[1\]](#cite_note-1)，红色三菱中发现了 [PMMA](/药物/PMMA.md) [\[2\]](#cite_note-2)）。 |
| 新型或有毒的[研究用化学品](/文档/研究用化学品.md) | 研究用化学品（RCs）构成重大健康风险，并不适合人类消费，具有未知的长期影响和潜在的严重医疗后果。避免新流行的 RCs 尤为重要，因为它们的安全状况知之甚少，重大危害往往在多年使用后才显现出来。例如，在十年后，25I-NBOMe 与全球至少 42 例死亡有关，凸显了对其危险性认识的滞后。[\[3\]](#cite_note-25I-NBOMe-3) 另一方面，像 [5-MeO-MiPT](/药物/5-MeO-MiPT.md) 这样的物质，虽然其毒性仍有不确定性，但已为人所知约 40 年，可能具有更确定的安全状况。这是因为其长期使用并未导致大量报道的死亡或伤害，表明与较新的、研究较少的化合物相比，风险可能较低。然而，必须注意的是，没有伤害报告并不能保证安全，因为长期影响和罕见的不良反应可能仍然未知。 |
| [25I-NBOMe](/药物/25I-NBOMe.md) | “PubMed 查询显示自 Suzuki 等人 2015 年发表综述以来提及 NBOMe 摄入的情况。在 2014 年 10 月至 2021 年 6 月 8 日发表的英语出版物中，有 16 份病例报告和 6 个病例系列，详细记录了总共 42 例 NBOMe 中毒病例。”[\[3\]](#cite_note-25I-NBOMe-3) |
| [5F-ADB](/药物/5F-ADB.md) | 2015 年至 2017 年间在欧洲造成 25 人死亡。[\[4\]](#cite_note-4) |
| [酒精](/药物/酒精.md) | 国际癌症研究机构（IARC）1 类致癌物。 |
| [苯海拉明](/药物/苯海拉明.md)（为了[苯海拉明挑战](https://en.wikipedia.org/wiki/Benadryl_challenge)） | 一种危险的社交媒体趋势，涉及苯海拉明过量服用。 |
| [蟾毒色胺类物质](/文档/药物分类/蟾毒色胺类物质.md)（*蟾蜍* 静修） | 由于动物剥削而存在伦理问题。合成的[蟾毒色胺](/药物/蟾毒色胺.md)和 [5-MeO-DMT](/药物/5-MeO-DMT.md) 是替代品。此外，尽管某些蟾蜍是获取分泌物（蟾蜍静修）的常见目标，但它们仍携带具有心脏毒性的蟾毒色胺类物质，这已与死亡事件有关。其他蟾蜍中蟾毒色胺类物质的影响尚不清楚。 |
| [可卡因](/药物/可卡因.md) | 可卡因通常掺有[左旋咪唑](https://en.wikipedia.org/wiki/Levamisole)，这可能导致一系列严重的并发症，包括可卡因和左旋咪唑诱导的[血管炎](https://en.wikipedia.org/wiki/Vasculitis) (CLIV)、[\[5\]](#cite_note-5) [左旋咪唑诱导的坏死综合征](https://en.wikipedia.org/wiki/Levamisole_induced_necrosis_syndrome) (LINES)、[\[6\]](#cite_note-6) 和可卡因/左旋咪唑相关的[自身免疫综合征](https://en.wikipedia.org/wiki/Autoimmune_disease) (CLAAS)。[\[7\]](#cite_note-7) |
| [Desmethylprodine](/药物/Desmethylprodine.md) | 在制造 MPPP 的过程中可能会意外产生 MPTP。1-甲基-4-苯基吡啶 (MPP+)，即 MPTP 的代谢物，会导致类似于帕金森病的不可逆症状迅速发作。[\[8\]](#cite_note-8)[\[9\]](#cite_note-Surprising_Clue_to_Parkinsons-9) |
| [地索吗啡](/药物/地索吗啡.md) | 不纯的 desmethylprodine，被称为“krokodil”（鳄鱼），与严重的组织损伤和毒性有关。 |
| [Ephylone](/药物/Ephylone.md) | 据美国缉毒局（DEA）报告，2015 年至 2018 年间，Ephylone 与美国约 151 例致命的[药物过量](/文档/药物过量.md)有关。[\[10\]](#cite_note-10) |
| [芬太尼](/药物/芬太尼.md)和芬太尼类似物 | 在美国，芬太尼及其类似物在 2017 年导致超过 29,000 人死亡，较前四年大幅增加。[\[11\]](#cite_note-NYT2017-11)[\[12\]](#cite_note-NIDA-deaths-12) |
| [灰色死亡](https://en.wikipedia.org/wiki/Gray_death) | 具有不可预测效力的危险阿片类药物混合物。例如：在阿根廷发现一种被误导性地称为 [2C-B](/药物/2C-B.md) 的混合物中含有芬太尼。[\[13\]](#cite_note-13) |

| [Kambo](/药物/Kambo.md)（静修） | 由于动物剥削而存在伦理问题。此外，这是一种来自青蛙分泌物的潜在危险传统药物。Kambo 已导致[数例死亡](https://en.wikipedia.org/wiki/Kambo_(drug)#Notable_deaths)。 |
| [Lean (紫水)](https://en.wikipedia.org/wiki/Lean_(drug)) ([可待因](/药物/可待因.md)/异丙嗪) | 可待因止咳糖浆和苏打水的混合物，有成瘾风险。Lean 已导致多人死亡，包括一些[著名事件](https://en.wikipedia.org/wiki/Lean_(drug)#Notable_incidents_of_use)。 |

| [MDMB-FUBINACA](/药物/MDMB-FUBINACA.md) | 具有严重副作用的[合成大麻素类物质](/文档/药物分类/合成大麻素类物质.md)。截至 2015 年 3 月，已有 40 例死亡是由于 MDMB-FUBINACA 中毒引起的。[\[16\]](#cite_note-16)[\[17\]](#cite_note-17) |
| [MDPV](/药物/MDPV.md) | 具有高成瘾潜力和危险副作用的[兴奋剂](/文档/药物分类/兴奋剂.md)。据九个欧洲国家报告，2009 年 9 月至 2013 年 8 月期间，共有 107 例非致命中毒和 99 例经分析确认的与 MDPV 相关的死亡。[\[18\]](#cite_note-EMCDDA-18) |

## 按给药途径分类

### 注射

#### 非注射用剂型

| 物质 | 避免原因 |
| --- | --- |
| 药片 | **切勿注射药片，这一点极其重要**，因为它们含有药片粘合剂，可能会聚集并导致血栓和其他严重的身体伤害。使用微米过滤器和提取药物活性成分可以降低这种风险，但不建议注射任何类型的药片。这个问题最常见于含有[羟考酮](/药物/羟考酮.md)的药片。当这些药片被压碎时，看起来可能适合注射，但它们可能含有药片粘合剂和其他材料，这些材料会堵塞静脉并在血液中造成阻塞，导致循环问题和肾脏并发症。[\[27\]](#cite_note-27)<br>压碎药片中的滑石粉会导致：<br>**[肺气肿](https://en.wikipedia.org/wiki/Emphysema)** - 肺气肿是一种下呼吸道疾病。静脉注射[哌甲酯](/药物/哌甲酯.md)（通常以利他林品牌销售，广泛用作治疗注意力缺陷多动障碍的兴奋剂药物）会导致被称为[利他林肺](https://en.wikipedia.org/wiki/Emphysema#Ritalin_lung)的肺气肿变化。**[肺滑石粉尘肺](https://en.wikipedia.org/wiki/Pulmonary_talcosis)** - 肺滑石粉尘肺是由滑石粉引起的肺部疾病。注射压碎药片中的滑石粉与静脉药物使用者中的肺滑石粉尘肺有关。[\[28\]](#cite_note-pmid6655726-28) |
| 透皮贴剂 | 主要原因：[剂型\#药物储库中物质分布不均](/文档/给药途径.md)<br>芬太尼贴剂设计用于透皮使用，而非静脉注射。因此，在制造过程中应用于透皮贴剂的芬太尼溶液并不保证分布均匀。例如，顶部角落可能含有很少的芬太尼，而贴剂中心可能含有明显更高的浓度。 |
| 电子烟液 | 病例报告：静脉注射电子尼古丁输送系统补充液致死。[\[29\]](#cite_note-29) |

#### 通用

| 物质 | 避免原因 |
| --- | --- |
| 任何水溶性差的物质 | 避免加热水溶性差的物质溶液（例如[育亨宾](/药物/育亨宾.md)和[咖啡因](/药物/咖啡因.md)），因为一旦注射到体内，它们会恢复到固体状态。如有需要，用更多的水稀释它们并使用更大容量的注射器。 |
| 任何强腐蚀性/酸性物质 | 避免在未先中和 pH 值的情况下注射具有显著低或高 pH 值的物质。腐蚀性物质的例子：[4-FA](/药物/4-FA.md) 或[菲尼布特](/药物/菲尼布特.md)盐酸盐。 |
| 任何植物和真菌的自制提取物 | [大麻](/药物/大麻.md)：[静脉大麻综合征](https://en.wikipedia.org/wiki/Intravenous_marijuana_syndrome)是一种独特的短期临床综合征，与静脉注射经棉布过滤的煮沸大麻肉汤有关。致幻菇茶：一名男子在静脉注射[致幻菇](/药物/致幻菇.md)茶后因真菌感染险些丧命。[\[30\]](#cite_note-30) |

#### 不可注射的物质

| 物质 | 避免原因 |
| --- | --- |
| [黑焦油海洛因](/药物/海洛因.md) | 注射黑焦油海洛因的人可能会增加患危及生命的细菌感染的风险，特别是坏死性软组织感染。[\[31\]](#cite_note-31) “皮下注射”（skin-popping）的做法易导致坏死性筋膜炎或由*产气荚膜梭菌*引起的坏死性蜂窝织炎，而深部肌肉注射易导致坏死性肌炎。[\[32\]](#cite_note-32)<br>焦油海洛因注射也可能与*肉毒杆菌*感染有关。 |
| [可待因](/药物/可待因.md) | 可注射可待因仅供皮下或肌肉注射；静脉注射是禁忌的，因为这可能导致非免疫性肥大细胞脱颗粒并导致类过敏反应。 |
| [Ethchlorvynol](/药物/Ethchlorvynol.md) | Ethchlorvynol 不适合静脉注射，以这种方式使用时可能会发生严重伤害（包括因血管损伤导致的截肢）或死亡。[\[33\]](#cite_note-33) |
| 眼药水 | 含有药物（例如仅用于眼部给药的谵妄剂托吡卡胺）的眼药水在注射时可能是致命的。使用不含药物的眼药水（针具交换计划经常分发这类眼药水）作为溶解物质的溶液。 |
| [羟嗪](/药物/羟嗪.md)（商品名 Atarax 和 Vistaril） | “不建议通过 SC（皮下）、IA（关节内）或 IV（静脉）途径给药，因为通过这些途径给药后曾发生不良反应。”[\[34\]](#cite_note-34)[\[35\]](#cite_note-35) |

### 鼻吸

## 按静修中心分类

关于致幻剂（entheogen）静修的说明：虽然许多人通过有组织的仪式寻求变革性体验，但许多人没有意识到在家订购精神活性物质的选择。死藤水成分、[5-MeO-DMT](/药物/5-MeO-DMT.md) 和 yopo 种子（[蟾毒色胺](/药物/蟾毒色胺.md)）通常可以合法地用于个人用途。然而，许多人，包括那些前往全球各地参加静修的人，仍然没有意识到这一选择，因为他们已经内化了萨满对这种体验至关重要的信念。如果一个人做好了心理准备，从非常低的剂量开始，逐渐增加剂量，并有一个值得信赖的[旅程保姆](/文档/旅程保姆.md)，家庭使用也可以是安全的。

对于通过专业指导的迷幻静修寻求变革性体验的个人来说，至关重要的是要意识到“塑料萨满”（plastic shamans）现象。这个术语指的是那些虚假声称自己是精神领袖或治疗师的人，他们通常为了个人利益、自我膨胀或经济利润而利用他们几乎没有或根本没有真正联系的土著传统。在考虑静修时，必须彻底研究促进者的背景和资格。此外，请记住，为了这些仪式而旅行会对环境产生重大影响，导致碳排放和生态压力。
