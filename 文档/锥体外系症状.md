# 锥体外系症状

[◀返回](index.md)

| ![Four\_types\_of\_EPS.png](/文件/Four_types_of_EPS.png) |
| ------------------------------------------------------ |
| 一张用插图快速总结四种锥体外系症状的图片。                                  |

**锥体外系症状**（EPS），又称**锥体外系副作用**（EPSE），是药物诱发的运动障碍，包括急性症状和迟发性/慢性症状。它们主要由多巴胺在多巴胺能受体处受到阻断所引起。

## 类型及其症状

### 急性肌张力障碍

| ![Medication-induced\_dystonia.JPG](/文件/Medication-induced_dystonia.JPG) |
| ------------------------------------------------------------------------ |
| 一名因多巴胺阻断药物而出现急性肌张力障碍的患者。                                                 |

急性肌张力障碍是一种疼痛且通常较严重的锥体外系症状，发生在[抗多巴胺能](/文档/多巴胺.md)药物给药后不久（不到一天内）。

头部、颈部和四肢可能出现[持续性肌肉收缩](/药效/肌肉痉挛.md)，并可能导致舌头僵硬、背部弓起或颈部扭转。肌张力障碍还包括肌肉运动能力不足和肌张力低下。

急性肌张力障碍的治疗通常使用抗胆碱能药物和/或苯二氮卓类物质。

### 静坐不能

静坐不能的特征包括烦躁不安、需要移动、不断坐下又站起、来回踱步等。静坐不能的治疗方法包括抗胆碱能药物、β受体阻滞剂，或[苯二氮卓类物质](/文档/药物分类/苯二氮卓类物质.md)。静坐不能通常会在某个药物治疗方案进行期间的某个时间点出现。

### 假性帕金森综合征

假性帕金森综合征通常比静坐不能出现得更晚。它常可通过以下表现被注意到：

* 双手以“搓丸样”姿势震颤；
* 弯腰驼背的姿势；
* 动作迟缓；
* 拖曳式、小步行走；

等等。其他表现还包括：当此人转身时，必须非常缓慢地进行；有时在没有支撑的情况下，无法改变姿势，例如从站立变为坐下。

### 迟发性运动障碍

这种障碍会在很长一段时间内发展，并可通过以下体征被注意到：不自主咀嚼、咂嘴或吸吮动作、抽动，和/或四肢不自主运动，且这些运动通常呈急促抽动样。治疗方式是让患者进入多巴胺能药物治疗方案，例如使用缬苯那嗪、四苯喹嗪、罗匹尼罗或普拉克索。

## 原因

锥体外系症状最常由典型抗精神病药引起，这类药物会[拮抗](/文档/受体拮抗剂.md)[多巴胺](/文档/多巴胺.md)D2[受体](/文档/受体.md)。

与EPS相关的最常见典型[抗精神病药](/文档/抗精神病药.md)是[氟哌啶醇](/药物/氟哌啶醇.md)和氟奋乃静。

其他抗多巴胺能药物，例如[止吐](/药效/恶心抑制.md)药甲氧氯普胺，也可能导致锥体外系副作用。

短期和长期使用抗抑郁药，例如选择性血清素再摄取抑制剂（[SSRI](/文档/SSRI.md)）、血清素-去甲肾上腺素再摄取抑制剂（SNRI）和去甲肾上腺素-多巴胺再摄取抑制剂（NDRI），也曾导致锥体外系症状。具体而言，度洛西汀、舍曲林、艾司西酞普兰、氟西汀和安非他酮都曾与锥体外系症状的诱发有关。

## 治疗

| ![Akineton\_Advertisement%2C\_1966.jpg](/文件/Akineton_Advertisement%2C_1966.jpg) |
| ------------------------------------------------------------------------------- |
| 安坦宁（比哌立登）的广告；该药用于假性帕金森综合征。                                                      |

[抗胆碱能](/文档/乙酰胆碱.md)药物用于控制神经阻滞剂诱发的EPS，不过静坐不能可能需要β受体阻滞剂，甚至[苯二氮卓类物质](/文档/药物分类/苯二氮卓类物质.md)。如果EPS由[抗精神病药](/文档/抗精神病药.md)诱发，则可通过剂量[减量调整](/文档/减量戒断法.md)，或换用非典型[抗精神病药](/文档/抗精神病药.md)来减轻EPS，例如阿立哌唑、齐拉西酮、[喹硫平](/药物/喹硫平.md)、奥氮平、[利培酮](/药物/利培酮.md)或氯氮平。

常用于EPS的药物包括[抗胆碱能](/文档/乙酰胆碱.md)药物，例如苯托品（Cogentin）、比哌立登（Akineton）、[苯海拉明](/药物/苯海拉明.md)（Benadryl）和苯海索（Artane）。另一种常见治疗方案包括[多巴胺](/文档/多巴胺.md)[激动剂](/文档/受体激动剂.md)类药物，例如普拉克索和罗匹尼罗。这些药物可逆转由[抗精神病药](/文档/抗精神病药.md)或其他直接或间接抑制[多巴胺能](/文档/多巴胺.md)神经传递的药物所引起的锥体外系副作用症状。
