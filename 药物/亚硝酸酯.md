# Poppers (亚硝酸酯)



[![](/文件/HOpoppers.jpg)](/文件/HOpoppers.jpg)

[](/wiki/File:HOpoppers.jpg "Enlarge")

各类 Poppers

**亚硝酸酯**（也称为 **Poppers**）是一类挥发性液体物质，包括**亚硝酸戊酯**、**亚硝酸丁酯**、**亚硝酸环己酯**、**亚硝酸乙酯**、**亚硝酸己酯**、**亚硝酸异丁酯**、**亚硝酸异丙酯**和**亚硝酸甲酯**（更多详细信息请参见[本章节](#亚硝酸酯)）。

亚硝酸酯是挥发性物质，其烟雾被吸入用于娱乐目的，特别是作为性活动前的准备。[\[1\]](#cite_note-MerckManuals-1)

它们以产生强烈但短促的肌肉松弛效果而闻名，这种效果在大约 15 秒后开始，持续长达 3 分钟。

最常见的 Poppers 使用方法是打开瓶子，将其放在鼻孔下吸入。其他使用方法，如[口服](/文档/给药途径.md#口服)极其危险，可能导致死亡或昏迷。值得注意的是，亚硝酸酯接触皮肤会引起灼伤，且极易燃。

亚硝酸酯
--------------

以下列出了不同亚硝酸酯的沸点，沸点越低，挥发性越强。

| 亚硝酸酯名称 | 英文名称 | 沸点 (°C, 760 mmHg) | 挥发性 (感知到的“强度”) |
| :--- | :--- | :--- | :--- |
| **亚硝酸甲酯** | Methyl nitrite | -12.9±3.0 °C | 极易挥发 (强)[\[2\]](#cite_note-2) |
| **亚硝酸乙酯** | Ethyl nitrite | 20.0±3.0 °C | 极易挥发 (强)[\[3\]](#cite_note-3) |
| **亚硝酸异丙酯** | Isopropyl nitrite | 39.0±9.0 °C | 极易挥发 (强)[\[4\]](#cite_note-4) |
| **亚硝酸异丁酯** | Isobutyl nitrite | 70.5±9.0 °C | 中等挥发[\[5\]](#cite_note-5) |
| **亚硝酸丁酯** | Butyl nitrite | 78.8±3.0 °C | 中等挥发[\[6\]](#cite_note-6) |
| **亚硝酸戊酯** | Amyl nitrite | 99.1±9.0 °C | 缓慢挥发 (弱)[\[7\]](#cite_note-7) |
| **亚硝酸己酯** | Hexyl nitrite | 129.6±3.0 °C | 缓慢挥发 (弱)[\[9\]](#cite_note-9) |
| **亚硝酸环己酯** | Cyclohexyl nitrite | 138.7±7.0 °C | 缓慢挥发 (弱)[\[8\]](#cite_note-8) |

亚硝酸戊酯是目前销售最广泛的亚硝酸酯。由于欧盟在 2007 年禁止了亚硝酸异丁酯，亚硝酸异丙酯随之流行起来。Poppers 是 20 世纪 70 年代至 80 年代迪斯科俱乐部文化的一部分。20 世纪 90 年代的锐舞场景进一步普及了它们的使用。[\[10\]](#cite_note-10) 如今，它们主要在性用品商店以小瓶装或胶囊瓶的形式出售。

化学
---------

[![200px-Nitrite-ester-2D.png](/文件/200px-Nitrite-ester-2D.png)](/文件/200px-Nitrite-ester-2D.png)

[](/wiki/File:200px-Nitrite-ester-2D.png "Enlarge")

Poppers 是一种被称为**亚硝酸酯**的化学物质。这些化学化合物的结构为 R–O-N=O。用更正式的术语来说，它们是亚硝酸的烷基酯。亚硝酸酯属于**吸入剂**，这意味着它们具有非常低的蒸汽点，在室温下几乎立即挥发到空气中。有机亚硝酸酯由醇和亚硝酸钠在硫酸溶液中制备。它们在静置时会缓慢分解。分解产物为氮氧化物、水和醇，以及醛的聚合产物。

------------

| --- | --- |

亚硝酸酯的药理学尚未完全被理解，但其发挥作用的主要机制被认为是还原为一氧化氮，一氧化氮在哺乳动物体内作为气体递质（一种气体神经递质）起作用，导致平滑肌松弛。静脉和动脉主要由血管平滑肌组成，通过释放一氧化氮，Poppers 引起静脉和动脉扩张，降低血压并使血液以更快的速度泵送至全身。[\[11\]](#cite_note-11) 阴道和肛门括约肌平滑肌组织的松弛是用户在性活动中使用 Poppers 的一个原因，目的是为了更轻松地进行插入。

+ **[血压降低](/药效/血压降低.md)** - 由于亚硝酸酯具有血管扩张作用，它们通常会大幅降低血压。[\[13\]](#cite_note-IsraelstamLambert2017-13)
+ **[心率增快](/药效/心率增快.md)**
+ **[肌肉松弛](/药效/肌肉松弛.md)** - 吸入亚硝酸酯会放松全身的平滑肌，并且在放松肛门括约肌方面特别有效。它们在 LGBT 群体中常因此目的被使用。[\[13\]](#cite_note-IsraelstamLambert2017-13)
+ **[躯体欣快感](/药效/躯体欣快感.md)**
+ **[血管扩张](/药效/血管扩张.md)** - Poppers 的血管扩张作用强烈而突然。这会导致血液涌向大脑的感觉，伴随着全身发热的感觉（特别是面部/头部区域）。[\[13\]](#cite_note-IsraelstamLambert2017-13)

毒性与危害可能
---------------------------

[![](/文件/Harmchart.png)](/文件/Harmchart.png)

[](/wiki/File:Harmchart.png "Enlarge")

显示亚硝酸酯相对躯体伤害、社会危害和依赖性的雷达图[\[14\]](#cite_note-14)

2005 年的默克诊疗手册报告称，几乎没有证据表明吸入亚硝酸酯有重大危害。[\[1\]](#cite_note-MerckManuals-1) 英国政府顾问根据卫生专业人员对个人和社会的危害意见制定的一项药物危害性研究和排名将亚硝酸酯列为危害较小的物质，相比之下包括[酒精](/药物/酒精.md)和[烟草](/药物/烟草.md)。[\[15\]](#cite_note-15) 使用 Poppers 的主要风险是如果溅到皮肤上会造成化学灼伤。

所有形式的亚硝酸酯都极易燃，如果正在吸烟或附近有蜡烛，不应使用。

在英国和法国最近的病例报告中，亚硝酸酯是导致少数黄斑病变（眼部损伤）病例的一个可能且罕见的担忧原因。[\[16\]](#cite_note-16) 一些研究得出结论，对于某些习惯性使用 Poppers 的用户，至少存在暂时性视网膜损伤的风险增加；在给《新英格兰医学杂志》的一封信中，一位眼科医生描述了四个案例，其中娱乐性使用 Poppers 的用户经历了暂时的视力变化。[\[17\]](#cite_note-17) 另一项研究描述了六名习惯性 Poppers 用户的中心凹（日光视觉）损伤。[\[18\]](#cite_note-18) 2014 年，视光师和眼科医生报告称，他们注意到英国长期使用 Poppers 的用户视力丧失有所增加，这与 2006 年亚硝酸异丙酯取代亚硝酸异丁酯有关。[\[19\]](#cite_note-19)[\[20\]](#cite_note-20)

##### 亚硝酸异丁酯中毒治疗

已知亚硝酸异丁酯会导致高铁血红蛋白血症。[\[21\]](#cite_note-21) 严重的高铁血红蛋白血症可以用亚甲蓝治疗。[\[22\]](#cite_note-22)

### 致死剂量

使用 Poppers 唯一始终安全的[给药途径](/文档/给药途径.md)是吸入。口服液体可能致命，达到致死剂量。[\[23\]](#cite_note-23)[\[24\]](#cite_note-Hagan2007-24) 通过常规使用 Poppers 几乎不可能过量。

通过口服（而非吸入）过量使用可能导致发绀、意识丧失、昏迷甚至死亡。[\[25\]](#cite_note-Medsafe-25)[\[26\]](#cite_note-Dixon-1981-26)[\[27\]](#cite_note-Pruijm-2002-27)[\[28\]](#cite_note-28)[\[29\]](#cite_note-29) 意外吸入（呛入肺部）亚硝酸戊酯或亚硝酸丁酯可能导致脂质性肺炎。[\[30\]](#cite_note-Hagan--30) 亚甲蓝是治疗与使用 Poppers 相关的高铁血红蛋白血症的药物。[\[24\]](#cite_note-Hagan2007-24)[\[31\]](#cite_note-31)[\[32\]](#cite_note-32)[\[33\]](#cite_note-33) 意外吸入亚硝酸戊酯或亚硝酸丁酯可能导致脂质性肺炎的发展。[\[24\]](#cite_note-Hagan2007-24)

强烈建议在使用此类物质时采取[伤害减少措施](/文档/负责任的用药索引页.md)。

### 耐药性与成瘾性

使用 Poppers 不会产生真正的耐药性。它们被认为几乎没有或没有成瘾潜力。

### 口服危险

摄入亚硝酸酯会导致高铁血红蛋白血症，目前尚未排除吸入导致这种病症的可能性。[\[34\]](#cite_note-pmid3430141-34)

### 危险的相互作用

亚硝酸酯与其他[血管扩张](/药效/血管扩张.md)药物如西地那非（万艾可，瑞伐季奥）、伐地那非（艾力达）、他达拉非（希爱力，Adcirca）和硝酸甘油相互作用，会导致血压严重下降，从而导致昏厥。[\[35\]](#cite_note-35)

法律地位
------------

* **加拿大**：加拿大已禁止销售任何配方的 Poppers。虽然不被视为麻醉品，拥有或使用也不违法，但它们被视为药物。未经授权的销售现在可被处以罚款和监禁。[\[36\]](#cite_note-36)
* **欧盟**：自 2007 年以来，欧洲销售含有亚硝酸异丙酯的重配方 Poppers；亚硝酸异丁酯被禁止，因为它被认为具有致癌性。[\[37\]](#cite_note-37)
* **法国**：在法国，自 1990 年以来，出于对消费者构成危险的理由，已禁止销售含有亚硝酸丁酯、亚硝酸戊酯或其异构体的产品。[\[38\]](#cite_note-38) 2007 年，政府将此禁令扩大到所有未获准作为药物销售的亚硝酸酯。[\[39\]](#cite_note-39)
* **德国**：Poppers 不受 BtMG（麻醉品法）或 NpSG（新精神活性物质法）管制。[\[40\]](#cite_note-40)[\[41\]](#cite_note-41) 根据 §2 AMG，只要它们不是为了人类消费而销售，它们就是合法的。[\[42\]](#cite_note-42)
* **英国**：在英国，Poppers 在同性恋俱乐部/酒吧、性用品商店、烟具店、互联网和市场上出售。根据 1968 年药品法，将其作为供人类消费的产品进行宣传销售是违法的，为了绕过这一点，它们通常作为除臭剂出售。
* **美国**：在美国，除亚硝酸戊酯以外的含有亚硝酸酯的 Poppers 很容易买到。有时它们作为录像磁头清洁剂、指甲油去除剂或房间除臭剂出售。它们没有恢复到 20 世纪 70 年代的流行程度。
