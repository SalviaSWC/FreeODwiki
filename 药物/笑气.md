# 笑气

笑气是一种解离剂，分子式为N2O。

**冷知识**：笑气并不属于管制精神药品，因此本身并不符合"吸毒"的定义。然而，如果使用者承认自己是以娱乐为目的使用笑气，将会被警察认定为吸毒，并承受相应的后果。

| 名称 | 笑气 | 
|------|-----|
| 分子式 | N2O |
| CAS号 | 10024-97-2 |
|管制状态|危险化学品|
|药效类型|解离剂|

## 药效

吸入笑气可以产生短暂的解离药效，药效的持续时间约为1分钟。

据报告，联用笑气与其他解离剂(如右美沙芬)可以产生极强的解离效果，这可能是因为两者以不同的方式拮抗NMDA受体导致的。

## 危害及反制方式

笑气是一种氧化剂，吸入后，会氧化人体内的钴离子，造成维生素B12缺乏。维生素B12缺乏会导致神经损伤，严重的可以导致瘫痪。因此，笑气的使用应当严格限制频率。在两次相邻的使用之间间隔至少2周。使用笑气后应当立即服用维生素B12补充剂。

吸入笑气时未能采用正确的吸入方式：

 - 对着打发气瓶直接吸入笑气，会造成冻伤。这是因为笑气瓶中的高压笑气快速膨胀，根据热力学第一定律`U = Q + W`，气体的温度会迅速降低。
 - 直接带着笑气面罩吸入会导致缺氧。请勿使用这种方式使用笑气！

吸入笑气前，应该提前将气体通过打发气瓶装入气球，待其恢复室温后之后吸入气球内的气体，以免出现上述的危险情况。
