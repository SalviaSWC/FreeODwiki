**L-茶氨酸**（也称为**茶氨酸**、**L-γ-谷氨酰乙酰胺**和**N5-乙基-L-谷氨酰胺**）是谷氨酸和谷氨酰胺氨基酸的类似物。它于1949年作为绿茶的一种成分被发现，并于1950年从玉露叶中分离出来，玉露叶含有高含量的茶氨酸。[\[2\]](#cite_note-2)

如果不带前缀出现“茶氨酸”这一名称，通常理解为暗示L-[对映异构体](/文档/药物分类/home.md)，这是新鲜茶叶和一些（但不是所有）膳食补充剂中存在的形式。相反的D-对映异构体的药理特性研究较少，但存在于外消旋化学制剂中，并且在一些研究过的茶氨酸补充剂中大量存在。

这种化合物被用作[益智药](/文档/药物分类/益智药.md)，具有镇静和放松的特性。它经常与[咖啡因](./咖啡因.md)结合使用，因为它已被证明可以减轻咖啡因的负面影响，如焦虑、[血压升高](/药效/血压升高.md)和睡眠质量下降，同时可能改善其积极方面。[\[3\]](#cite_note-3)[\[4\]](#cite_note-4)[\[5\]](#cite_note-5)[\[6\]](#cite_note-Kahathuduwa2017-6) 其增强注意力的能力已被反复验证。[\[6\]](#cite_note-Kahathuduwa2017-6)[\[7\]](#cite_note-Owen2008-7)[\[8\]](#cite_note-Haskell2008-8)[\[9\]](#cite_note-9)[\[10\]](#cite_note-10)[\[11\]](#cite_note-11)[\[12\]](#cite_note-12)

最近一项关于茶氨酸和咖啡因影响的系统评价证实，这种组合似乎能改善注意力的各个方面。[\[13\]](#cite_note-13) L-茶氨酸和咖啡因的组合可能比单用咖啡因更能提高注意力。[\[7\]](#cite_note-Owen2008-7)[\[8\]](#cite_note-Haskell2008-8)

化学
---------

茶氨酸，或N-乙基-L-谷氨酰胺，是L-谷氨酰胺的氨基酸类似物。其结构由称为戊酸的五碳直链羧酸组成，该羧酸在R2处与氨基键合，在R5处有额外的酮基。同样在戊酸基团的R5处被取代的是连接在其氨基成分上的乙氨基链。茶氨酸通常指左旋对映异构体，其已有详细记载，而相对未被研究的是右旋对映异构体。

药理学
------------

茶氨酸在结构上类似于兴奋性[神经递质](/文档/科学信息索引页.md)谷氨酸，因此可以结合到谷氨酸[受体](/文档/受体激动剂.md)上，尽管相比之下亲和力要低得多。具体来说，它结合到离子型谷氨酸受体，包括AMPA和红藻氨酸受体，并在较小程度上结合到NMDA受体。[\[14\]](#cite_note-Nathan2006-14)[\[15\]](#cite_note-15)[\[16\]](#cite_note-16)[\[17\]](#cite_note-Kakuda2002-17) 它作为前两个位点的[受体拮抗剂](/文档/受体拮抗剂.md)[\[17\]](#cite_note-Kakuda2002-17)和后一个位点的[受体激动剂](/文档/受体激动剂.md)起作用。[\[18\]](#cite_note-Wakabayashi2012-18) 此外，它抑制谷氨酰胺转运蛋白和谷氨酸转运蛋白，从而作为谷氨酰胺和谷氨酸的[神经递质再摄取抑制剂](/文档/神经递质再摄取抑制剂.md)起作用。[\[17\]](#cite_note-Kakuda2002-17)[\[19\]](#cite_note-19)[\[20\]](#cite_note-20)

茶氨酸增加大脑各个区域的[多巴胺](/文档/多巴胺.md)、GABA和甘氨酸水平。[\[14\]](#cite_note-Nathan2006-14)[\[18\]](#cite_note-Wakabayashi2012-18)[\[21\]](#cite_note-21)[\[22\]](#cite_note-Yokogoshi1998-22) 它也以一种在科学界仍有争议的方式影响[血清素](/文档/血清素.md)，不同的研究显示在使用类似的实验方案时，脑内血清素水平有增加也有减少。[\[22\]](#cite_note-Yokogoshi1998-22)[\[23\]](#cite_note-23)

神经递质水平的这些各种变化促成了茶氨酸的镇静和益智特性。

### **身体效应** ![Child.png](/文件/Child.png)

*     + **[镇静](/药效/镇静.md)** \& **[刺激](/药效/刺激.md)** - 茶氨酸在[轻微](/文档/药物剂量分类.md)到[中等](/文档/药物剂量分类.md)剂量下有轻微的刺激作用，但在约300mg起的[强烈](/文档/药物剂量分类.md)剂量下变得具有镇静作用。
    + **[肌肉松弛](/药效/肌肉松弛.md)** - 在[严重](/文档/药物剂量分类.md)剂量下，茶氨酸诱导的肌肉松弛与[苯二氮卓类物质](/文档/药物分类/苯二氮卓类物质.md)产生的松弛隐约可比。
    + **自发性躯体感觉** - 在较高剂量下，茶氨酸可以产生一种遍布全身的温暖发光的感觉。
    + **[头痛](/药效/头痛.md)** - 头痛通常只在远高于200mg的高剂量下才有报道。

### **认知效应** ![User.png](/文件/User.png)

*     + **焦虑抑制** - 茶氨酸产生抗焦虑作用，相当于轻微剂量的苯二氮卓类药物，但没有伴随的[去抑制](/药效/home.md)作用。
    + **正念** - 一些用户报告说，茶氨酸可以产生独特的正念状态。
    + **记忆增强**[\[24\]](#cite_note-24) - 茶氨酸已被证明可以改善轻度认知障碍受试者的记忆和注意力。
    + **成瘾抑制** - 一些轶事报告表明，茶氨酸可能减轻[阿片类药物戒断](/文档/药物戒断反应.md)的一些影响，以及GABA能物质如[酒精](./酒精.md)和[苯二氮卓类物质](/文档/药物分类/苯二氮卓类物质.md)的戒断反应。
    + **思维组织**
    + **分析增强**
    + **专注增强**
    + **动力增强**

毒性和危害潜力
---------------------------

茶氨酸不具成瘾性，不为人知会导致脑损伤，并且相对于剂量具有极低的毒性。急性茶氨酸暴露相关的身体副作用相对较少。各种研究表明，在合理的剂量和谨慎的背景下，它不会带来负面的认知、精神或有毒的身体后果。

强烈建议在使用该物质时采取[伤害减少措施](/文档/负责任的用药索引页.md)。

### 耐受性和成瘾潜力

茶氨酸不成瘾。

对茶氨酸效应的耐受性会在长期和重复使用后建立。在建立明显的耐受性后，大约需要5天时间耐受性减半，10天恢复到基线。茶氨酸与目前已知的其他化合物不存在交叉耐受，这意味着在服用茶氨酸后，所有其他精神活性化合物的效果不会减弱。
