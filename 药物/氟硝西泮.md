**氟硝西泮**（商品名 **Rohypnol**）是一种[苯二氮卓类](./../文档/药物分类/苯二氮卓类物质.md)[抑制剂](./../文档/药物分类/抑制剂.md)，[给药](./../文档/给药途径.md)后主要产生催眠和遗忘效应，但也具有抗焦虑、抗惊厥和[镇静](./../药效/镇静.md)作用。[\[2\]](#cite_note-Mandroili2008-2)

氟硝西泮于1972年由罗氏公司（Hoffmann-La Roche）首次合成。在一些国家，它被用于短期治疗[失眠](./../药效/镇静.md)（注：原文insomnia通常对应药效中的镇静或相关睡眠文档）和作为术前镇静剂。按重量计算，它的效力约为[地西泮](./../药物/地西泮.md)的10倍。

用户应注意，对于长期规律使用的个人来说，[苯二氮卓类药物的突然停药](./../文档/药物戒断反应.md)可能是危险的或危及生命的，有时会导致癫痫发作或死亡。[\[3\]](#cite_note-Lann2009-3) 强烈建议通过长期逐渐减少每天的服用量来减少剂量，而不是突然停止。[\[4\]](#cite_note-4)

历史与文化
-------------------

氟硝西泮是在罗氏公司（Roche）作为Leo Sternbach领导的苯二氮卓类药物研究的一部分被发现的。它于1974年首次上市，并于1975年以Rohypnol的名称进入欧洲商业市场。在1980年代，它开始在其他国家上市。[\[5\]](#cite_note-Wikipedia2022-5)

它可能作为约会强奸药Rohypnol（街头名称为“roofie”）更为人所知。1998年，由于该药的滥用和娱乐用途，罗氏公司修改了其1mg片剂，使其溶解性降低，并添加了蓝色染料以便于在饮料中检测。[\[6\]](#cite_note-6)

在瑞典的研究中，氟硝西泮是自杀中最常用的第二大药物，约占案例的16%。在一项对1587例死亡的回顾性瑞典研究中，有159例发现了苯二氮卓类药物。在涉及苯二氮卓类药物的自杀中，与自然死亡相比，氟硝西泮和[硝西泮](./../药物/硝西泮.md)（注：原文为Nitrazepam，若无对应文件则翻译，此处假设硝西泮暂无文件或指向抑制剂类，但根据上下文通常指向同类药物页面，若无特定页面保留翻译）的浓度显著更高。[\[5\]](#cite_note-Wikipedia2022-5)

化学
---------

氟硝西泮被归类为硝基苯二氮卓类药物。它是硝西泮的氟化N-甲基衍生物。其他硝基苯二氮卓类药物包括硝西泮（母体化合物）、尼美西泮（甲氨基衍生物）和氯硝西泮（2'-氯化衍生物）。[\[7\]](#cite_note-7)

药理学
------------

苯二氮卓类药物通过结合苯二氮卓受体位点并通过作用于其[受体](./../文档/受体激动剂.md)，放大神经递质[γ-氨基丁酸 (GABA)](./../文档/药物分类/抑制剂.md)的效率和效应，从而产生多种效应。[\[8\]](#cite_note-8) 由于该位点是大脑中最多产的抑制性受体组，其调节导致了氟硝西泮对神经系统的[镇静](./../药效/镇静.md)（或[镇静效应](./../药效/镇静.md)）。

苯二氮卓类药物的抗惊厥特性可能部分或完全归因于结合电压依赖性钠通道，而不是苯二氮卓受体。[\[9\]](#cite_note-9)

众所周知，氟硝西泮在足够剂量下会引起顺行性遗忘（个人无法回忆起受药物影响期间经历的某些事件）。如果氟硝西泮被用于协助实施性侵犯，这种效应可能特别危险，因为受害者可能无法清楚地回忆起侵犯过程、攻击者或侵犯周围的事件。[\[5\]](#cite_note-Wikipedia2022-5)

口服摄入的氟硝西泮有80%被吸收，但栓剂形式的生物利用度接近50%。[\[5\]](#cite_note-Wikipedia2022-5)

------------------

*同样值得注意的是，这些效应不一定会以可预测或可靠的方式发生，尽管较高的剂量更可能诱发全方位的效应。*同样，**不良反应**随着剂量的增加变得越来越可能，并可能包括**成瘾、严重伤害或死亡** ☠。*

氟硝西泮的整体精神状态被描述为一种强烈的镇静、嗜睡、放松、焦虑抑制、记忆抑制和去抑制状态，类似于较高剂量的[地西泮](./../药物/地西泮.md)和[替马西泮](./../药物/替马西泮.md)（注：假设替马西泮暂无页面，此处保留翻译）的精神状态。

*     + **[镇静](./../药效/镇静.md)** - 氟硝西泮可产生极度镇静，通常导致压倒性的嗜睡状态。在较高水平下，这会导致用户突然感觉极度睡眠不足，好像几天没睡觉一样，迫使他们坐下，通常感觉随时可能昏倒，而不是从事体力活动。这种睡眠不足的感觉与剂量成正比增加，最终变得足以迫使人完全失去知觉。
    + **[头晕](./../药效/头晕.md)**
    + **[恶心](./../药效/恶心.md)**
    + **[肌肉松弛](./../药效/肌肉松弛.md)**
    + **[运动控制丧失](./../药效/运动控制丧失.md)**
    + **[呼吸抑制](./../药效/呼吸抑制.md)**
    + **[癫痫发作抑制](./../药效/癫痫发作抑制.md)**
    + **[心率减慢](./../药效/心率减慢.md)**
    + **[血压降低](./../药效/血压降低.md)**
    + **[躯体沉重感](./../药效/躯体沉重感.md)** - 氟硝西泮已知会导致身体沉重感。这种效应的范围可以从较低剂量的运动障碍和移动困难到高剂量的完全嗜睡或无法站立或移动。
    + **[躯体欣快感](./../药效/躯体欣快感.md)** - 在氟硝西泮上感受到的欣快感明显强于其他[苯二氮卓类物质](./../文档/药物分类/苯二氮卓类物质.md)如[阿普唑仑](./../药物/阿普唑仑.md)。
    + **[口干](./../药效/口干.md)**

### **矛盾效应** ![Paradox placeholder.svg](./../文件/Paradox_placeholder.svg)

* 对[苯二氮卓类物质](./../文档/药物分类/苯二氮卓类物质.md)的矛盾反应，如癫痫发作增加（在癫痫患者中）、攻击性、焦虑增加、暴力行为、冲动控制丧失、易怒和自杀行为有时会发生（尽管在一般人群中很少见，发生率低于1%）。[\[10\]](#cite_note-10)[\[11\]](#cite_note-11) 这些矛盾效应在娱乐性滥用者、精神障碍患者、儿童和高剂量治疗方案的患者中发生频率更高。[\[12\]](#cite_note-12)[\[13\]](#cite_note-13)

### **认知效应** ![User.svg](./../文件/User.svg)

* 氟硝西泮的认知效应可以分解为几个部分，这些部分随着剂量的增加而逐渐增强。它包含大量的典型[抑制剂](./../文档/药物分类/抑制剂.md)认知效应。

    最突出的这些认知效应通常包括：
    + **遗忘** - 氟硝西泮会导致相当程度的遗忘。
    + **认知欣快感** - 在氟硝西泮上感受到的认知欣快感明显强于其他[苯二氮卓类物质](./../文档/药物分类/苯二氮卓类物质.md)如[阿普唑仑](./../药物/阿普唑仑.md)。
    + **焦虑抑制**
    + **思维减速**
    + **分析抑制**
    + **去抑制**
    + **强迫性加药**
    + **情感抑制** - 虽然这种化合物主要抑制焦虑，但它也以一种独特但不如[抗精神病药](./../文档/药物分类/home.md)强烈的方式迟钝其他情绪。
    + **清醒错觉** - 这是一种错误的信念，即尽管有明显的证据表明相反，如严重的认知障碍和无法与他人充分沟通，但一个人还是完全清醒的。这种现象最常发生在严重剂量下。
    + **记忆抑制** - 氟硝西泮主要抑制短期记忆，导致健忘和/或行为紊乱。
    + **困惑** - 在严重剂量下，氟硝西泮会导致困惑。这种效应是药物在严重剂量下抑制基本认知功能（如理解、记忆和推理能力）的结果。
    + **动力抑制** - 由于氟硝西泮的严重镇静和嗜睡，在这种化合物影响下进行任何类型的需要移动或大量努力的活动可能很难做到，特别是在较高剂量下。
    + **语言抑制** - 氟硝西泮导致言语不清和难以清晰或连贯地表达词语。
    + **梦境增强** - 虽然大多数人报告像氟硝西泮这样的苯二氮卓类药物倾向于导致无梦睡眠状态，但也报告了相反的效应，并且似乎在用户经历GABA能物质（酒精是一个著名的例子）的戒断时特别普遍。目前这种差异的原因尚不清楚。

### **药效残余** ![Aftereffects (3).svg](./../文件/Aftereffects_(3).svg)

*     + **反跳性焦虑** - 反跳性焦虑是使用[苯二氮卓类物质](./../文档/药物分类/苯二氮卓类物质.md)等[焦虑缓解](./../药效/镇静.md)物质时常见的效应。它通常与处于该物质影响下的总时长以及在特定时期内的总消耗量相对应，这种效应很容易导致依赖和成瘾的循环。
    + **残留嗜睡** - 虽然苯二氮卓类药物可以用作有效的[助眠](./../药效/镇静.md)辅助，但它们的效应可能会持续到第二天早上，这可能会导致用户感到“昏沉”或“迟钝”长达几个小时。
    + **梦境增强** 或 **梦境抑制**
    + **残留嗜睡**
    + **思维减速**
    + **思维紊乱**
    + **易怒**

毒性与伤害潜力
---------------------------

[![](./../文件/Harmchart.png)](./../文件/Harmchart.png)

[放大](./../文件/Harmchart.png)

雷达图显示了苯二氮卓类药物与其他药物相比的相对身体伤害、社会伤害和依赖性。[\[14\]](#cite_note-14)

更多信息：[研究用化学品 § 毒性与伤害潜力](./../文档/研究用化学品.md)

相对于剂量，氟硝西泮的毒性可能较低。[\[2\]](#cite_note-Mandroili2008-2) 然而，当与[酒精](./../药物/酒精.md)或[阿片类药物](./../文档/药物分类/阿片类药物.md)等[抑制剂](./../文档/药物分类/抑制剂.md)混合时，它具有潜在的[致死性](./../药效/呼吸抑制.md)。

强烈建议在使用这种药物时采取[伤害减少措施](./../文档/负责任的用药索引页.md)。

### 致死剂量

氟硝西泮的小鼠口服[LD50](./../文档/药物剂量分类.md)（50%人口的致死剂量）为1200 mg/kg，大鼠为415 mg/kg。

### 耐受性与成瘾潜力

氟硝西泮具有极强的身体和心理成瘾性。

对镇静-催眠效应的耐受性将在连续使用几天内形成。停止使用后，耐受性在7 - 14天内恢复到基线。然而，在某些情况下，这可能需要更长的时间，具体取决于长期使用的持续时间和强度。

在连续给药几周或更长时间后突然停止使用，可能会出现戒断症状或反跳症状，可能需要逐渐减少剂量。有关以受控方式减少苯二氮卓类药物的更多信息，请参阅[本指南](http://www.benzo.org.uk/manual/bzcha02.htm)。

[苯二氮卓类药物的停药](./../文档/药物戒断反应.md)众所周知是困难的；对于经常使用的个人来说，不在数周内逐渐减少剂量而停止使用可能会危及生命。存在[高血压](./../药效/血压升高.md)、[癫痫发作](./../药效/癫痫发作.md)和死亡的风险增加。[\[3\]](#cite_note-Lann2009-3) 在戒断期间应避免使用降低癫痫发作阈值的药物，如[曲马多](./../药物/曲马多.md)（注：假设曲马多暂无页面，此处保留翻译，若有可指向相关阿片类）。

氟硝西泮与所有[苯二氮卓类物质](./../文档/药物分类/苯二氮卓类物质.md)均表现出交叉耐受性，这意味着在服用它之后，所有苯二氮卓类药物的效应都会降低。

### 危险的相互作用

尽管许多药物单独使用是安全的，但当与其他物质结合使用时，它们可能会变得危险甚至危及生命。下面的列表包含一些常见的潜在危险组合，但可能不包括所有组合。某些组合在各自低剂量下可能是安全的，但仍然会增加死亡的潜在风险。在消费之前，应始终进行[独立研究](https://www.google.com/)以确保两种或多种物质的组合是安全的。

* **[抑制剂](./../文档/药物分类/抑制剂.md)** (*[1,4-丁二醇](./../药物/1,4-Butanediol.md)、[2-甲基-2-丁醇](./../药物/2-methyl-2-butanol.md)、[酒精](./../药物/酒精.md)、[巴比妥类物质](./../文档/药物分类/巴比妥类物质.md)、[GHB](./../药物/GHB.md)/[GBL](./../药物/GBL.md)、[甲喹酮](./../药物/甲喹酮.md)、[阿片类药物](./../文档/药物分类/阿片类药物.md)*) - 这种组合可能导致危险甚至致命水平的[呼吸抑制](./../药效/呼吸抑制.md)。这些物质会增强彼此引起的[肌肉松弛](./../药效/肌肉松弛.md)、[镇静](./../药效/镇静.md)和[遗忘](./../药效/认知效应.md)（注：遗忘通常归类于认知效应），并可能导致高剂量下意外失去知觉。在失去知觉期间呕吐和由此导致的窒息死亡的风险也会增加。如果发生这种情况，用户应尝试以[恢复体位](./../文档/恢复体位.md)入睡，或者让朋友将他们移到该位置。
* **[解离剂](./../文档/药物分类/解离剂.md)** - 这种组合可能导致在失去知觉期间呕吐和由此导致的窒息死亡的风险增加。如果发生这种情况，用户应尝试以[恢复体位](./../文档/恢复体位.md)入睡，或者让朋友将他们移到该位置。
* **[兴奋剂](./../文档/药物分类/兴奋剂.md)** - 将苯二氮卓类药物与[兴奋剂](./../文档/药物分类/兴奋剂.md)联用是危险的，因为存在过度中毒的风险。兴奋剂会降低苯二氮卓类药物的[镇静](./../药效/镇静.md)效应，这是大多数人用来确定其中毒水平的主要因素。一旦兴奋剂失效，苯二氮卓类药物的效应将显著增加，导致强化的[去抑制](./../药效/认知效应.md)以及[其他效应](./../文档/药物分类/苯二氮卓类物质.md)。如果联用，应严格限制自己每小时只服用一定量的苯二氮卓类药物。如果不监测水分，这种组合还可能导致严重脱水。

### 药物过量

当大量服用[苯二氮卓类药物](./../文档/药物分类/苯二氮卓类物质.md)或与其他抑制剂同时服用时，可能会发生苯二氮卓类药物过量。这在与其他GABA能抑制剂如[巴比妥类物质](./../文档/药物分类/巴比妥类物质.md)和[酒精](./../药物/酒精.md)联用时特别危险，因为它们的作用相似，但结合到GABA-A受体上不同的变构位点。因此，它们的效应会相互增强。苯二氮卓类药物增加了氯离子通道在GABA-A受体上开放的频率，而巴比妥类药物增加了它们开放的持续时间，这意味着当两者都被消耗时，离子通道将更频繁地开放并保持开放更长时间[\[15\]](#cite_note-15)。苯二氮卓类药物过量是一种医疗紧急情况，如果不及时和适当地治疗，可能会导致昏迷、永久性脑损伤或死亡。

苯二氮卓类药物过量的症状可能包括严重的[思维减速](./../药效/思维减速.md)、[言语不清](./../药效/语言抑制.md)、[困惑](./../药效/困惑.md)、[妄想](./../药效/妄想.md)（注：Delusions通常对应妄想，此处假设药效文件夹下有相关文件，若无则保留翻译）、[呼吸抑制](./../药效/呼吸抑制.md)、昏迷或死亡。苯二氮卓类药物过量可以在医院环境中得到有效治疗，结果通常良好。苯二氮卓类药物过量有时用[氟马西尼](./../药物/氟马西尼.md)（注：假设有对应页面）治疗，这是一种GABA-A拮抗剂[\[16\]](#cite_note-16)。然而，护理主要是支持性的。

* **国际**：氟硝西泮是1971年国际精神药物公约下的附表III药物。\[*[来源请求](./../文档/科学信息索引页.md)*\]
* **澳大利亚**：氟硝西泮是附表8（S8）或受管制药物。\[*[来源请求](./../文档/科学信息索引页.md)*\]
* **奥地利**：根据AMG（Arzneimittelgesetz Österreich），氟硝西泮在医疗用途上是合法的，但在SMG（Suchtmittelgesetz Österreich）下，未经处方销售或持有是非法的。\[*[来源请求](./../文档/科学信息索引页.md)*\]
* **加拿大**：氟硝西泮是附表I受管制物质，仅凭处方可得。\[*[来源请求](./../文档/科学信息索引页.md)*\]
* **德国**：自2011年11月起，氟硝西泮受BtMG附表3管制，需要特别处方。\[*[来源请求](./../文档/科学信息索引页.md)*\]
* **瑞士**：氟硝西泮被列为受管制的麻醉物质，但在重大病例中也极少作为助眠药开具处方。氟硝西泮片剂仅有1mg剂量，染成绿色和蓝色，当溶解在饮料中摄入时，会显著地使口腔染色。[\[17\]](#cite_note-17)
* **英国**：根据2001年滥用药物条例，氟硝西泮是C类，附表4受管制药物。[\[18\]](#cite_note-18)\[*[来源请求](./../文档/科学信息索引页.md)*\]
* **美国**：氟硝西泮在美国受管制物质法案下是附表IV药物，但不用于医疗。\[*[来源请求](./../文档/科学信息索引页.md)*\]
