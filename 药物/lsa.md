**麦角酸酰胺**（Lysergic acid amide，也称为 **ergine**、**d-麦角酸酰胺**、**d-赖氨酰胺**和 **LSA**）是赖氨酰胺类的一种[天然存在](../文档/科学信息索引页.md)的[迷幻剂](../文档/药物分类/迷幻剂.md)。LSA 是一种麦角生物碱，是[牵牛花](#牵牛花种子_mgs)种子的主要精神活性成分。\[来源请求\] LSA 在化学上与 [LSD](./LSD.md) 相关，据称产生类似的效果，尽管其相似程度尚不清楚。

LSA 最早于 1932 年作为麦角生物碱研究的一部分被描述。[\[1\]](#cite_note-1) 1947 年，Albert Hofmann 合成并测试了其对人类的活性。[肌内注射](../文档/给药途径.md) 500 微克剂量产生了“疲倦、梦幻般的状态，无法保持清晰的思维”。[\[2\]](#cite_note-TiKHAL-2) 1970 年，LSA 被检测出是柯氏异叶牵牛（Hawaiian baby woodrose）种子的成分，当时这些种子被磨成胶囊并在街头作为“麦斯卡林”出售。[\[3\]](#cite_note-3) 今天，LSA 通常通过食用[牵牛花](#牵牛花种子_mgs)和[柯氏异叶牵牛](#柯氏异叶牵牛种子_hbwr)种子来摄取。[\[4\]](#cite_note-4)

用户报告描述 LSA 的效果主要是[镇静](../药效/镇静.md)和梦幻般的，伴有轻度到中度的迷幻成分。LSA 的迷幻效果发生得不一致，不能直接与经典的迷幻剂如 [LSD](./LSD.md)、[赛洛西宾蘑菇](../药物/4-AcO-DMT.md)（注：原文指蘑菇，此处链接至相关或不做链接，因文件树无蘑菇，暂不链接）或[麦斯卡林](./麦斯卡林.md)相提并论。LSA 被描述为主要具有躯体和认知效果，视觉效果较少。许多用户报告在服用含 LSA 的种子时会出现严重的[恶心](../药效/恶心.md)和身体不适（“身体负荷”）。

像其他[迷幻剂](../文档/药物分类/迷幻剂.md)一样，LSA 不被认为具有成瘾性。[\[5\]](#cite_note-5) 然而，不良反应如严重的[焦虑](../药效/不适性整体躯体效应.md)、偏执和[精神病](../文档/兴奋剂精神病.md)总是可能的，特别是对于那些有精神疾病倾向的人。[\[6\]](#cite_note-6) 因此，如果使用这种物质，强烈建议采取[伤害减少措施](../文档/负责任的用药索引页.md)。

化学
---------

LSA，或 d-麦角酸酰胺，是一种属于赖氨酰胺类的有机生物碱。LSA 的化学结构包含一个麦角酸的核心结构，其中一个胺官能团结合在 RN 上。麦角酸的结构由一个双环六氢吲哚融合到一个双环喹啉基团（麦角酸）组成。在喹啉的碳 8 位，结合了一个乙酰胺基团。LSA 还在碳 6 位被一个甲基取代。

LSA 是一种手性化合物，在 R5 和 R8 处有两个立体中心。LSA，也称为 (+)-D-LSA，具有 (5*R*, 8*R*) 的绝对构型。LSA 的其他三种立体异构体不具有精神活性。LSA 在结构上类似于 [LSD](./LSD.md)，例外是 LSA 缺乏 LSD 在其甲酰胺基团的 RN 处的二乙基取代。它可以用作 LSD 的前体。

药理学
------------

更多信息：[血清素能迷幻剂](../文档/药物分类/迷幻剂.md)

LSA 的[迷幻](../文档/药物分类/迷幻剂.md)效果被认为来自于其作为[5-HT2A 受体](../文档/血清素.md)的[部分激动剂](../文档/受体激动剂.md)的功效。然而，这些相互作用的作用以及它们如何导致[迷幻](../文档/药物分类/迷幻剂.md)体验仍然是科学研究的主题。

关于 LSA 是牵牛花和柯氏异叶牵牛种子中主要[迷幻](../文档/药物分类/迷幻剂.md)成分的观点受到了挑战，因为据报道，分离出的合成 LSA 的性质仅具有轻微的迷幻性。因此，有人提出，整体体验可能实际上是由这些植物材料中的多种赖氨酰胺生物碱（包括 iso-LSA 和 LSH）的混合物产生的，而不是单一的精神活性化合物。

### **物理效应**

+ **[镇静](../药效/镇静.md)** - LSA 主要是镇静的；然而，这取决于环境。例如，当在有大量刺激的环境中或进行剧烈体力活动（如散步、跑步、攀爬或跳舞）时，LSA 能够变得具有刺激性和充满活力。相反，当在安静的环境中，如带有舒适座位的黑暗房间里服用时，它往往是放松和镇静的。
+ **自发性躯体感觉** - LSA 的“身体 high”可以被描述为一种温和但令人愉悦且柔和的刺痛感。这主要在高剂量下被注意到，并伴随着强烈的[躯体欣快感](../药效/躯体欣快感.md)波浪，这些波浪通常在旅程中不同的不可预测的点自发显现，但也可能保持持续的存在。与 [LSD](./LSD.md) 相比，LSA 的物理效应相对于其视觉和认知效应往往主导体验。
    - **[躯体沉重感](../药效/躯体沉重感.md)**
    - **[躯体欣快感](../药效/躯体欣快感.md)** - 据报道，LSA 比 [LSD](./LSD.md) 更容易产生这种效果。然而，它可能会被强烈的、不舒服的恶心和血管收缩感所掩盖，特别是当直接食用含 LSA 的种子而未进行任何提取时。
+ **[运动控制丧失](../药效/运动控制丧失.md)** - 这种效果在高剂量下比低剂量下明显得多。它可以与[酒精](./酒精.md)引起的醉酒状态下的运动控制丧失相提并论，并因[躯体沉重感](../药效/躯体沉重感.md)而增强。
+ **[体温调节抑制](../药效/体温调节抑制.md)**\[来源请求\]
+ **[恶心](../药效/恶心.md)** - LSA 的恶心效果被认为主要是由种子（例如牵牛花、柯氏异叶牵牛等）的其他成分引起的，而不是 LSA 本身。可以使用各种提取方法来显著减少甚至消除这种物质可能产生的恶心。轶事报告还表明，姜茶或大麻可能有助于对抗恶心。
+ **血管收缩**\[来源请求\] - 据普遍报道，LSA 会产生强烈且明显的血管收缩感。这在个体之间的强度各不相同，但与其他[迷幻剂](../文档/药物分类/迷幻剂.md)相比，通常被认为是中度到极度不舒服的。这种效果通常被报道为引起关节和肢体疼痛。
+ **[心率增快](../药效/心率增快.md)**
+ **[血压升高](../药效/血压升高.md)** 或 **[血压降低](../药效/血压降低.md)**\[来源请求\] - 据报道，LSA 能够产生血压的升高和降低，有时在体验的不同点交替出现，例如在[药效上升](../文档/药效上升期.md)或[药效褪去](../文档/药效下降期.md)期间，尽管尚不清楚这在多大程度上取决于种子和生物碱含量的变化。

### **视觉效应**

LSA 的视觉效果主要在服用大剂量时出现。与 [LSD](./LSD.md) 和[赛洛辛](../药物/4-HO-MET.md)（注：原文为Psilocin，此处链接至结构类似的4-HO-MET或留空，因无赛洛辛文件，此处暂不链接）相比，LSA 的视觉效果相对于其伴随的认知和物理效果的强度来说是温和的。

#### 增强
+ **[颜色增强](../药效/颜色增强.md)**
+ **[模式识别增强](../药效/模式识别增强.md)**
+ **[视觉锐度增强](../药效/视觉锐度增强.md)**
+ **帧率增强**

#### 扭曲
视觉扭曲和改变比其他[迷幻剂](../文档/药物分类/迷幻剂.md)产生的闭眼扭曲要简单得多。体验到的效果详述如下：

+ **[深度感知扭曲](../药效/深度感知扭曲.md)**
+ **[漂移](../药效/漂移.md)** *([融化](../药效/漂移.md)、[呼吸](../药效/漂移.md)、[变形](../药效/漂移.md)和[流动](../药效/漂移.md))* - 与其他迷幻剂相比，这种效果可以描述为温和但细节丰富，外观像卡通一样。扭曲在运动中快速而流畅，在持久性上转瞬即逝。这是一种不一致显现的效果，有些人从未报告过这种效果。
+ **[颜色偏移](../药效/颜色偏移.md)**
+ **[残影](../药效/残影.md)**
+ **景色分割**
+ **对称纹理重复**

#### 几何
LSA 产生的视觉几何形状在外观上可以描述为更类似于 [4-AcO-DMT](./4-AcO-DMT.md) 和[死藤水](./死藤水.md)，而不是 [LSD](./LSD.md) 或 [2C-B](./2C-B.md)。它可以描述为在复杂性上主要是错综复杂的，形式上是抽象的，外观上是有机的，组织上是非结构化的，光线昏暗，配色多彩，阴影有光泽，边缘柔和，尺寸小，速度慢，运动平滑，角落圆润，深度非沉浸式，强度一致。在高剂量下，它明显更有可能产生 8B 几何形状而不是 8A 几何形状。

#### 幻觉状态
当大剂量服用时，LSA 能够一致地产生全方位的高级幻觉状态。然而，达到这些状态所需的剂量可能会产生非常不舒服且可能危险的副作用。这些状态包括：

+ **变换** - 这在 LSA 中极其常见，部分遵循用户当前的思维过程内容。
+ **内部幻觉** (*自主实体*; *设置、景色和景观*; *透视幻觉*和*场景与情节*) - 与 [LSD](./LSD.md) 不同，LSA 在高剂量下一致地产生中度到高度的幻觉状态。这种特定的效果可以比作清醒梦状态。幻觉在黑暗环境中更常见，并且可以通过其变化来全面描述，如在可信度上清晰，在风格上互动，在内容上是新体验，可控，并且几乎完全在整体主题上具有个人、宗教、精神、科幻、幻想、超现实、荒谬或超验的性质。
+ **外部幻觉** (*自主实体*; *设置、景色和景观*; *透视幻觉*和*场景与情节*) - 这种效果比其他致幻剂发生得少得多且不频繁。然而，它们偶尔会在较重剂量下发生。它们可以被描述为在可信度上清晰，在可控性上自主，在风格上坚实。
+ **[周边信息误判](../药效/周边信息误判.md)**

### **认知效应**

许多人描述 LSA 的认知效果与 [LSD](./LSD.md) 或赛洛辛等其他常用迷幻剂相比，风格上极其放松但头脑清晰。虽然它主要是镇静的，但它可以随机间隔产生快节奏的思维爆发和刺激。

LSA 产生大量的[迷幻](../文档/药物分类/迷幻剂.md)认知效果。这些典型效果中最突出的包括：


+ **分析增强** - 这种效果以内省为主导。
+ **[焦虑](../药效/不适性整体躯体效应.md)** 或 **焦虑抑制**
+ **概念性思维**
+ **认知欣快**
+ **既视感**
+ **妄想**
+ **情感增强**
+ **同理心、情感和社交能力增强**
+ **音乐欣赏能力增强** - 听音乐可以强烈增强整体体验。
+ **沉浸感增强**
+ **语言抑制** - 与 [LSD](./LSD.md) 等经典迷幻剂的语言抑制相比温和。
+ **幽默感增强**
    - **大笑发作**
+ **记忆抑制**
    - **自我死**
+ **正念**
+ **新奇感增强**
+ **复原感**
+ **自主声音交流**
+ **思维加速**
+ **思维连通性**
+ **思维循环**
+ **时间扭曲**
+ **清醒度**

### 药物联用效应

* **[大麻](./大麻.md)** - 大麻可以极大地增强 LSA 的感官和认知效果。混合使用这些物质时建议极度谨慎，因为它可以显著增加发生负面心理反应（如焦虑、困惑和[精神病](../文档/兴奋剂精神病.md)）的机会。建议使用此组合的人从通常大麻剂量的一小部分开始，并在吸食之间长时间休息，以避免负面反应。
* **[解离剂](../文档/药物分类/解离剂.md)** - 解离剂极大地增强了 LSA 的认知、视觉和一般幻觉效果。在 LSA 影响下由解离剂诱导的空洞、空间和虚空具有比单独使用解离剂更生动的视觉效果，以及更强烈的内部幻觉，并相应地增加了困惑、妄想和[精神病](../文档/兴奋剂精神病.md)的风险。
* **[酒精](./酒精.md)** - 酒精的中枢抑制作用可用于减少 LSA 产生的一些焦虑和紧张。然而，酒精会导致[脱水](../药效/脱水.md)、[恶心](../药效/恶心.md)和身体疲劳，这可能会显著恶化体验。如果使用酒精，建议控制节奏，饮用量仅为通常的一小部分。
* **[苯二氮卓类物质](../文档/药物分类/苯二氮卓类物质.md)** - 取决于剂量，苯二氮卓类药物可以轻微到完全减少 LSA 旅程的认知、物理和视觉效果的强度。它们在停止["恶性旅程"](../文档/恶性旅程.md)方面非常有效，代价是健忘和旅程强度降低。为此目的获取它们时建议谨慎，因为它们非常容易被滥用。
* **[迷幻剂](../文档/药物分类/迷幻剂.md)** - 当与其他迷幻剂联合使用时，每种物质的物理、认知和视觉效果都会强烈地增强和协同。这些物质之间的协同作用是不可预测的，因此一般不建议这样做。如果选择联合使用迷幻剂，建议从比单独使用任何一种物质都要低得多的剂量开始。

毒性和危害潜力
---------------------------

**毒性和危害潜力部分是一个[存根](http://psychonautwiki.org/w/index.php?title=Special%3AWhatLinksHere&target=Template%3AStub&namespace=)。**因此，它可能包含不完整甚至**危险的错误**信息！你可以通过编辑来帮助扩展或更正它。
*注意：如果使用这种物质，请务必进行独立研究并使用**[伤害减少措施](../文档/负责任的用药索引页.md)**。*

更多信息：[负责任的用药 § 致幻剂](../文档/负责任的用药索引页.md)

娱乐性使用 LSA 的毒性和长期健康影响尚未在任何科学背景下进行研究，确切的毒性剂量尚不清楚。

轶事证据表明，仅以低至中等剂量单独尝试 LSA 并非常谨慎地使用（但这不能完全保证）没有负面健康影响。在食用之前，应始终进行[独立研究](https://www.google.com/)以确保两种或多种物质的组合是安全的。

强烈建议在使用这种物质时使用[伤害减少措施](../文档/负责任的用药索引页.md)。

### 血管收缩

不应长期定期使用 LSA。当在短时间内重复使用时，LSA 的血管收缩作用会累积，而精神活性作用会减弱。血管收缩累积的一个常见迹象可以描述为腿部疼痛或不舒服的感觉（参见[不宁腿](../药效/不宁腿.md)）。[\[9\]](#cite_note-9) 这是由于进入肌肉的血液量不足造成的。大腿上部肌肉是身体中最大、最耗能的肌肉，如果流向它们的血流量哪怕稍微降低，也会感到酸痛。

当食用 HBWR、牵牛花种子或纯 LSA 种子并感到腿部酸痛时，据报道休息会有所帮助。对于 LSA，可能需要禁欲长达 3 天才能恢复到血管收缩基线。

### 依赖和滥用潜力

LSA 被认为是非成瘾性的，滥用潜力低。没有文献报告成功尝试训练动物自我给药 LSA，这是预测滥用倾向的动物模型，表明它不具有引发或维持依赖所需的药理学特性。\[来源请求\] 停止使用时几乎没有戒断综合征。

对 LSA 效果的耐受性在摄入后几乎立即形成。之后，耐受性大约需要 7 天才能恢复到基线（在没有进一步摄入的情况下）。LSA 与所有[迷幻剂](../文档/药物分类/迷幻剂.md)产生交叉耐受，这意味着在使用 LSA 后，所有迷幻剂的效果都会降低。

### 危险的相互作用

***警告：*** *许多单独使用相当安全的精神活性物质，当与某些其他物质结合使用时，可能会突然变得危险甚至危及生命。以下列表列出了一些已知的危险相互作用（尽管不保证包括所有相互作用）。*

*始终进行独立研究（例如 [Google](https://www.google.com)、[DuckDuckGo](https://www.duckduckgo.com)、[PubMed](https://pubmed.ncbi.nlm.nih.gov/)）以确保两种或多种物质的组合是安全的。一些列出的相互作用来自 [TripSit](https://combo.tripsit.me)。*

* **锂盐 (Lithium)** - 锂通常用于治疗双相情感障碍。大量的轶事证据表明，将其与[迷幻剂](../文档/药物分类/迷幻剂.md)一起服用会显著增加[精神病](../文档/兴奋剂精神病.md)和[癫痫发作](../药效/癫痫发作.md)的风险。因此，严格不建议这种组合。
* **[大麻](./大麻.md)** - 大麻可能与 LSA 的效果产生出乎意料的强烈且不可预测的协同作用。建议对此组合保持谨慎，因为它会显著增加不良心理反应的风险，如焦虑、偏执、恐慌发作和[精神病](../文档/兴奋剂精神病.md)。建议用户从通常大麻剂量的一小部分开始，并在吸食之间长时间休息，以避免无意的过量。
* **[兴奋剂](../文档/药物分类/兴奋剂.md)** - 像[苯丙胺](./苯丙胺.md)、[可卡因](./可卡因.md)（注：文件树无，暂不链接）或[哌甲酯](./哌甲酯.md)这样的兴奋剂会影响大脑的许多部分并改变[多巴胺](../文档/多巴胺.md)功能。这种组合会增加焦虑、偏执、恐慌发作和思维循环的风险。这种相互作用还可能导致躁狂和[精神病](../文档/兴奋剂精神病.md)的风险增加。\[来源请求\]
* **曲马多** - 曲马多被充分证明会降低癫痫发作阈值[\[10\]](#cite_note-10)，而[迷幻剂](../文档/药物分类/迷幻剂.md)可能会触发易感个体的癫痫发作。\[来源请求\]


### 其他相互作用

* **[选择性血清素再摄取抑制剂](../文档/神经递质再摄取抑制剂.md) (SSRIs)** - 据报道，SSRIs 会抑制 LSA 的视觉和认知效果。
* **[单胺氧化酶抑制剂](../文档/单胺氧化酶抑制剂.md) (MAOIs)** - MAO 抑制剂，如西番莲和骆驼蓬，会增强迷幻剂的视觉和内省效果。建议在将 [MAOIs](../文档/单胺氧化酶抑制剂.md) 与迷幻剂结合使用时要小心，因为它们会增加发生[恶性旅程](../文档/恶性旅程.md)的机会，并可能与其他物质（如 SSRIs 和[兴奋剂](../文档/药物分类/兴奋剂.md)）以及一些迷幻剂（如 [MDA](./MDA.md)）发生危险的相互作用。


法律地位
------------

* **澳大利亚**：LSA 在澳大利亚生产、销售和消费是非法的。\[来源请求\]
* **奥地利**：含 LSA 的种子不受管制，因此拥有、生产和销售是合法的。\[来源请求\]
* **德国**：截至 2019 年 7 月 18 日，LSA 根据 NpSG（*新精神活性物质法*）受到管制。[\[11\]](#cite_note-11)[\[12\]](#cite_note-12) 以将其投放市场为目的的生产和进口、对他人的管理和交易都是受惩罚的。拥有是非法的，但不予处罚。[\[13\]](#cite_note-13) 含有 LSA 的植物部分不受法律涵盖。[\[14\]](#cite_note-14)
* **拉脱维亚**：LSA 本身作为 LSD 的结构类似物受到管制。但是，没有关于含 LSA 种子的信息。在拉脱维亚，植物只有在含有附表 I 管制物质时才是非法的，而 LSA 不是附表 1 管制物质。[\[15\]](#cite_note-15)
* **荷兰**：LSA 是非法的。然而，柯氏异叶牵牛和牵牛花种子像其他致幻植物一样，消费、销售或拥有并不违法。\[来源请求\]
* **新西兰**：LSA 的消费、销售和拥有是非法的。牵牛花物种的植物和种子可以合法拥有、种植、购买和分发。\[来源请求\]
* **瑞典**：卫生当局正在研究 LSA，在不久的将来 LSA 有可能成为非法的。[\[16\]](#cite_note-16)
* **瑞士**：作为一种化学物质，LSA 根据 Verzeichnis E 第 263 点作为麦角酸的定义衍生物可被视为管制物质。当用于科学或工业用途时是合法的。[\[17\]](#cite_note-17) 植物物质并不违法。
* **土耳其**：LSA 的消费、销售和拥有是非法的。[\[18\]](#cite_note-18) 牵牛花物种的植物和种子可以合法拥有、种植、购买和分发。\[来源请求\]
* **英国**：LSA 是 A 类管制物质，被归类为 [LSD](./LSD.md) 的前体。\[来源请求\]
* **美国**：作为 LSD 的前体，LSA 是 DEA 附表 III 管制物质。\[来源请求\] 然而，可以在互联网上和园艺商店合法购买含 LSA 的种子。
* **波兰**：*Argyreia nervosa*（柯氏异叶牵牛）和 *Rivea corymbosa* 是非法的 (I-N)，拥有将被判处最高 3 年监禁。LSA 本身受类似物法 (VI-NPS) 涵盖，因此是非法的，拥有将被处以罚款。牵牛花种子是合法的，并在园艺商店公然出售。[\[19\]](#cite_note-19)
