**苦茶碱**（也称为 **Teacrine**）是一种[益智药](/文档/药物分类/益智药.md)性质的[兴奋剂](/文档/药物分类/兴奋剂.md)，也是[咖啡因](/药物/咖啡因.md)的结构类似物，似乎在某些植物中是由咖啡因合成的喵。它是一种苦味的白色结晶[嘌呤生物碱]，可以增加大脑活动并诱发暂时的改善，包括增强警觉性、[清醒度](/药效/清醒度.md)和[刺激](/药效/刺激.md)呢。相对于咖啡因，它在产生的刺激感方面表现出约三分之二的效力哦。

苦茶碱的机制与咖啡因很大程度上平行，因为它在啮齿动物中似乎具有刺激作用，但这仅在较高剂量下发生（苦茶碱达到峰值的确切口服剂量尚不清楚）。与咖啡因类似，它在相对较低的剂量下会产生镇静效果，但咖啡因的这种镇静效果出现在极低且不切实际的剂量下，而苦茶碱的镇静剂量则是苦茶（Kucha tea）通常消耗的剂量喵；这可能就是为什么苦茶往往被推荐用于放松而非刺激的原因呢。

化学
---------

苦茶碱的化学结构与[咖啡因](/药物/咖啡因.md)在结构上非常相似喵。它是一种具有[取代黄嘌呤](/文档/药物分类/吗啡喃类物质.md)核心的合成[生物碱]。黄嘌呤是一种取代嘌呤，包含两个并环：嘧啶和咪唑。嘧啶是一个在 R 和 R 处含有氮成分的六元环；咪唑是一个在 R 和 R 处含有氮取代基的五元环。黄嘌呤包含与 R 和 R 双键结合的氧基团。像咖啡因一样，它在其结构的 R、R 和 R 处含有额外的甲基取代，与黄嘌呤骨架的开放氮基团结合。它是一种非手性芳香族化合物。

苦茶碱的化学名称是 1,3,7,9-四甲基尿酸；相比之下，咖啡因的化学名称是 1,3,7-三甲基黄嘌呤，结构上的唯一区别是 9 号碳原子上多了一个甲基，并且多了一个酮基，这使得咖啡因的黄嘌呤转化为了尿酸部分呢。

药理学
------------

苦茶碱通过几种机制发挥作用；然而，像[咖啡因](/药物/咖啡因.md)一样，它最重要的作用是拮抗一种叫做[腺苷](https://en.wikipedia.org/wiki/adenosine)的物质，这种物质在全身（特别是在神经系统中）以高水平自然循环。在大脑中，腺苷通常起到保护作用，其中一部分作用是降低神经活动水平喵。苦茶碱分子在结构上与咖啡因和腺苷都很相似，因此能够结合到细胞表面的腺苷受体上而不激活它们，从而起到竞争性抑制剂的作用。

除此之外，苦茶碱被认为对大多数其他主要神经递质也有类似的影响，包括[多巴胺](/文档/多巴胺.md)、[乙酰胆碱](/文档/乙酰胆碱.md)、[血清素](/文档/血清素.md)，在高剂量下还有[去甲肾上腺素](/文档/去甲肾上腺素.md)，以及小程度的[肾上腺素](/文档/肾上腺素.md)、[谷氨酸](/文档/谷氨酸.md)和皮质醇呢。

主观效应
------------------

下列效应基于[主观效应索引](/药效/home.md)以及本站贡献者的个人经验。所列效应很少（如果有的话）会全部同时发生，但较大剂量会增加发生几率，并更有可能诱发全范围的效应喵。

### **躯体效应** ![](/文件/Child.png)

+ **[刺激](/药效/刺激.md)** - 就其对用户身体能量水平的影响而言，苦茶碱通常被认为是轻度到中度充满活力和刺激性的，其方式与传统的娱乐性兴奋剂如[苯丙胺](/药物/苯丙胺.md)、[MDMA](/药物/MDMA.md)或[可卡因](/药物/可卡因.md)相比要弱得多。这会鼓励进行诸如做家务和重复性任务等体力活动，否则这些活动会显得无聊且费力呢。苦茶碱呈现的这种特定风格的刺激可以被描述为“强制性的”。这意味着在较高剂量下，随着磨牙、不自主的身体抖动和震动出现，保持静止变得困难或不可能，导致全身剧烈颤抖、手部不稳以及普遍缺乏运动控制喵。
+ **[镇痛](/药效/镇痛.md)** - 苦茶碱具有与其抗炎作用相关的减轻疼痛的特性。据信这与布洛芬的效果相当哦。
+ **[食欲抑制](/药效/食欲抑制.md)**
+ **[恶心](/药效/恶心.md)** - 这主要发生在高剂量时。
+ **[磨牙](/药效/磨牙.md)** - 这种效应并不像在其他[兴奋剂](/文档/药物分类/兴奋剂.md)（如 [MDMA](/药物/MDMA.md)）上那样一致。

### **[认知效应](/药效/认知效应.md)** ![User.png](/文件/User.png)

苦茶碱的认知效应可以细分为几个组成部分，这些部分随剂量成比例增强。它包含大量典型的[兴奋剂](/文档/药物分类/兴奋剂.md)认知效应。虽然在低到中等剂量下负面副作用通常很轻微，但随着剂量增加或长期使用，它们变得越来越可能显现喵。这在药效褪去期间尤为明显。

与[咖啡因](/药物/咖啡因.md)相比，在同等剂量下，苦茶碱的效果往往比刺激感更能带来专注方面的益处呢。这些认知效应中最突出的通常包括：

+ **[分析能力增强](/药效/分析能力增强.md)**
+ **[焦虑](/药效/焦虑.md)**
+ **[强迫性补量](/药效/强迫性补量.md)** - 这种效应不如[咖啡因](/药物/咖啡因.md)持久，因为它往往具有更长的持续时间。
+ **[认知欣快](/药效/认知欣快.md)** - 这种效应通常很轻微。
+ **[音乐欣赏能力增强](/药效/音乐欣赏能力增强.md)**
+ **[专注力强化](/药效/专注力强化.md)** - 与咖啡因相比，这种效应往往更加突出呢。
+ **[性欲增强](/药效/性欲增强.md)**
+ **[记忆增强](/药效/记忆增强.md)**
+ **[动机增强](/药效/动机增强.md)**
+ **[思维加速](/药效/思维加速.md)**
+ **[清醒度](/药效/清醒度.md)**

毒性和伤害潜力
---------------------------

目前尚不清楚苦茶碱是否会导致脑损伤，且相对于剂量而言，其毒性极低。与苦茶碱暴露相关的身体副作用相对较少喵。各种研究表明，在谨慎的情况下以合理剂量使用，它不会产生任何形式的负面认知、精神或有毒的身体后果呢。

强烈建议在使用此药物时采取[伤害减少措施](/文档/负责任的用药索引页.md)哦。

### 耐受性与成瘾潜力

与[咖啡因](/药物/咖啡因.md)一样，苦茶碱长期使用会产生依赖性，且滥用潜力较低。当产生依赖后，如果突然停止使用，会出现渴望和[药物戒断反应](/文档/药物戒断反应.md)喵。

苦茶碱的许多效应会随着长期和重复使用而产生耐受性。这导致用户必须施用越来越大的剂量才能达到相同的效果。之后，大约需要 3 - 7 天耐受性降至一半，1 - 2 周恢复到基线（在不再消耗的情况下）。但需要注意的是，耐受性的建立并不像[咖啡因](/药物/咖啡因.md)那样迅速呢。苦茶碱与腺苷受体拮抗剂存在交叉耐受性，这意味着在消耗苦茶碱后，某些[兴奋剂](/文档/药物分类/兴奋剂.md)如[咖啡因](/药物/咖啡因.md)和[可可碱](/药物/可可碱.md)的效果会降低。
