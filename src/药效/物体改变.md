# 物体改变

[◀返回](./home.html)

**物体改变** 的定义是体验到感知到的一个物体、多个物体或整个场景在三维空间中逐渐扭曲、移动、拉伸、移位或以其他方式动起来，这种方式可以从微妙到极端不等哦。[\[1\]](#cite_note-KleinmanGillin1977-1)[\[2\]](#cite_note-ObreshkovaKandilarov2017-2)[\[3\]](#cite_note-Gallimore2015-3)[\[4\]](#cite_note-JuszczakSwiergiel2013-4) 当体验者把视线移开或重新审视时，物体会立即重置为原始形状。如果再次直接盯着物体看，它可能会开始像以前一样或以不同的方式扭曲呢。这种改变不会以统一的方式发生，也无法可靠地预测。这种效应的强度通常与和这种效应同时发生的精神状态的强度和进展有关。

例如，盯着像椅子这样的物体看可能会导致它的三维形状开始剧烈地拉长或倾斜成夸张的形式，同时保留其原始的颜色和纹理。

这种效应也通常表现为感知到纹理逐渐拉伸并从它们所在的表面向外投射，形成详细的三维结构[\[4\]](#cite_note-JuszczakSwiergiel2013-4)，有点像复杂的、不透明的、固化的烟雾。这些结构通常保持与其延伸出的纹理宽度一致的大小。纹理可能在三维空间中向外投射几英寸到几米长。例如，如果盯着墙上的一幅画，它可能会在二维平面上向一个方向延伸，直到观察者把视线移开。

虽然这种主观效应成分被归类为视觉扭曲，[\[4\]](#cite_note-JuszczakSwiergiel2013-4)[\[5\]](#cite_note-EspiardLecardeur2005-5)[\[6\]](#cite_note-MehtaNaveen_Kumar2017-6) 但它很可能是外部幻觉被应用到用户环境中的物体上的间接结果，其发生方式并不会引入新数据，而只是改变了对三维结构内容的感知。

物体改变经常伴随着其他并发效应，比如 [谵妄](../文档/药物分类/谵妄剂.html) 和 [精神病](/FreeODwiki/药效/精神病发作.html)。[\[6\]](#cite_note-MehtaNaveen_Kumar2017-6) 它们最常见于使用 [严重](./药物剂量分类.html#heavy) [剂量](./药物剂量分类.html) 的 [谵妄剂](./药物分类/谵妄剂.html) 类化合物，比如 [苯海拉明](./药物/苯海拉明.html) 和 [曼陀罗](./药物/曼陀罗.html) 时发生。不过呢，它们也可能在 [兴奋剂精神病](../文档/兴奋剂精神病.html) 和 [睡眠剥夺](./睡眠剥夺.html) 的影响下发生。

### 物体激活

主条目：[物体激活](./物体激活.html)

在物体改变的极端一端，物体可能会变得如此完全地动画化，以至于这种效果开始呈现出标准 [幻觉状态](./幻觉状态.html) 的特质，超越了单纯的 [视觉扭曲](./视觉扭曲.html)。

这种独特的效应被称为物体激活，当一个人看着一个物体时，物体会自己移动、变活或显得有生命力。例如，如果一个人看着微波炉，门可能会自己打开和关闭，或者桌子上的杯子可能会开始滑动或倾倒。这种“被激活的物体”通常以在日常生活中发生的熟悉方式移动，进一步证明大脑正在将外部幻觉融合到环境中了呢。

然而，被激活的物体也可能执行完全不现实的动作。例如，一件家具可能看起来分解成许多悬浮的复杂旋转部分，然后重新组装成以前的状态。这些幻觉通常只有在一个人长时间直接盯着一个物体看时才会发生，并且是高级幻觉状态的罕见和极端迹象哦。
