# 暂时性勃起功能障碍

**暂时性勃起功能障碍**（Temporary erectile dysfunction）可以描述为难以勃起或难以维持足够的勃起状态以进行性交。这种效应会在某些物质的影响下以不同程度出现。

暂时性勃起功能障碍通常伴随着其他并发效应，比如[刺激](/FreeODwiki/药效/刺激.html)、[排尿困难](/FreeODwiki/药效/排尿困难.html) 和[体温调节抑制](/FreeODwiki/药效/体温调节抑制.html)，这些都会进一步加剧这个问题呢。它最常发生在使用[严重](/FreeODwiki/文档/给药剂量.html) [剂量](/FreeODwiki/文档/给药剂量.html) 的[刺激性](/FreeODwiki/药效/刺激.html) 化合物时，比如传统的[兴奋剂](/FreeODwiki/文档/药物分类/兴奋剂.html) 和某些[迷幻剂](/FreeODwiki/文档/药物分类/迷幻剂.html)。不过嘛，在使用[阿片类药物](/FreeODwiki/文档/药物分类/阿片类药物.html)、[解离剂](/FreeODwiki/文档/药物分类/解离剂.html)、[GABA能药物](/FreeODwiki/文档/药物分类/加巴喷丁类物质.html) 和[谵妄剂](/FreeODwiki/文档/药物分类/谵妄剂.html) 时，也可能会出现这种情况哦。
