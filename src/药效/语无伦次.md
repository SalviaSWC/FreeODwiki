# 语无伦次

[◀返回](./home.html)

**语无伦次 (Glossolalia)** 指的是一种不由自主地说话或思考，内容像是毫无意义的言语，但结构上却类似于真实语言的效应。[\[1\]](#cite_note-NewbergWintering2006-1)[\[2\]](#cite_note-GradyLoewenthal1997-2) 语言学家通常将其定义为一种类似说话音节的旋律流畅的发声，但没有任何易于理解的含义。[\[3\]](#cite_note-Motley2009-3)[\[4\]](#cite_note-Goodman1969-4) 值得注意的是，这种效应与精神分裂症患者那种“词语沙拉”特征的[思维解体](/FreeODwiki/药效/思维混乱.md "Thought disorganization")截然不同。[\[2\]](#cite_note-GradyLoewenthal1997-2)[\[3\]](#cite_note-Motley2009-3)

虽然有大量研究在宗教背景下描述了这种效应，但这种背景并不是必须的；目前已提出两种类型的语无伦次：[\[2\]](#cite_note-GradyLoewenthal1997-2)[\[5\]](#cite_note-Kavan2004-5)

*   **A型（平静型）**：发生于私密的、世俗的环境中。它依赖于语境，当时人虽然正在“说话”，但仍有自我意识，例如他们可以注意到其他需要关注的事情。这种现象出现得很频繁（每天或每周数次）。
*   **B型（兴奋型）**：发生于公共场合，表现为强烈的爆发性发声，是宗教性意识改变状态的产物。此时人没有自我意识，无法注意到其他事物。这种现象偶尔出现（每周或更少）。

在体验这种效应的过程中，说话的人可能完全意识不到他们说的并不是母语。这可能会导致[困惑](/FreeODwiki/药效/混乱.md "Confusion")和挫败感，因为他们很难理解为什么周围的人听不懂他们在说什么。

语无伦次通常伴随着其他并发效应，例如[语言能力抑制](/FreeODwiki/药效/语言能力抑制.md "Language suppression")、[宣泄](/FreeODwiki/药效/情绪宣泄.md "Catharsis")、[灵性增强](/FreeODwiki/药效/灵性增强.md "Spirituality enhancement")和[谵妄](/FreeODwiki/药效/谵妄.md "Delirium")。它最常在[严重](/FreeODwiki/文档/药物剂量分类.html#heavy "Dosage") [剂量](/FreeODwiki/文档/给药剂量.md "Dosage")的[致幻](/FreeODwiki/文档/药物分类/致幻剂.md "Hallucinogen")化合物影响下被诱发，例如[迷幻剂](/FreeODwiki/文档/药物分类/迷幻剂.md "Psychedelic")、[谵妄剂](/FreeODwiki/文档/药物分类/谵妄剂.md "Deliriant")和[解离剂](/FreeODwiki/文档/药物分类/解离剂.md "Dissociative")。
