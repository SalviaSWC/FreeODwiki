# 瞳孔缩小

| ![Pupilc](/FreeODwiki/文件/Pupilc.jpg) |
| :---: |
| 瞳孔缩小的图片示例 |

**瞳孔缩小**（也称为**针尖样瞳孔**）可以被描述为在正常光照条件下，一个人的瞳孔尺寸减小。这通常会降低一个人在低光条件下看东西的能力哦。

瞳孔缩小最常见于服用[中等](/FreeODwiki/文档/给药剂量.html#中等)[剂量](/FreeODwiki/文档/给药剂量.html)的[阿片类药物](/FreeODwiki/文档/药物分类/阿片类药物.html)时，如[海洛因](/FreeODwiki/药物/海洛因.html)、[卡痛叶](/FreeODwiki/药物/卡痛叶.html)（原文Kratom，假设已有条目，若无则为`卡痛.md`，这里根据习惯暂定）、[曲马多](/FreeODwiki/药物/曲马多.html)和[芬太尼](/FreeODwiki/药物/芬太尼.html)。
