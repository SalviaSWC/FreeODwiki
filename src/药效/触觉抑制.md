# 触觉抑制

**触觉抑制**可以描述为一个人感受触觉的能力下降了，这种方式可能导致全身普遍的麻木感喵。在更高的强度下，这最终可能增加到身体感觉被完全阻断、身体完全麻醉的程度呢。

触觉抑制通常伴随着其他同时发生的效应，如[镇痛](/FreeODwiki/药效/镇痛.html)和[躯体欣快感](/FreeODwiki/药效/躯体欣快感.html)。它最常在[中等](/FreeODwiki/文档/药物剂量分类.html)[剂量](/FreeODwiki/文档/给药剂量.html)的[解离剂](/FreeODwiki/文档/药物分类/解离剂.html)类化合物影响下产生，比如[氯胺酮](/FreeODwiki/药物/氯胺酮.html)、[苯环利定](/FreeODwiki/药物/PCP.html)和[右美沙芬](/FreeODwiki/药物/右美沙芬.html)。不过嘛，它也可能出现在[阿片类药物](/FreeODwiki/文档/药物分类/阿片类药物.html)和某些[GABA能](/FreeODwiki/文档/GABA.html)[抑制剂](/FreeODwiki/文档/药物分类/抑制剂.html)的影响下哦。

# 认知失联

**认知失联**是一种感觉与自己的身份感、思维流和一般认知过程疏远和分离的体验喵。这会导致诸如[人格解体](/FreeODwiki/药效/人格解体.html)、[现实解体](/FreeODwiki/药效/现实解体.html)、[记忆抑制](/FreeODwiki/药效/记忆抑制.html)、“自我死亡”以及一系列普遍的[认知抑制](/FreeODwiki/药效/认知抑制.html)状态。这种体验还可能导致一个人对自己意识的感知发生广泛的主观变化呢。下面列出了这些描述和记录：

* 感觉自己的意识思维流和记忆不属于自己
* 感觉自己的意识思维过程遥远且模糊
* 感觉自己的意识思维过程在结构或行为上变得自主和机械化
* 感觉自己的整体思维速度、连通性和分析能力的认知能力下降

认知失联通常伴随着其他同时发生的效应，如[视觉失联](/FreeODwiki/药效/视觉失联.html)和[触觉失联](/FreeODwiki/药效/触觉失联.html)。这导致一种感觉，即一个人正在部分或完全地从他们的感官输入和意识官能中分离出来。这在[中等](/FreeODwiki/文档/药物剂量分类.html)[剂量](/FreeODwiki/文档/给药剂量.html)的[解离剂](/FreeODwiki/文档/药物分类/解离剂.html)类化合物影响下几乎是一种普遍效应，比如[氯胺酮](/FreeODwiki/药物/氯胺酮.html)、[苯环利定](/FreeODwiki/药物/PCP.html)和[右美沙芬](/FreeODwiki/药物/右美沙芬.html)喵。
