# 深度感知扭曲

**深度感知扭曲**定义为一个人对视野中各种物体距离的感知发生了改变呢。[\[1\]](#cite_note-FischerHill1970-1) 在这种状态下，景色的各个层次可能会变得夸张、倾斜，或者完全重新排列哦。[\[2\]](#cite_note-KleinmanGillin1977-2)[\[3\]](#cite_note-Marsh1979-3)[\[4\]](#cite_note-4)[\[5\]](#cite_note-5) 举个栗子，环境中的层次可能会交换，背景里的东西看起来像是在前景，而前景的东西看起来像跑到了背景去啦。在其他情况下，这种扭曲会应用到整个视野，比如所有东西看起来都又小又远，或者又大又近。

这类扭曲的另一个例子是深度感知的完全丧失喵。[\[3\]](#cite_note-Marsh1979-3)[\[6\]](#cite_note-SechehayeRubin-Rabson1951-6) 当场景的不同部分看起来统一成一个平面的二维图像，而不管它们之间或与观察者之间的实际距离如何时，就会发生这种情况。

深度感知扭曲通常伴随着其他同时发生的效应，比如 [透视扭曲](/FreeODwiki/药效/透视扭曲.html) 和 [漂移](/FreeODwiki/药效/漂移.html) 呢。它们最常在 [中等](/FreeODwiki/文档/药物剂量分类.html#中等) [剂量](/FreeODwiki/文档/药物剂量分类.html) 的 [迷幻剂](/FreeODwiki/文档/药物分类/迷幻剂.html) 影响下被诱发，比如 [LSD](/FreeODwiki/药物/LSD.html)、[赛洛西宾](/FreeODwiki/药物/赛洛西宾蘑菇.html) 和 [麦斯卡林](/FreeODwiki/药物/麦斯卡林.html)。
