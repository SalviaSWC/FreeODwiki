# 肌肉紧张

**肌肉紧张**（Muscle tension）可以描述为肌肉持续的部分收缩或过度缩短，这可能会导致持续的低程度疼痛和酸痛哦。肌肉紧张通常是由压力的生理效应引起的，并可能导致背痛发作呢。

肌肉紧张通常伴随着其他的效应，比如[肌肉颤动](/FreeODwiki/药效/肌肉颤动.html)和[肌肉痉挛](/FreeODwiki/药效/肌肉痉挛.html)。它最常见于在使用了[严重](/FreeODwiki/文档/药物剂量分类.html)[剂量](/FreeODwiki/文档/给药剂量.html)的[兴奋](/FreeODwiki/文档/药物分类/兴奋剂.html)性[致幻](/FreeODwiki/文档/药物分类/致幻剂.html)化合物之后，比如[LSD](/FreeODwiki/药物/LSD.html)、[2C-E](/FreeODwiki/药物/2C-E.html)、[DOC](/FreeODwiki/药物/DOC.html)和[αMT](/FreeODwiki/药物/αMT.html)。不过呢，在使用某些[GABA](/FreeODwiki/文档/GABA.html)能[抑制剂](/FreeODwiki/文档/药物分类/抑制剂.html)如[GHB](/FreeODwiki/药物/GHB.html)和[菲尼布特](/FreeODwiki/药物/菲尼布特.html)时，也可能发生这种情况哦。
