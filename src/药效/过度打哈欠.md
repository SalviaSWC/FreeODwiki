# 过度打哈欠

**过度打哈欠**（Excessive yawning）嘛，指的是重复、加剧、过于频繁且自发的打哈欠体验，尽管这时候并没有真正的[镇静](/FreeODwiki/药效/镇静.html)感或[困倦](/FreeODwiki/药效/困倦.html)感哦。

过度打哈欠通常伴随着其他并存的效应，如[唾液分泌增加](/FreeODwiki/药效/唾液分泌增加.html)和[流涕](/FreeODwiki/药效/流涕.html)。它通常是在服用[中等](/FreeODwiki/文档/给药剂量.html#中等)[剂量](/FreeODwiki/文档/给药剂量.html)的[色胺](/FreeODwiki/文档/药物分类/色胺类物质.html)类[迷幻剂](/FreeODwiki/文档/药物分类/迷幻剂.html)时诱发的，比如[赛洛西宾蘑菇](/FreeODwiki/药物/赛洛西宾蘑菇.html)、[4-AcO-DMT](/FreeODwiki/药物/4-AcO-DMT.html)、[4-HO-MET](/FreeODwiki/药物/4-HO-MET.html)和[死藤水](/FreeODwiki/药物/死藤水.html)呢。
