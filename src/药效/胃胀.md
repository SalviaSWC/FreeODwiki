# 胃胀

| | |
|---|---|
| ![](https://psychonautwiki.org/w/thumb.php?f=Stomach_bloating.jpg&width=300) | 上图左边是正常的胃，右边是胀气的胃哦 |

**胃胀**（Stomach bloating）可以描述为一种不舒服的身体副作用，结果就是胃部暂时肿胀和膨胀，看起来有点像怀孕了呢。这种效应可能会让人有点不舒服，但并不疼痛也不危险。它的总体持续时间可能从几个小时到几天不等，多喝水可以缓解哦。

在药物使用的背景下，胃胀可能是由于摄入了胃部不习惯消化的物质从而导致胃部刺激而产生的。这些物质可能包括化学粉末或植物物质等。或者嘛，胃胀可能是该特定物质如何影响肠壁内存在的大量[血清素](/FreeODwiki/文档/血清素.html) [受体](/FreeODwiki/文档/受体激动剂.html) 的直接药理结果。

胃胀通常伴随着其他并发效应，比如[恶心](/FreeODwiki/药效/恶心.html) 和[呕吐](/FreeODwiki/药效/呕吐.html)。它最常发生在使用[严重](/FreeODwiki/文档/给药剂量.html) [剂量](/FreeODwiki/文档/给药剂量.html) 的[迷幻剂](/FreeODwiki/文档/药物分类/迷幻剂.html) 化合物时，比如[LSD](/FreeODwiki/药物/LSD.html)、[赛洛西宾蘑菇](/FreeODwiki/药物/赛洛西宾蘑菇.html) 和[麦斯卡林](/FreeODwiki/药物/麦斯卡林.html)。不过嘛，在使用[兴奋剂](/FreeODwiki/文档/药物分类/兴奋剂.html)、[阿片类药物](/FreeODwiki/文档/药物分类/阿片类药物.html) 和[镇静剂](/FreeODwiki/文档/镇静剂.html) 时，也可能会出现这种情况。
