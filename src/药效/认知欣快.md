

# 认知欣快

[◀返回](./home.html)

**认知欣快**（在语义上是[认知不快](/FreeODwiki/药效/认知不快.html)的反义词）在医学上被认为是一种认知和情感状态，在这种状态下，人会体验到强烈的幸福、兴高采烈、快乐、兴奋和喜悦的感觉哦。[1] 虽然欣快是一种[效应](/FreeODwiki/药效/home.html)（即某种物质具有致欣快作用），[2][3] 但这个词在口语中也常用来定义一种超然的幸福状态，并伴随着强烈的满足感呢。[4] 不过嘛，最近的心理学研究表明，欣快感虽然在很大程度上能促成幸福感，但不应将其直接等同于幸福哦。[5]

认知欣快通常伴随着其他并发效应，例如[躯体欣快](/FreeODwiki/药效/躯体欣快.html)和[触觉增强](/FreeODwiki/药效/触觉增强.html)。这种状态最常在[中等](/FreeODwiki/文档/药物剂量分类.html#中等)[剂量](/FreeODwiki/文档/给药剂量.html)的[阿片类药物](/FreeODwiki/文档/药物分类/阿片类药物.html)、[共情剂](/FreeODwiki/文档/药物分类/共情剂.html)、[兴奋剂](/FreeODwiki/文档/药物分类/兴奋剂.html)以及[GABA类](/FreeODwiki/文档/GABA.html)[抑制剂](/FreeODwiki/文档/药物分类/抑制剂.html)的影响下被诱发。然而，在[迷幻剂](/FreeODwiki/文档/药物分类/迷幻剂.html)、[解离剂](/FreeODwiki/文档/药物分类/解离剂.html)和[大麻类](/FreeODwiki/文档/药物分类/大麻类.html)等[致幻剂](/FreeODwiki/文档/药物分类/致幻剂.html)类化合物的影响下，也可能出现程度较轻的欣快感呢。
