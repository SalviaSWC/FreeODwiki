
# 宣泄

[◀返回](./home.html)

**宣泄**（Catharsis，来自希腊语 *katharsis*）准确的定义是一种净化哦，但关于它的确切含义还没有达成实质性的共识呢。[\[1\]](#cite_note-:1-1) 一般来说，这种效应是一种情感洞察的形式。[\[2\]](#cite_note-2)[\[3\]](#cite_note-3)[\[4\]](#cite_note-:3-4)[\[5\]](#cite_note-5)[\[6\]](#cite_note-:4-6) 这个过程通常开始时很难完全面对，有时伴随着强烈的身体感觉，通常会导致明显的 [情绪强化](/FreeODwiki/药效/情绪强化.html)、深刻的 [内省](/FreeODwiki/药效/内省.html) 以及对个人性格和过去事件的 [分析](/FreeODwiki/药效/分析能力增强.html)。[\[7\]](#cite_note-BelserAgin-Liebes2017-7) 在这种体验中，许多人描述说会重温创伤事件，目睹痛苦的记忆，拥有增强的心理意象，重温过去的经历，感受到普遍的痛苦感觉，以及释放以前被压抑的情绪。[\[4\]](#cite_note-:3-4)[\[6\]](#cite_note-:4-6)[\[7\]](#cite_note-BelserAgin-Liebes2017-7) 这种将冲突和创伤的表现整合到长期稳定记忆中的过程，经常被描述为感觉非常自然呢。

这种效应有助于帮助个体克服诸如 [成瘾](/FreeODwiki/药效/成瘾抑制.html)、[\[4\]](#cite_note-:3-4)[\[8\]](#cite_note-8) 创伤后应激障碍 (PTSD) 以及其他与遭受的过去创伤有关的个人痛苦。[\[9\]](#cite_note-:5-9) 在这段经历结束后，大多数使用者报告说，在事件结束后的几天、几周甚至几年里，生活满意度、[复原](/FreeODwiki/文档/恢复体位.html) 感和 [灵性增强](/FreeODwiki/药效/精神探索.html) 的感觉都有所增加哦。[\[6\]](#cite_note-:4-6)[\[10\]](#cite_note-10)

宣泄最常在 [治疗环境](/FreeODwiki/文档/情景与心境.html) 中，在 [中等](/FreeODwiki/文档/药物剂量分类.html#中等) [剂量](/FreeODwiki/文档/给药剂量.html) 的 [迷幻](/FreeODwiki/文档/药物分类/迷幻剂.html) 化合物影响下被诱发，如 [LSD](/FreeODwiki/药物/LSD.html)、[赛洛西宾](/FreeODwiki/药物/赛洛西宾蘑菇.html) 和 [麦斯卡林](/FreeODwiki/药物/麦斯卡林.html)。[\[9\]](#cite_note-:5-9)[\[11\]](#cite_note-11)[\[12\]](#cite_note-12)[\[13\]](#cite_note-:6-13)[\[14\]](#cite_note-:7-14)[\[15\]](#cite_note-15) 然而，它也可能在 [共情剂](/FreeODwiki/文档/药物分类/共情剂.html)、[解离剂](/FreeODwiki/文档/药物分类/解离剂.html)[\[9\]](#cite_note-:5-9)[\[13\]](#cite_note-:6-13)[\[14\]](#cite_note-:7-14) 和 [冥想](/FreeODwiki/文档/精神探索.html) 的影响下在较小程度上发生。

### 分析

宣泄在文化上占主导地位的定义——即释放负面情绪的压力——是由约瑟夫·布洛伊尔（Josef Breuer）和西格蒙德·弗洛伊德（Sigmund Freud）作为精神分析中的“液压模型”推广开来的，[\[16\]](#cite_note-:0-16)[\[17\]](#cite_note-:2-17) 还有雅各布·伯奈斯（Jakob Bernays）在语言学中的“净化理论”。[\[1\]](#cite_note-:1-1)[\[18\]](#cite_note-18)[\[19\]](#cite_note-19) 关于这些理论对愤怒情绪的不适用性存在大量讨论，表明表现出攻击性会产生更多的攻击性。[\[16\]](#cite_note-:0-16)[\[17\]](#cite_note-:2-17)[\[20\]](#cite_note-20)[\[21\]](#cite_note-21) 攻击性研究对宣泄的适用性值得怀疑，特别是关于体验这种效应所需的安全性质。[\[22\]](#cite_note-22) 值得注意的是，弗洛伊德本人在实践中放弃了这个模型，转而支持自由联想的精神分析技术。[\[23\]](#cite_note-23)
