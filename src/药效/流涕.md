# 流涕

| | |
|---|---|
| ![](https://psychonautwiki.org/w/thumb.php?f=Runny_nose.jpg&width=200) | 这是一个流鼻涕的孩子哦 |

**流涕**（Runny nose）可以描述为鼻腔充满大量粘液（通常称为“鼻涕”）的情况。这在健康人中相对经常发生，是过敏或某些疾病（如普通感冒或花粉热）的常见症状。它也可能是哭泣和暴露在寒冷温度下的副作用呢。

流涕通常伴随着其他并发效应，比如[过度打哈欠](/FreeODwiki/药效/过度打哈欠.html)、[唾液分泌增加](/FreeODwiki/药效/唾液分泌增加.html)、[痰液增多](/FreeODwiki/药效/痰液增多.html) 和[流泪](/FreeODwiki/药效/流泪.html)。它最常发生在使用[严重](/FreeODwiki/文档/给药剂量.html) [剂量](/FreeODwiki/文档/给药剂量.html) 的[色胺类物质](/FreeODwiki/文档/药物分类/色胺类物质.html) [迷幻剂](/FreeODwiki/文档/药物分类/迷幻剂.html) 化合物时，比如[赛洛西宾](/FreeODwiki/药物/赛洛西宾蘑菇.html)、[4-AcO-DMT](/FreeODwiki/药物/4-AcO-DMT.html) 和[4-HO-MET](/FreeODwiki/药物/4-HO-MET.html)。不过嘛，在使用[阿片类药物](/FreeODwiki/文档/药物分类/阿片类药物.html) [戒断反应](/FreeODwiki/文档/药物戒断反应.html)[\[1\]](#cite_note-1) 以及作为[可卡因](/FreeODwiki/药物/可卡因.html) 滥用的症状时，也可能出现这种情况[\[2\]](#cite_note-2)。
