# 畏光

**畏光**（Photophobia）指的是对光线的视觉感知产生异常的身体不耐受。作为一个医学症状，畏光并不是一种病态的恐惧或心理上的“恐惧症”，而是一种因光线照射而导致眼睛不适或疼痛的体验，有点像吸血鬼怕晒太阳那样呢。

畏光几乎总是伴随着其他并发效应，比如[瞳孔扩大](/FreeODwiki/药效/瞳孔扩大.html)，这可能会导致眼睛无法根据当前的光照水平进行自我调节，从而引发这种效应。它最常发生在使用[严重](/FreeODwiki/文档/给药剂量.html) [剂量](/FreeODwiki/文档/给药剂量.html) 的[迷幻剂](/FreeODwiki/文档/药物分类/迷幻剂.html) 类化合物时，比如[LSD](/FreeODwiki/药物/LSD.html)、[赛洛西宾蘑菇](/FreeODwiki/药物/赛洛西宾蘑菇.html) 和[麦斯卡林](/FreeODwiki/药物/麦斯卡林.html)。不过嘛，在使用某些[兴奋剂](/FreeODwiki/文档/药物分类/兴奋剂.html) 时，这种现象也可能以较轻的程度出现哦。
