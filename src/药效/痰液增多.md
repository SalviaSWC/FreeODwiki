# 痰液增多

**痰液增多**（Increased phlegm production）可以描述为喉咙和呼吸系统产生过量粘液的体验，这些粘液通常通过咳嗽排出。这通常感觉好像一个人的喉咙和口腔后部充满了某种浓稠的粘液状物质，需要定期吞咽或吐出以避免不适感哦。

痰液增多通常伴随着其他的效应，比如[过度打哈欠](/FreeODwiki/药效/过度打哈欠.html)、[唾液分泌增加](/FreeODwiki/药效/唾液分泌增加.html)、[流涕](/FreeODwiki/药效/流涕.html)和[流泪](/FreeODwiki/药效/流泪.html)。它最常见于在使用了[严重](/FreeODwiki/文档/药物剂量分类.html)[剂量](/FreeODwiki/文档/给药剂量.html)的[色胺](/FreeODwiki/文档/药物分类/色胺类物质.html)类[致幻](/FreeODwiki/文档/药物分类/致幻剂.html)化合物之后，比如[赛洛西宾](/FreeODwiki/药物/赛洛西宾蘑菇.html)、[5-MeO-MiPT](/FreeODwiki/药物/5-MeO-MiPT.html)、[4-AcO-DMT](/FreeODwiki/药物/4-AcO-DMT.html)和[4-HO-MET](/FreeODwiki/药物/4-HO-MET.html)。
