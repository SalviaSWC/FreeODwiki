# 磨牙

**磨牙**（Teeth grinding，也称为**bruxism**、**jaw clenching** 或 **gurning**）可以描述为一种强迫性且通常无法控制的磨牙冲动。在极端情况下，这可能会导致舌头、牙齿和口腔内壁受到疼痛的损伤，听起来就很痛呢。

快速缓解不舒服的磨牙的最有效方法包括使用口香糖或婴儿安抚奶嘴（也叫奶嘴）来占据下巴，以免造成伤害。补充[镁](/FreeODwiki/文件/Magnesium.html)（最好是甘氨酸镁）对于缓解磨牙也非常有效哦，建议在摄入[MDMA](/FreeODwiki/药物/MDMA.html) 或[苯丙胺](/FreeODwiki/药物/苯丙胺.html) 等兴奋剂前6小时服用200毫克，然后在摄入前1-3小时再服用一次。[\[1\]](#cite_note-1)

磨牙通常伴随着其他并发效应，比如[刺激](/FreeODwiki/药效/刺激.html) 和[清醒](/FreeODwiki/药效/清醒.html)。它最常发生在使用[中等](/FreeODwiki/文档/给药剂量.html) [剂量](/FreeODwiki/文档/给药剂量.html) 的[兴奋剂](/FreeODwiki/文档/药物分类/兴奋剂.html) 化合物时，比如[甲基苯丙胺](/FreeODwiki/药物/甲基苯丙胺.html)、[MDMA](/FreeODwiki/药物/MDMA.html)、[哌甲酯](/FreeODwiki/药物/哌甲酯.html) 和[可卡因](/FreeODwiki/药物/可卡因.html)。不过嘛，在使用某些[刺激性](/FreeODwiki/药效/刺激.html) [迷幻剂](/FreeODwiki/文档/药物分类/迷幻剂.html) 如[2C-E](/FreeODwiki/药物/2C-E.html)、[DOC](/FreeODwiki/药物/DOC.html) 和[αMT](/FreeODwiki/药物/αMT.html) 时，也可能出现这种情况。
