# 体温调节抑制

**体温调节抑制**定义为没法保持正常体温了嘛。这会导致一种感觉，一会觉得冷得难受[1]，一会又觉得热得难受，一直在波动呢。有时候，甚至可能同时觉得又冷又热，好奇怪的欸。

体温调节抑制通常伴随着其他并发效应，比如[刺激](/FreeODwiki/药效/刺激.html)和[出汗增加](/FreeODwiki/药效/出汗增加.html)哦。它最常在使用了[严重](/FreeODwiki/文档/药物剂量分类.html#严重) [剂量](/FreeODwiki/文档/给药剂量.html) 的[兴奋性](/FreeODwiki/文档/药物分类/兴奋剂.html) [致幻](/FreeODwiki/文档/药物分类/致幻剂.html) 化合物后被诱发，比如[LSD](/FreeODwiki/药物/LSD.html)、[2C-B](/FreeODwiki/药物/2C-B.html)和[αMT](/FreeODwiki/药物/αMT.html) 呢。不过嘛，在使用[兴奋剂](/FreeODwiki/文档/药物分类/兴奋剂.html)如[MDMA](/FreeODwiki/药物/MDMA.html)和[甲基苯丙胺](/FreeODwiki/药物/甲基苯丙胺.html)时，也可能发生这种情况哦。
