# 不适性身体效应

**不适性身体效应**是指任何与全身有关，且无法被归类为[心血管效应](./心血管效应.html)或[脑血管效应](./脑血管效应.html)的不舒服的[躯体效应](./不适性躯体效应.html)哦。

### 背痛

| ![Spinal_column_curvature_2011.png](../文件/Spinal_column_curvature_2011.png) |
| :---: |
| 脊柱的不同区域（弯曲）呢。 |

**背痛**可以描述为遍布背部的酸痛感。它可能表现为颈部疼痛（颈椎）、背中部疼痛（胸椎）、下背部疼痛（腰椎）或尾骨痛（尾骨或骶骨疼痛）。不过嘛，腰部是最常见的疼痛区域，因为它支撑着上半身的大部分重量。疼痛本身的程度可以从轻微可忽略到强烈且明显不适不等哦。

在精神活性物质的语境下，这种效应可能通过[触觉增强](./触觉效应.html#触觉增强)发生，这会放大原本不明显的既有病症的症状；也可能是由于[肌肉紧张](./肌肉紧张.html)造成的；或者是肾脏问题导致的\[*来源请求*]；还可能是作为用于[镇痛](./镇痛.html)的物质（如[阿片类药物](../文档/药物分类/阿片类药物.html)）的[戒断症状](./药物戒断反应.html)出现的呢。

背痛最常在[严重](../文档/药物剂量分类.html#严重)[剂量](../文档/给药剂量.html)的[刺激](../药效/刺激.html)性[致幻剂](../文档/药物分类/致幻剂.html)影响下被诱发，比如[LSD](../药物/LSD.html)、[2C-B](../药物/2C-B.html)和[麦斯卡林](../药物/麦斯卡林.html)。

### 躯体压力感

**躯体压力感**可以描述为身体不同部位自发产生的压力体验。这些压力感可能是静止固定在某个位置的，也可能出现在身体看似随机的各个点上。根据感觉的强度，这种压力感可能从不痛不痒到让人极度不舒服呢。

躯体压力感最常在[中等](../文档/药物剂量分类.html#中等)[剂量](../文档/给药剂量.html)的[刺激](../药效/刺激.html)性[致幻剂](../文档/药物分类/致幻剂.html)影响下被诱发，比如[2C-E](../药物/2C-E.html)、[DPT](../药物/DPT.html)和[5-MeO-DMT](../药物/5-MeO-DMT.html)。

### 便秘

| ![Nhs-stool-chart.gif](../文件/Nhs-stool-chart.gif) |
| :---: |
| 1-2型粪便可归类为便秘哦。 |

**便秘**可以描述为排便频率低或排便困难。它通常会导致排便疼痛和粪便细小、坚硬。通过增加水果、纤维和水的摄入量，可以减轻药物引起的便秘症状哦。泻药也可以用来暂时缓解呢。

便秘通常伴随着其他并发效应，如[恶心](./恶心.html)、[脱水](./脱水.html)和[排尿困难](./排尿困难.html)。它最常在[严重](../文档/药物剂量分类.html#严重)[剂量](../文档/给药剂量.html)的[阿片类药物](../文档/药物分类/阿片类药物.html)影响下被诱发，比如[海洛因](../药物/海洛因.html)、[曲马多](../药物/曲马多.html)、[芬太尼](../药物/芬太尼.html)和Kratom。

### 脱水

**脱水**可以描述为由于缺乏水分摄入而导致的令人不适的[口干](./口干.html)和普遍的口渴感。如果不及时处理，脱水通常会导致[谵妄](../文档/药物分类/谵妄剂.html)、意识丧失、舌头肿胀，甚至（在极端情况下）死亡呢。脱水的正式定义是生物体内体液过度流失，并伴随代谢过程的紊乱。

在较轻的程度上，药物引起的脱水通常表现为持续且令人不适的口渴感，需要不断小口喝水来维持体液水平并避免口干舌燥。在极端程度下（通常只在使用[谵妄剂](../文档/药物分类/谵妄剂.html)时发生），脱水可能会变得非常严重，以至于眼睛和粘膜都会干得发痛，导致吞咽极度困难哦。

脱水通常伴随着其他并发效应，如[口干](./口干.html)、[头痛](./头痛.html)、[头晕](./头晕.html)、[血压降低](./血压降低.html)和站立时晕厥。它最常在[中等](../文档/药物剂量分类.html#中等)[剂量](../文档/给药剂量.html)的各种化合物影响下被诱发，比如[兴奋剂](../文档/药物分类/兴奋剂.html)、[致幻剂](../文档/药物分类/致幻剂.html)、[阿片类药物](../文档/药物分类/阿片类药物.html)、[解离剂](../文档/药物分类/解离剂.html)、[谵妄剂](../文档/药物分类/谵妄剂.html)、[大麻类](../文档/药物分类/大麻类.html)、[酒精](../药物/酒精.html)和[抗精神病药](../文档/抗精神病药.html)。

#### 水中毒

要注意的是，无论一个人在药物影响下变得多么脱水，都应该小心谨慎，确保不要过量饮水，因为这会导致一种被称为水中毒的状态喵。这可能是致命的，被归类为脑功能紊乱，是由于过度补水导致体内电解质平衡被打破并超出安全限度引起的。虽然极其罕见，但也有一些著名的死亡案例明显是由药物引起的脱水导致过量饮水引发的。

人类的平均水中毒剂量大约是十升。不过呢，只要意识到这一点，注意小口喝水，避免不必要地大量饮水，就可以轻松避免水中毒啦。

### 腹泻

| ![Nhs-stool-chart.gif](../文件/Nhs-stool-chart.gif) |
| :---: |

**腹泻**是指每天至少有三次稀便或水样便的情况。它通常持续几天，并可能因体液流失而导致[脱水](./脱水.html)。随着病情加重，这可能会发展为尿量减少、肤色苍白、心率加快和反应迟钝。在[精神活性物质](../文档/药物分类/药物全索引.html)使用的背景下，已知某些化合物会诱发腹泻，或者至少会增加发生腹泻的可能性呢。[\[1\]](#cite_note-1)[\[2\]](#cite_note-2)[\[3\]](#cite_note-3) 这并不像感染引起的腹泻那样危险，因为它只会持续到药物影响消退为止哦。

腹泻通常伴随着其他并发效应，如[恶心](./恶心.html)和[脱水](./脱水.html)。它最常在[严重](../文档/药物剂量分类.html#严重)[剂量](../文档/给药剂量.html)的某些[致幻剂](../文档/药物分类/致幻剂.html)影响下被诱发，比如[死藤水](../药物/死藤水.html)、[麦斯卡林](../药物/麦斯卡林.html)和[赛洛西宾蘑菇](../药物/赛洛西宾蘑菇.html)。不过，它也可能在某些[兴奋剂](../文档/药物分类/兴奋剂.html)、[莫达菲尼](../药物/莫达菲尼.html)和[咖啡因](../药物/咖啡因.html)的影响下发生。

### 排尿困难

**排尿困难**，也称为**尿潴留**，可以描述为排尿能力下降的体验。这可能是由于尿道内疼痛的灼烧感，或者是由于膀胱失去控制，即使膀胱满了也无法排尿。

排尿困难通常伴随着其他并发效应，如[刺激](../药效/刺激.html)和[便秘](./便秘.html)。它最常在[严重](../文档/药物剂量分类.html#严重)[剂量](../文档/给药剂量.html)的[兴奋剂](../文档/药物分类/兴奋剂.html)和[阿片类药物](../文档/药物分类/阿片类药物.html)影响下被诱发，比如[海洛因](../药物/海洛因.html)、[芬太尼](../药物/芬太尼.html)、Kratom、[苯丙胺](../药物/苯丙胺.html)、[MDMA](../药物/MDMA.html)（[Leah Betts之死](https://en.wikipedia.org/wiki/Death_of_Leah_Betts)）和[4-FA](../药物/4-FA.html)。不过，它也可能在[刺激](../药效/刺激.html)性[致幻剂](../文档/药物分类/致幻剂.html)和[谵妄剂](../文档/药物分类/谵妄剂.html)的影响下发生。

### 口干

**口干**（也称为**口腔干燥症**）定义为口腔感觉干燥，通常伴有吞咽困难。这通常是[脱水](./脱水.html)的直接结果，但无论口腔实际干燥程度如何，都可能会感觉到这种感觉呢。

慢性口腔干燥症或“口干综合征”是指由于自然原因或长期使用导致口干的物质而经常和/或持续体验到口干的情况。为了治疗这种状况，通常使用像[西维美林](https://en.wikipedia.org/wiki/Cevimeline)这样刺激唾液分泌的物质。然而，重复使用此类治疗方法的一个问题是，随着时间的推移，口干症状总体上会恶化。如果身体适应了化学辅助来增加唾液分泌，停止使用该化学物质治疗可能会导致生理依赖，突然停止治疗会出现反弹效应哦。

一般来说，如果口干是由精神活性物质、药物或其组合引起的，不建议进行医疗干预。除非这种效应是慢性的，并且在受影响的人身上持续造成困扰或导致牙齿/牙龈问题，否则药物引起的口干被认为是良性的。作为一种基本的[伤害减少措施](../文档/负责任的用药索引页.html)，应该注意保持充足的水分，避免用嘴呼吸，如果这种效应变得不舒服、难以控制或即使不在精神活性物质影响下仍然持续存在，则应限制物质的使用喵。

口干通常伴随着其他并发效应，如[尿频](./尿频.html)（由于喝了过量的水）和[脱水](./脱水.html)。它最常在[中等](../文档/药物剂量分类.html#中等)[剂量](../文档/给药剂量.html)的各种化合物影响下被诱发，比如[兴奋剂](../文档/药物分类/兴奋剂.html)、[致幻剂](../文档/药物分类/致幻剂.html)、[阿片类药物](../文档/药物分类/阿片类药物.html)、[抗精神病药](../文档/抗精神病药.html)、[谵妄剂](../文档/药物分类/谵妄剂.html)、[SSRI](../文档/神经递质再摄取抑制剂.html)和[大麻类](../文档/药物分类/大麻类.html)。这也是许多物质的常见副作用，特别是多种物质联用时，这可能会产生或放大已经存在的口干体验，产生显著的相互作用呢。

### 尿频

**尿频**可以定义为比平时更频繁地需要排尿。它通常（虽然不一定）与尿失禁和总尿量大有关。然而，在其他情况下，尿频总体上只涉及正常的尿量。

尿频通常伴随着其他并发效应，如[脱水](./脱水.html)和[口干](./口干.html)，因为为了缓解口干而过量饮水会进一步放大排尿的需求。它最常在[中等](../文档/药物剂量分类.html#中等)[剂量](../文档/给药剂量.html)的各种化合物影响下被诱发，比如[兴奋剂](../文档/药物分类/兴奋剂.html)、[致幻剂](../文档/药物分类/致幻剂.html)、[解离剂](../文档/药物分类/解离剂.html)和[谵妄剂](../文档/药物分类/谵妄剂.html)。

### 出汗增加

**出汗增加**，或**多汗症**，可以描述为一种以出汗增多为特征的状况，其出汗量超过了调节体温所需的量。

出汗增加是交感神经唤醒（“战斗或逃跑”反应）的标志性症状，也是[兴奋剂](../文档/药物分类/兴奋剂.html)药物的常见效应。任何产生显著[血清素](../文档/血清素.html)、[多巴胺](../文档/多巴胺.html)或[肾上腺素](../文档/肾上腺素.html)效应的精神活性药物都可能导致出汗增加。这也是[苯二氮卓类](../文档/药物分类/苯二氮卓类物质.html)和[酒精](../药物/酒精.html)戒断的常见症状。[乙酰胆碱](../文档/乙酰胆碱.html)类药物，以及在较小程度上，[阿片类药物](../文档/药物分类/阿片类药物.html)也被认为会导致这种情况哦。\[*来源请求*]

### 痰液增多

**痰液增多**可以描述为喉咙和呼吸系统产生过量粘液的体验，通常通过咳嗽排出。这通常感觉像是一个人的喉咙和口腔后部反复充满了一种粘稠的像史莱姆一样的物质，需要定期吞下或吐出以避免不适呢。

痰液增多通常伴随着其他并发效应，如[过度打哈欠](./过度打哈欠.html)、流口水、[流涕](./流涕.html)和[流泪](./流泪.html)。它最常在[严重](../文档/药物剂量分类.html#严重)[剂量](../文档/给药剂量.html)的[色胺类](../文档/药物分类/色胺类物质.html)[致幻剂](../文档/药物分类/致幻剂.html)化合物影响下被诱发，比如[赛洛西宾](../药物/赛洛西宾蘑菇.html)、[5-MeO-MiPT](../药物/5-MeO-MiPT.html)、[4-AcO-DMT](../药物/4-AcO-DMT.html)和[4-HO-MET](../药物/4-HO-MET.html)。

### 瘙痒感

**瘙痒感**是引起一个人想要抓挠皮肤的欲望或反射的感觉。在较轻的程度上，瘙痒感可能是一种容易被忽视的细微烦恼。然而，在较高的程度上，瘙痒感可能会变得非常强烈，令人难以置信地不舒服，甚至可能导致人因反复抓挠动作而损伤皮肤呢。

瘙痒感最常在[严重](../文档/药物剂量分类.html#严重)[剂量](../文档/给药剂量.html)的[阿片类药物](../文档/药物分类/阿片类药物.html)影响下被诱发，比如[海洛因](../药物/海洛因.html)、[芬太尼](../药物/芬太尼.html)、[曲马多](../药物/曲马多.html)和Kratom。这是由于阿片类药物激活[组胺](../文档/组胺.html)受体并触发组胺释放的方式造成的。在物质使用的情况下，对抗瘙痒感的一个有效技巧是服用[抗组胺药](../文档/受体拮抗剂.html)，如[苯海拉明](../药物/苯海拉明.html)（**DPH**，Benadryl）。

### 肌肉痉挛

**肌肉痉挛**可以描述为肌肉不自主的暂时收缩或过度缩短，这可能会导致严重的酸痛。这些肌肉痉挛通常发作得很突然，而痉挛通常会在几秒钟或几分钟内自行缓解哦。

肌肉痉挛通常伴随着其他并发效应，如[肌肉颤动](./肌肉颤动.html)和[刺激](../药效/刺激.html)。它们最常在[严重](../文档/药物剂量分类.html#严重)[剂量](../文档/给药剂量.html)的[刺激](../药效/刺激.html)性[致幻剂](../文档/药物分类/致幻剂.html)影响下被诱发，比如[LSD](../药物/LSD.html)、[2C-E](../药物/2C-E.html)、[DOC](../药物/DOC.html)和[αMT](../药物/αMT.html)。不过，它们也可能在某些[GABA类](../文档/GABA.html)[抑制剂](../文档/药物分类/抑制剂.html)如[GHB](../药物/GHB.html)和[菲尼布特](../药物/菲尼布特.html)的影响下发生。

可以通过补充剂来避免肌肉痉挛，包括：镁（最好是甘氨酸镁）和/或维生素B复合物。\[*来源请求*]

### 肌肉颤动

**肌肉颤动**可以描述为肌肉群细微且局部的颤抖感。这些震动通常强大到可以透过皮肤看到。它们引起的感觉在某些情况下可能令人不舒服，但通常体验是中性的呢。

肌肉颤动最常在[中等](../文档/药物剂量分类.html#中等)[剂量](../文档/给药剂量.html)的[刺激](../药效/刺激.html)性[致幻剂](../文档/药物分类/致幻剂.html)影响下被诱发，比如[LSD](../药物/LSD.html)、[2C-E](../药物/2C-E.html)、[DOC](../药物/DOC.html)和[αMT](../药物/αMT.html)。不过，它也可能在传统[兴奋剂](../文档/药物分类/兴奋剂.html)的影响下发生。

### 肌肉紧张

**肌肉紧张**可以描述为肌肉长时间的部分收缩或过度缩短，这可能导致持续的低水平酸痛。肌肉紧张通常是由压力的生理效应引起的，并可能导致背痛发作。

肌肉紧张通常伴随着其他并发效应，如[肌肉颤动](./肌肉颤动.html)和[肌肉痉挛](./肌肉痉挛.html)。它最常在[严重](../文档/药物剂量分类.html#严重)[剂量](../文档/给药剂量.html)的[刺激](../药效/刺激.html)性[致幻剂](../文档/药物分类/致幻剂.html)影响下被诱发，比如[LSD](../药物/LSD.html)、[2C-E](../药物/2C-E.html)、[DOC](../药物/DOC.html)和[αMT](../药物/αMT.html)。不过，它也可能在某些[GABA类](../文档/GABA.html)[抑制剂](../文档/药物分类/抑制剂.html)如[GHB](../药物/GHB.html)和[菲尼布特](../药物/菲尼布特.html)的影响下发生。

### 恶心

**恶心**可以描述为上腹部的不安和不适感，并伴有不自主的呕吐冲动。[\[4\]](#cite_note-4)[\[5\]](#cite_note-5)[\[6\]](#cite_note-6) 它通常（但不总是）发生在呕吐之前。这种效应通常发生在体验的开始阶段，并随着药效达峰而消散。

在物质使用的背景下，恶心和呕吐可能是由于食用了胃不习惯消化的物质（如化学粉末或植物物质）刺激胃部而发生的。或者，恶心可能是特定物质影响大脑的直接药理结果。如果是这种情况，恶心就与体验本身密不可分，无论[给药途径](../文档/给药途径.html)如何，都可能在不同程度上发生哦。

恶心通常伴随着其他并发效应，如[胃胀](./胃胀.html)、[胃痉挛](./胃痉挛.html)和[头晕](./头晕.html)。它最常在[严重](../文档/药物剂量分类.html#严重)[剂量](../文档/给药剂量.html)的各种化合物影响下被诱发，比如[致幻剂](../文档/药物分类/致幻剂.html)、[阿片类药物](../文档/药物分类/阿片类药物.html)、[GABA类药物](../文档/GABA.html)、[谵妄剂](../文档/药物分类/谵妄剂.html)、[解离剂](../文档/药物分类/解离剂.html)和[兴奋剂](../文档/药物分类/兴奋剂.html)。

### 呕吐

**呕吐**，也就是把胃里的东西通过嘴（有时是鼻子）不自主地、强力地吐出来。这种效应通常发生在物质效应的顶峰期。一旦结束，通常会大大减轻身体的副作用呢。例如，在许多致幻化合物的影响下，人们通常会觉得吐完之后旅程变得愉快多了，因为不舒服的胃部症状突然消退了嘛。

值得注意的是，呕吐后不应立即刷牙哦。这是因为胃酸的腐蚀性加上刷牙的摩擦力，如果长期重复，会对牙齿造成永久性损伤。相反，应该用水、漱口水、冲牙器或小苏打水混合物（以中和酸性）漱口呢。

### 眼球滑动

**眼球滑动**可以描述为一种抑制眼睛协调和控制的躯体效应，使人无法保持眼球静止。这导致眼球向各个方向连续移动，并伴有无法注视任何特定点的感觉。

眼部疾病[眼球震颤](./视物振动.html)会导致患者出现非常相似的症状，尽管眼球震颤的移动频率通常要高得多。

眼球滑动通常伴随着其他并发效应，如[视觉锐度抑制](./视觉锐度抑制.html)和[复视](./复视.html)。它最常在[严重](../文档/药物剂量分类.html#严重)[剂量](../文档/给药剂量.html)的[解离剂](../文档/药物分类/解离剂.html)影响下被诱发，比如[氯胺酮](../药物/氯胺酮.html)、[PCP](../药物/PCP.html)和[右美沙芬](../药物/右美沙芬.html)。不过，在极高剂量的[GABA类](../文档/GABA.html)[抑制剂](../文档/药物分类/抑制剂.html)影响下，也可能发生较轻程度的眼球滑动。

### 畏光

**畏光**可以描述为对光线视觉感知的异常身体不耐受。作为一种医学症状，畏光不是一种病态的恐惧或心理恐惧，而是由于光线照射导致眼睛不适或疼痛的体验。

畏光几乎总是伴随着其他并发效应，如[瞳孔扩大](./瞳孔扩大.html)，这可能会通过使眼睛无法根据当前光线水平进行调节来触发这种效应。它最常在[严重](../文档/药物剂量分类.html#严重)[剂量](../文档/给药剂量.html)的[致幻剂](../文档/药物分类/致幻剂.html)影响下被诱发，比如[LSD](../药物/LSD.html)、[赛洛西宾](../药物/赛洛西宾蘑菇.html)和[麦斯卡林](../药物/麦斯卡林.html)。不过，在某些[兴奋剂](../文档/药物分类/兴奋剂.html)的影响下，也可能发生较轻程度的畏光。

### 躯体疲劳

**躯体疲劳**可以描述为一种全身精疲力竭的感觉。这种效应的强度和持续时间通常取决于所消耗的物质及其剂量。睡眠不足或缺乏食物等各种因素也会进一步加剧这种感觉。这种疲惫感涉及各种各样的症状，通常包括以下部分或全部效应：

*   **[镇静](./镇静.html)**
*   **[躯体沉重感](./躯体沉重感.html)**
*   **[性欲减退](./性欲减退.html)**
*   **[运动控制丧失](./运动控制丧失.html)**
*   **[嗜睡](./镇静.html)**

感到疲劳的人可能会发现难以完成身体动作，甚至可能无法下床或做日常家务。通常可以通过休息或睡眠来缓解呢。

躯体疲劳通常伴随着其他并发效应，如[认知疲劳](./认知效应.html)。它最常在[中等](../文档/药物剂量分类.html#中等)[剂量](../文档/给药剂量.html)的[抗精神病药](../文档/抗精神病药.html)影响下被诱发，比如[喹硫平](../药物/喹硫平.html)、[氟哌啶醇](../文档/抗精神病药.html)和[利培酮](../文档/抗精神病药.html)。不过，它也可能出现在许多[抑制剂](../文档/药物分类/抑制剂.html)的[戒断](../药物戒断反应.html)症状期间，以及许多[兴奋剂](../文档/药物分类/兴奋剂.html)的[药效褪去](../药效/home.html#药效褪去)期。

### 呼吸抑制

**呼吸抑制**（也称为**换气不足**）可以描述为呼吸冲动减少，根据其强度可能是致命的。在相对安全的水平下，这种效应通常会导致“叹气”样的呼吸模式，即深呼吸被异常长的停顿隔开。然而，在较高水平下，一个人可能会完全停止呼吸，如果不立即治疗，会迅速致命。

这种效应能够在以下4种不同强度水平上表现出来：

1.  **轻微呼吸抑制** - 在最低水平，呼吸抑制通常微细到难以察觉，并伴有轻微的[镇静](./镇静.html)。
2.  **中度呼吸抑制** - 在这个水平，人会意识到自己每分钟的呼吸次数比平时少。这种程度的呼吸抑制并不让人难受，也不会导致呼吸短促、挣扎或受损。
3.  **重度呼吸抑制** - 在这个水平，人的呼吸频率会明显减慢，导致人感觉呼吸异常、气短，吸入的空气不足。这会迫使人将注意力转移到手动调节呼吸上，以免感到极度不适。此时，通常也会出现极度的[镇静](./镇静.html)，如果睡着了，人可能会挣扎着醒来，大口喘气。这个阶段经常会出现困惑、恐慌和[焦虑](../药效/焦虑.html)，进一步增加了那种感觉呼吸会因睡眠或注意力不集中而完全停止的强烈感觉。
4.  **呼吸衰竭** - 在最高水平，人的氧气和二氧化碳水平会受到危险的损害。人会陷入半昏迷状态，完全失去意识，陷入昏迷，或完全停止呼吸。皮肤、指甲或嘴唇也可能呈现蓝色。如果没有立即的医疗救助，这种程度的呼吸衰竭很可能是致命的。[\[7\]](#cite_note-7) 阿片类药物和巴比妥类药物，在过量使用或与其他抑制剂联用时，因导致此类死亡而臭名昭著。

呼吸抑制通常伴随着其他并发效应，如[镇静](./镇静.html)和[嗜睡](./镇静.html)。它最常在[严重](../文档/药物剂量分类.html#严重)[剂量](../文档/给药剂量.html)的[抑制剂](../文档/药物分类/抑制剂.html)影响下被诱发，特别是[阿片类药物](../文档/药物分类/阿片类药物.html)，如[海洛因](../药物/海洛因.html)和[芬太尼](../药物/芬太尼.html)，或[GABA类药物](../文档/GABA.html)，如[酒精](../药物/酒精.html)和[GHB](../药物/GHB.html)。值得注意的是，当与哪怕少量的其他类别的抑制剂联用时，原本安全的剂量也可能变得致命。例如，[苯二氮卓类](../文档/药物分类/苯二氮卓类物质.html)与[阿片类药物](../文档/药物分类/阿片类药物.html)联用是导致致命呼吸抑制的极常见原因。因此，强烈不建议在任何剂量范围内联用这些抑制剂哦。

#### 治疗

为了防止死亡，如果发生严重的呼吸抑制，建议立即联系紧急医疗服务。如果是由阿片类药物过量引起的，应使用阿片受体拮抗剂，如[纳洛酮](../药物/纳洛酮.html)。许多伤害减少组织免费向使用者提供纳洛酮，或者可以在药店购买。除非并发了其他[抑制剂](../文档/药物分类/抑制剂.html)的影响，否则纳洛酮会迅速逆转呼吸抑制。

对于其他药物引起的呼吸抑制，可能需要住院治疗和机械呼吸机的辅助呢。

### 不宁腿

**不宁腿**（也称为**不宁腿综合征**或**RLS**）在医学上定义为一种不可抗拒的想要移动身体以停止不适或奇怪感觉的冲动。[\[8\]](#cite_note-ICD-11-Restless-legs-syndrome-8) 它最常影响腿部，但也可能影响手臂、躯干和头部。在这种状态下，移动受影响的身体部位可以减轻不适感，提供暂时的缓解。

RLS的感觉范围包括疼痛、肌肉酸痛、“抓不到的痒”、令人不快的“停不下来的发痒”，甚至是爬行感。这种感觉通常在安静清醒时开始或加剧，比如放松、阅读、学习或试图睡觉时。

不宁腿综合征最常在许多[抑制剂](../文档/药物分类/抑制剂.html)的[戒断](../药物戒断反应.html)症状期间被诱发，如[阿片类药物](../文档/药物分类/阿片类药物.html)或[苯二氮卓类](../文档/药物分类/苯二氮卓类物质.html)，以及许多[兴奋剂](../文档/药物分类/兴奋剂.html)的[药效褪去](../药效/home.html#药效褪去)期，如[甲基苯丙胺](../药物/甲基苯丙胺.html)、[可卡因](../药物/可卡因.html)和[MDMA](../药物/MDMA.html)。不过，它也可能在[谵妄剂](../文档/药物分类/谵妄剂.html)如[苯海拉明](../药物/苯海拉明.html)和[曼陀罗](../药物/曼陀罗.html)的影响下发生。

### 皮肤潮红

皮肤潮红最常在[严重](../文档/药物剂量分类.html#严重)[剂量](../文档/给药剂量.html)的[阿片类药物](../文档/药物分类/阿片类药物.html)影响下被诱发，比如[海洛因](../药物/海洛因.html)、[曲马多](../药物/曲马多.html)、[芬太尼](../药物/芬太尼.html)和Kratom。不过，它也可能在[酒精](../药物/酒精.html)、某些[致幻剂](../文档/药物分类/致幻剂.html)如[5-MeO-DMT](../药物/5-MeO-DMT.html)和[兴奋剂](../文档/药物分类/兴奋剂.html)如[咖啡因](../药物/咖啡因.html)的影响下发生。

**胃胀**可以描述为一种不舒服的躯体副作用，导致胃部暂时胀大和扩张，看起来有点像怀孕。这种效应可能会让人感到中度不适，但并不痛苦也不危险。其总持续时间可能从几个小时到几天不等，多喝水可以缓解哦。

在物质使用的背景下，胃胀可能是由于食用了胃不习惯消化的物质（如化学粉末或植物物质）刺激胃部而发生的。或者，胃胀可能是特定物质影响肠壁内大量[血清素](../文档/血清素.html)[受体](../文档/神经递质.html)的直接药理结果。

胃胀通常伴随着其他并发效应，如[恶心](./恶心.html)和[呕吐](./呕吐.html)。它最常在[严重](../文档/药物剂量分类.html#严重)[剂量](../文档/给药剂量.html)的[致幻剂](../文档/药物分类/致幻剂.html)影响下被诱发，比如[LSD](../药物/LSD.html)、[赛洛西宾](../药物/赛洛西宾蘑菇.html)和[麦斯卡林](../药物/麦斯卡林.html)。不过，它也可能在[兴奋剂](../文档/药物分类/兴奋剂.html)、[阿片类药物](../文档/药物分类/阿片类药物.html)和[抑制剂](../文档/药物分类/抑制剂.html)的影响下发生。

### 胃痉挛

**胃痉挛**可以描述为胃部突然发生的剧烈疼痛或不适感。

在物质使用的背景下，胃痉挛可能是由于食用了胃不习惯消化的物质（如化学粉末或植物物质）刺激胃部而发生的。或者，痉挛可能是特定物质影响大脑的直接药理结果。如果是这种情况，胃痉挛就与体验本身密不可分，无论给药途径如何，都可能在不同程度上发生呢。

胃痉挛通常伴随着其他并发效应，如[胃胀](./胃胀.html)、[恶心](./恶心.html)和[呕吐](./呕吐.html)。它们最常在[严重](../文档/药物剂量分类.html#严重)[剂量](../文档/给药剂量.html)的各种化合物影响下被诱发，比如[致幻剂](../文档/药物分类/致幻剂.html)、[阿片类药物](../文档/药物分类/阿片类药物.html)、[GABA类药物](../文档/GABA.html)和[谵妄剂](../文档/药物分类/谵妄剂.html)。

### 磨牙

**磨牙**（也称为**bruxism**，**咬牙**和**gurning**）可以描述为一种强迫性的、通常无法控制的磨牙冲动。在极端情况下，这可能会导致舌头、牙齿和脸颊内侧的疼痛损伤。

快速缓解不舒服的磨牙的最有效方法包括使用口香糖或婴儿安抚奶嘴（也叫dummy）来占用下巴，避免造成损伤。镁（最好是甘氨酸镁）在摄入[MDMA](../药物/MDMA.html)或[苯丙胺](../药物/苯丙胺.html)等兴奋剂前6小时服用200mg，并在摄入前1-3小时再次服用，对于缓解磨牙也非常有效哦。[\[11\]](#cite_note-11)

磨牙通常伴随着其他并发效应，如[刺激](../药效/刺激.html)和[清醒](../药效/刺激.html)。它最常在[中等](../文档/药物剂量分类.html#中等)[剂量](../文档/给药剂量.html)的[兴奋剂](../文档/药物分类/兴奋剂.html)影响下被诱发，比如[甲基苯丙胺](../药物/甲基苯丙胺.html)、[MDMA](../药物/MDMA.html)、[哌甲酯](../药物/哌甲酯.html)和[可卡因](../药物/可卡因.html)。不过，它也可能在某些[刺激](../药效/刺激.html)性[致幻剂](../文档/药物分类/致幻剂.html)如[2C-E](../药物/2C-E.html)、[DOC](../药物/DOC.html)和[αMT](../药物/αMT.html)的影响下发生。

### 暂时性勃起功能障碍

**暂时性勃起功能障碍**可以描述为难以勃起并维持足够的硬度以进行性交。这种效应在某些物质的影响下会以不同程度发生。

暂时性勃起功能障碍通常伴随着其他并发效应，如[刺激](../药效/刺激.html)、[排尿困难](./排尿困难.html)和[体温调节抑制](./体温调节抑制.html)，这会进一步加剧问题。它最常在[严重](../文档/药物剂量分类.html#严重)[剂量](../文档/给药剂量.html)的[刺激](../药效/刺激.html)性化合物影响下被诱发，比如传统[兴奋剂](../文档/药物分类/兴奋剂.html)和某些[致幻剂](../文档/药物分类/致幻剂.html)。不过，它也可能在[阿片类药物](../文档/药物分类/阿片类药物.html)、[解离剂](../文档/药物分类/解离剂.html)、[GABA类药物](../文档/GABA.html)和[谵妄剂](../文档/药物分类/谵妄剂.html)的影响下发生。

### 视物振动

| ![Vibrating_vision_2.gif](../文件/Vibrating_vision_2.gif) |
| :---: |
| 不自主的眼球运动 |

**视物振动**，也称为**眼球震颤**，可以描述为持续、快速的不自主眼球运动，眼睛快速地左右移动，以至于人的视觉开始振动和模糊。这会严重损害视力，导致功能下降，难以完成需要视力的基本任务呢。

视物振动通常伴随着其他并发效应，如[刺激](../药效/刺激.html)和[思维加速](./思维加速.html)，并被这些效应增强。它最常在[严重](../文档/药物剂量分类.html#严重)[剂量](../文档/给药剂量.html)的[兴奋剂](../文档/药物分类/兴奋剂.html)影响下被诱发，比如[MDMA](../药物/MDMA.html)、[苯丙胺](../药物/苯丙胺.html)和[4-FA](../药物/4-FA.html)。

### 流泪

| ![Water_eyes.jpg](../文件/Water_eyes.jpg) |
| :---: |
| 流泪的图片示例 |

**流泪**可以描述为一种躯体效应，导致泪管持续不自主地流泪、哭泣和充满泪水。体验这种效应通常会让人觉得自己在无缘无故地哭泣，尽管完全没有通常预期的相关情绪。

流泪通常伴随着其他并发效应，如[过度打哈欠](./过度打哈欠.html)和[流涕](./流涕.html)。它最常在[中等](../文档/药物剂量分类.html#中等)[剂量](../文档/给药剂量.html)的[致幻](../文档/药物分类/致幻剂.html)[色胺类](../文档/药物分类/色胺类物质.html)化合物影响下被诱发，比如[赛洛西宾](../药物/赛洛西宾蘑菇.html)、[4-AcO-DMT](../药物/4-AcO-DMT.html)和[4-HO-MET](../药物/4-HO-MET.html)。
