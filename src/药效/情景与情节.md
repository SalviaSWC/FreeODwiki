

# 情景和情节

[◀返回](./home.html)

**情景和情节**（Scenarios and plots）是指在[外部幻觉](/FreeODwiki/药效/外部幻觉.html)和[内部幻觉](/FreeODwiki/药效/内部幻觉.html)中发生的情况、故事、场景和事件呢。这些情景的表现方式几乎与普通[梦](/FreeODwiki/药效/梦.html)境中发生的情节和场景一模一样，而且常常包含认知[妄想](/FreeODwiki/药效/妄想.html)，导致一个人将这些情节误认为是现实生活中发生的事件哦。不过在极少数情况下，它们也会立即被识别为仅仅是幻觉，而不是真实的生活事件嘛。

在这个效应期间，构成标准幻觉状态的典型组成部分（[场景、布景和景观](/FreeODwiki/药效/场景、布景和景观.html)以及[自主实体](/FreeODwiki/药效/自主实体.html)）开始以一种相互配合的方式行动，从而导致在幻觉内部体验到事件的发生呢。这些通常被感知为线性和连贯的情节，按照逻辑顺序发生，通过正常的因果关系引出其他事件哦。然而，它们同样可能表现得完全荒谬和不连贯呢。例如，它们可能由非线性或自发的事件组成，这些事件能够快速连续地结束、开始并在彼此之间反复切换嘛。

这些幻觉情节可能包括完全不同于现实世界的全新体验，或者是旧的体验（如准确的记忆回放），或者是两者的结合哦。然而，就其确切内容而言，这种效应无法以一种全面的方式定义，就像人们无法预测未知文学作品和电影的确切情节一样呢。不过嘛，它们可以被概括为基本的遭遇，通常包括访问一个包含互动对象和[自主实体](/FreeODwiki/药效/自主实体.html)的[场景](/FreeODwiki/药效/场景、布景和景观.html)。值得注意的是，由于这种效应，人们可能陷入的情况既可能是积极的体验，也可能是消极的，这取决于其内容以及经历这些的人哦。

幻觉情节和场景通常感觉像是实时经历的呢。例如，当幻觉中过去了20秒时，现实世界中通常也过去了相同的时间。但在其他时候，[时间扭曲](/FreeODwiki/药效/时间扭曲.html)可能会发生，导致情节和场景感觉像是持续了数天、数周、数月、数年、亿万年，甚至是无限长的时间哦。

情景和情节最常在[严重](/FreeODwiki/文档/药物剂量分类.html#严重) [剂量](/FreeODwiki/文档/给药剂量.html)的[致幻](/FreeODwiki/文档/药物分类/致幻剂.html)化合物影响下被诱发，比如[迷幻剂](/FreeODwiki/文档/药物分类/迷幻剂.html)、[解离剂](/FreeODwiki/文档/药物分类/解离剂.html)和[谵妄剂](/FreeODwiki/文档/药物分类/谵妄剂.html)呢。然而，它们也可能不太常见地在[兴奋剂精神病](/FreeODwiki/文档/兴奋剂精神病.html)和[睡眠剥夺](/FreeODwiki/药效/睡眠剥夺.html)的影响下发生哦。

# 联觉

[◀返回](./home.html)

**联觉**（Synaesthesia，也拼写为 synesthesia 或 synæsthesia）是一种状况，其中一个感官或认知通路的刺激会导致第二个感官或认知通路产生自动的、不自主的体验呢。[\[1\]](#cite_note-1) 例如，在这个体验过程中，一个人可能会开始看见音乐、尝到颜色、听到气味，或者是任何其他潜在的感官组合哦。[\[2\]](#cite_note-2) 在最高层级上，联觉会变得包罗万象，以至于每一种感官都完全交织在一起，并通其他所有的感官来体验嘛。这是人类感知的一种完全融合，通常在体验时被认为是非常深刻的呢。值得注意的是，相当大比例的人口在日常生活中不使用药物也会在不同程度上体验到联觉哦。[\[3\]](#cite_note-3)[\[4\]](#cite_note-4)

联觉通常在[严重](/FreeODwiki/文档/药物剂量分类.html#严重) [剂量](/FreeODwiki/文档/给药剂量.html)的[迷幻](/FreeODwiki/文档/药物分类/迷幻剂.html)化合物影响下被诱发，[\[5\]](#cite_note-5) 比如[LSD](/FreeODwiki/药物/LSD.html)、[赛洛西宾](/FreeODwiki/药物/赛洛西宾蘑菇.html)和[麦斯卡林](/FreeODwiki/药物/麦斯卡林.html)呢。然而，在[兴奋性](/FreeODwiki/文档/兴奋剂.html)迷幻剂如[2C-x](/FreeODwiki/文档/药物分类/2C-x.html)、[DOx](/FreeODwiki/文档/药物分类/DOx.html)和[NBOMe](/FreeODwiki/文档/药物分类/NBOMe类物质.html)系列的影响下，这种体验似乎最为常见哦。
