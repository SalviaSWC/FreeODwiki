# 视觉锐度抑制

| | |
|---|---|
| ![](../文件/London_acuity_suppression.jpg) | **由 [Josikins](../文件/User_Josikins) 创作的模糊的伦敦** - 这张图片准确地复制了在伦敦市中心金禧人行桥上看到的锐度抑制效果呢。 |

**视觉锐度抑制**定义为视力清晰度和锐度的下降，导致视觉变得部分或完全模糊不清呢。[1] 这种效应可能会影响一个人的整个视野，也可能只影响特定部分哦。这种锐度抑制的体验就像是透过失焦的镜头看东西，会降低人们在外部环境中能看到的细节嘛。根据强度的不同，这通常会导致需要视力的基本功能和任务执行能力下降欸。

视觉锐度抑制通常伴随着其他并发效应，比如[复视](/FreeODwiki/药效/复视.html)和[模式识别抑制](/FreeODwiki/药效/模式识别抑制.html)呢。这种效应最常在[中等](/FreeODwiki/文档/给药剂量.html#中等)剂量的[抑制剂](/FreeODwiki/文档/药物分类/抑制剂.html)和[解离剂](/FreeODwiki/文档/药物分类/解离剂.html)类物质影响下诱发，例如[酒精](/FreeODwiki/药物/酒精.html)[2]、[喹硫平](/FreeODwiki/药物/喹硫平.html)、[氯胺酮](/FreeODwiki/药物/氯胺酮.html)和[右美沙芬](/FreeODwiki/药物/右美沙芬.html)哦。
