# 视觉拉伸

**视觉拉伸**是指视野发生扭曲，看起来就像在水平或者垂直方向上被拉长了一样。这种效果的强度各不相同，从轻微得几乎看不见到极其强烈、包罗万象，甚至能完全剥夺视力哦。在更强烈的等级下，它能把视野拉伸到看似无限的长度，这时候视野通常会变成一条细细的感官数据带，两边要么是空荡荡的，要么围绕着微妙的[几何](/FreeODwiki/药效/几何.html)图形，是不是很夸张呀？

视觉拉伸最常在使用了[严重](/FreeODwiki/文档/药物剂量分类.html#严重)剂量的、起效迅速的[致幻](/FreeODwiki/文档/药物分类/致幻剂.html)化合物时被诱发，比如[氧化亚氮](/FreeODwiki/药物/氧化亚氮.html)、[DMT](/FreeODwiki/药物/DMT.html)和[鼠尾草素甲](/FreeODwiki/药物/鼠尾草素甲.html)。这种情况在[自我死](/FreeODwiki/文档/精神探索.html)即将来临的那一刻特别明显呢，喵。
