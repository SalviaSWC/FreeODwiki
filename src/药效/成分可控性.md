# 成分可控性

**成分可控性**是一种罕见的体验，指的是获得了部分或完全的有意识控制权，可以控制其他当前正在发生的[主观效应](/FreeODwiki/文档/药效/home.html)的细节、内容和强度哦。这发生的这种方式，与训练有素的清醒梦者在普通[梦境](/FreeODwiki/药效/梦境.html)中体验到的控制水平极为相似呢。

举个栗子，这种状态可以让一个人能够手动操纵和引导他们当前的视觉效应，允许他们凭借意志让特定的组件出现、停止、增加、减少或改变其行为喵。它还可以允许那个人通过选择和控制[主观效应索引](/FreeODwiki/文档/药效/home.html)中可能存在的任何特定组件组合的存在和强度，来操纵他们的认知或身体状态。然而，值得注意的是，这种体验是否反映了对观察到的效应的真正控制是值得怀疑的，因为它可能仅仅是一种给人控制感觉和感知的[妄想](/FreeODwiki/药效/妄想.html)罢了。

成分可控性与[循环放大](/FreeODwiki/药效/循环放大.html)和[正念](/FreeODwiki/药效/正念.html)一起发生，通常是在[严重](/FreeODwiki/文档/药物剂量分类.html#common) [剂量](/FreeODwiki/文档/给药剂量.html) 的[致幻](/FreeODwiki/文档/药物分类/致幻剂.html)化合物（如[LSD](/FreeODwiki/药物/LSD.html)、[赛洛西宾](/FreeODwiki/药物/赛洛西宾蘑菇.html)、[麦斯卡林](/FreeODwiki/药物/麦斯卡林.html)和[大麻](/FreeODwiki/药物/大麻.html)）的影响下诱发的。
