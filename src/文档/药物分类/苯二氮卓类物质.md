# 苯二氮卓类物质

![Skull and crossbones darktextred2.png](/FreeODwiki/文件/Skull_and_crossbones_darktextred2.png)

**虽然很啰嗦，但是强烈不建议把苯二氮卓类物质与 [阿片类药物](/FreeODwiki/文档/药物分类/阿片类药物.html)、[巴比妥类物质](/FreeODwiki/文档/药物分类/巴比妥类物质.html)、[加巴喷丁类物质](/FreeODwiki/文档/药物分类/加巴喷丁类物质.html)、[噻吩二氮卓类物质](/FreeODwiki/文档/药物分类/噻吩二氮卓类物质.html)、[酒精](/FreeODwiki/药物/酒精.html) 或其他 [GABA能物质](/FreeODwiki/文档/GABA.html) 等 [抑制剂](/FreeODwiki/文档/药物分类/抑制剂.html) 混在一起吃哦，因为可能会发生致命的 [药物过量](/FreeODwiki/文档/药物过量.html)。[\[1\]](#cite_note-tripsit-1)**

**苯二氮卓类物质**（通常被称为 **benzos**）是一类作为中枢神经系统 [抑制剂](/FreeODwiki/文档/药物分类/抑制剂.html) 的精神活性物质喵。这些物质通过结合并作用于主要抑制性 [神经递质](/FreeODwiki/文档/神经递质.html) **γ-氨基丁酸** ([GABA](/FreeODwiki/文档/GABA.html)) 的受体，来放大其效率和效应。[\[2\]](#cite_note-Haefely1984-2)

苯二氮卓类物质的特征效应包括 [焦虑抑制](/FreeODwiki/药效/焦虑抑制.html)、[镇静](/FreeODwiki/药效/镇静.html)、[肌肉松弛](/FreeODwiki/药效/肌肉松弛.html)、[去抑制](/FreeODwiki/药效/去抑制.html)、[困倦](/FreeODwiki/药效/困倦.html) 和 [失忆](/FreeODwiki/药效/失忆.html) 呢。在医学背景下，短效苯二氮卓类药物通常被推荐用于治疗失眠或急性惊恐发作，而长效药物则被推荐用于治疗焦虑症哦。[\[3\]](#cite_note-3)

对于长期大量使用的人来说，[突然停用苯二氮卓类药物](/FreeODwiki/文档/药物戒断反应.html) 可能会引起癫痫发作（在某些情况下可能危及生命[\[4\]](#cite_note-4)）。因此，建议在一段时间内逐渐减少每日剂量，而不是突然停止——这种技术被称为 [减量戒断法](/FreeODwiki/文档/减量戒断法.html) 喵。[\[5\]](#cite_note-5)

## 历史

| :---: |
| [放大](/FreeODwiki/文件/Chlordiazepoxide.png) |
| 氯氮卓的分子结构，这是第一种苯二氮卓类药物。它由 Hoffmann–La Roche 于 1960 年以 *Librium* 品牌上市。 |

第一种苯二氮卓类药物氯氮卓（*Librium*）是 Leo Sternbach 在1955年为 Hoffmann–La Roche 工作开发镇静剂时合成的。最初制备的化合物的药理学特性令人失望，Sternbach 放弃了这个项目。

两年后，即1957年4月，同事 Earl Reeder 在实验室进行春季大扫除时，注意到了那个已停产项目中遗留下来的“漂亮的结晶”化合物。这种化合物后来被命名为氯氮卓，由于 Sternbach 专注于其他问题，它在1955年没有经过测试。研究人员预计药理学结果会是阴性的，并希望发表与化学相关的发现，于是将其提交进行标准的动物测试。

然而，该化合物显示出非常强的 [镇静](/FreeODwiki/药效/镇静.html)、[抗惊厥](/FreeODwiki/文档/抗惊厥.html) 和 [肌肉松弛](/FreeODwiki/药效/肌肉松弛.html) 效应。这些令人印象深刻的临床发现导致它在1960年以品牌名 *Librium* 在全世界迅速推出。[\[6\]](#cite_note-pmid34039-6)[\[7\]](#cite_note-Miller-Gold-7) 继氯氮卓之后，Hoffmann–La Roche 于1963年推出了 [地西泮](/FreeODwiki/药物/地西泮.html)，品牌名为 *Valium*，这两种药物一度成为商业上最成功的药物。苯二氮卓类药物的引入导致了 [巴比妥类物质](/FreeODwiki/文档/药物分类/巴比妥类物质.html) 处方的减少，到了1970年代，它们已在很大程度上取代了老式药物用于镇静和 [催眠](/FreeODwiki/文档/催眠药.html) 用途喵。[\[8\]](#cite_note-isbn0-19-517668-5-8)

## 社会与文化

这组新药最初受到了医学界的乐观欢迎，但逐渐引发了担忧；特别是依赖性的风险在1980年代变得明显。苯二氮卓类药物有着独特的历史，因为它们引发了英国有史以来针对制药商的最大规模集体诉讼，涉及14,000名患者和1,800家律师事务所，指控制造商知道依赖性潜力但故意向医生隐瞒这一信息。

与此同时，117名全科医生和50个卫生当局被患者起诉，要求赔偿依赖和戒断造成的有害影响。这导致一些医生要求患者签署知情同意书，并建议在开始苯二氮卓类药物治疗之前充分警告所有患者依赖和戒断的风险。[\[9\]](#cite_note-9) 针对制药商的法庭案件从未达成判决；法律援助被撤回，并且有人指控作为专家证人的顾问精神科医生存在利益冲突。这场诉讼导致英国法律发生了变化，使得集体诉讼变得更加困难。[\[10\]](#cite_note-urlHouse_of_Commons-10)

尽管已经推出了具有抗焦虑特性的抗抑郁药，而且人们越来越意识到苯二氮卓类药物的不良反应，但用于短期缓解焦虑的处方并没有显着下降。[\[11\]](#cite_note-pmid18671662-11) 对于失眠的治疗，苯二氮卓类药物现在不如非苯二氮卓类药物受欢迎，后者包括 [唑吡坦](/FreeODwiki/药物/唑吡坦.html)、扎来普隆和 [右佐匹克隆](/FreeODwiki/药物/佐匹克隆.html)。[\[12\]](#cite_note-12) 非苯二氮卓类药物在分子上是不同的，但尽管如此，它们作用于相同的苯二氮卓受体并产生类似的镇静效应喵。[\[13\]](#cite_note-wake-13)

## 化学

苯二氮卓类物质是杂环化合物，由一个苯环与一个七元含氮二氮卓环稠合而成。苯二氮卓类药物在 R5 位置含有一个额外的取代苯环，产生 5-苯基-1,4-苯二氮卓，并在结构上连接不同的侧基，从而产生许多具有不同强度、持续时间和功效的药物。

苯二氮卓类药物通常在其环的 R7 位置含有芳香亲电取代，如芳香卤化或硝化。苯二氮卓类物质可细分为三唑苯二氮卓类和酮取代苯二氮卓类。三唑苯二氮卓类含有一个与苯二氮卓结构键合的三唑环，并以后缀“-zolam”区分。酮取代环在其苯二氮卓结构的 R2 位置含有一个酮氧键，并以后缀“-azepam”区分。

## 药理学

苯二氮卓类物质通过结合苯二氮卓受体位点并通过作用于其 [受体](/FreeODwiki/文档/受体激动剂.html)，放大神经递质 [γ-氨基丁酸 (GABA)](/FreeODwiki/文档/GABA.html) 的效率和效应，从而产生各种效应。[\[2\]](#cite_note-Haefely1984-2) 由于该位点是大脑中最高产的抑制性受体组，其调节导致了苯二氮卓类药物对神经系统的 [镇静](/FreeODwiki/药效/镇静.html)（或 [平静效应](/FreeODwiki/药效/焦虑抑制.html)）。

苯二氮卓类药物的 [抗惊厥](/FreeODwiki/文档/抗惊厥.html) 特性可能部分或全部归因于与电压依赖性钠通道的结合，而不是苯二氮卓受体。[\[14\]](#cite_note-14)

苯二氮卓类药物起弱腺苷再摄取抑制剂的作用。有人提出，它们的一些 [抗惊厥](/FreeODwiki/文档/抗惊厥.html)、[抗焦虑](/FreeODwiki/文档/抗焦虑.html) 和 [肌肉松弛](/FreeODwiki/药效/肌肉松弛.html) 效应可能部分由这种作用介导喵。[\[15\]](#cite_note-15)

## 医疗用途

当与苯二氮卓类药物结合使用时，[致幻剂](/FreeODwiki/文档/药物分类/致幻剂.html) 物质（特别是 [迷幻剂](/FreeODwiki/文档/药物分类/迷幻剂.html)）引起的视觉幻觉可能会显着减少，任何潜在的 [焦虑](/FreeODwiki/药效/焦虑.html) 也会随之减少。\[*[来源请求](/FreeODwiki/文档/来源请求.html)*\]

连同 [抗精神病药](/FreeODwiki/文档/抗精神病药.html) 如 [喹硫平](/FreeODwiki/药物/喹硫平.html) (*Seroquel*)，苯二氮卓类药物通常在医院环境中 [给药](/FreeODwiki/文档/给药途径.html)，以治疗出现致幻剂过量或 [精神病发作](/FreeODwiki/药效/精神病发作.html) 症状的患者哦。[\[24\]](#cite_note-24)

相对于剂量，苯二氮卓类药物的毒性较低，并且被认为单独使用实际上是非致命的。[\[27\]](#cite_note-27) 然而，当与 [酒精](/FreeODwiki/药物/酒精.html) 或 [阿片类药物](/FreeODwiki/文档/药物分类/阿片类药物.html) 等 [抑制剂](/FreeODwiki/文档/药物分类/抑制剂.html) 混合使用时，它们潜在的 [致命性](/FreeODwiki/药效/呼吸抑制.html) 会显着增加。

强烈建议在使用此类物质时采取 [伤害减少措施](/FreeODwiki/文档/负责任的用药索引页.html) 喵。

### 耐受性和成瘾潜力

众所周知，苯二氮卓类药物在身体和心理上都极易成瘾。\[*[来源请求](/FreeODwiki/文档/来源请求.html)*\]

连续使用几天后，会对镇静-催眠效应产生耐受性。[\[28\]](#cite_note-28) 停止使用后，耐受性会在 7-14 天内恢复到基线。在稳定给药几周或更长时间后突然停止使用可能会出现戒断症状或反跳症状，并且可能需要逐渐减少剂量。[\[29\]](#cite_note-29)[\[30\]](#cite_note-30)

### 药物过量

苯二氮卓类药物过量可能发生在极高剂量下，或者更常见的是，当它与其他 [抑制剂](/FreeODwiki/文档/药物分类/抑制剂.html) 一起服用时。这种风险在与其他 [GABA能](/FreeODwiki/文档/GABA.html) 抑制剂（如 [巴比妥类物质](/FreeODwiki/文档/药物分类/巴比妥类物质.html) 和 [酒精](/FreeODwiki/药物/酒精.html)）一起使用时尤其存在，因为它们以类似的方式起作用，但结合到 GABAA 受体上的不同位点，导致显着的交叉增强作用。\[*[来源请求](/FreeODwiki/文档/来源请求.html)*\]

苯二氮卓类药物过量是一种医疗紧急情况，如果不及时治疗，可能导致昏迷、永久性脑损伤或死亡。症状可能包括严重的 [语无伦次](/FreeODwiki/药效/语无伦次.html)、[混乱](/FreeODwiki/药效/混乱.html)、[妄想](/FreeODwiki/药效/妄想.html)、[呼吸抑制](/FreeODwiki/药效/呼吸抑制.html) 和无反应。使用者可能看起来像是在梦游。由于判断力受损，使用者也更容易消耗更多的相同或另一种物质，这在其他物质过量期间通常看不到。

苯二氮卓类药物过量可以在医院环境中得到有效治疗，通常结果良好。护理主要是支持性的，尽管有时会使用 [氟马西尼](/FreeODwiki/药物/氟马西尼.html)（一种 GABAA 拮抗剂）[\[31\]](#cite_note-31) 或如果涉及其他物质，则使用 [肾上腺素](/FreeODwiki/文档/肾上腺素.html) 注射等额外程序进行治疗。\[*[来源请求](/FreeODwiki/文档/来源请求.html)*\]

### 停药和戒断

苯二氮卓类药物的停药是出了名的困难；对于经常使用的人来说，如果不经过几周的时间逐渐减少剂量就停止使用，可能会危及生命。这会增加 [高血压](/FreeODwiki/药效/血压升高.html)、[癫痫发作](/FreeODwiki/文档/癫痫发作.html) 和死亡的风险。[\[32\]](#cite_note-32) 在戒断期间应避免使用降低癫痫发作阈值的物质，如 [曲马多](/FreeODwiki/药物/曲马多.html)。\[*[来源请求](/FreeODwiki/文档/来源请求.html)*\] 突然停药也会导致反跳性刺激，表现为 [焦虑](/FreeODwiki/药效/焦虑.html)、[失眠](/FreeODwiki/药效/清醒度.html) 和烦躁不安。\[*[来源请求](/FreeODwiki/文档/来源请求.html)*\]

如果希望在一段时间的常规使用后停药，最安全的方法是每天减少极少量的剂量，持续几周，直到接近戒断。如果使用的是半衰期短的苯二氮卓类药物，如 [阿普唑仑](/FreeODwiki/药物/阿普唑仑.html) 或 [依替唑仑](/FreeODwiki/药物/依替唑仑.html)，可以用作用时间较长的品种，如 [地西泮](/FreeODwiki/药物/地西泮.html) 或 [氯硝西泮](/FreeODwiki/药物/氯硝西泮.html) 来替代。症状可能仍然存在，但其严重程度将显着降低。

有关以受控方式逐渐戒断苯二氮卓类药物的更多信息，请参阅 [本指南](http://www.benzo.org.uk/manual/bzcha02.htm)。少量的 [酒精](/FreeODwiki/药物/酒精.html) 也有助于减轻症状，但除此之外不能用作有效的减量剂。

戒断症状的持续时间和严重程度取决于许多因素，包括所用物质的半衰期、耐受性和滥用的持续时间。对于持续时间较短的苯二氮卓类药物，主要症状通常在停药后几天内开始，并持续约一周。半衰期较长的苯二氮卓类药物将表现出发作缓慢且持续时间延长的戒断症状。\[*[来源请求](/FreeODwiki/文档/来源请求.html)*\]

### 危险的相互作用

***警告：*** *许多单独使用相当安全的精神活性物质，当与某些其他物质结合使用时，可能会突然变得危险甚至危及生命。以下列表提供了一些已知的危险相互作用（尽管不保证包含所有相互作用）。*

*务必进行独立研究（例如 [Google](https://www.google.com)、[DuckDuckGo](https://www.duckduckgo.com)、[PubMed](https://pubmed.ncbi.nlm.nih.gov/)）以确保两种或多种物质的组合可以安全食用。列出的一些相互作用来自 [TripSit](https://combo.tripsit.me)。*

*   **抑制剂** (*[1,4-丁二醇](/FreeODwiki/药物/1,4-丁二醇.html)、[2M2B](/FreeODwiki/药物/2M2B.html)、[酒精](/FreeODwiki/药物/酒精.html)、苯二氮卓类物质、[巴比妥类物质](/FreeODwiki/文档/药物分类/巴比妥类物质.html)、[GHB](/FreeODwiki/药物/GHB.html)/[GBL](/FreeODwiki/药物/GBL.html)、[甲喹酮](/FreeODwiki/药物/甲喹酮.html)、[阿片类药物](/FreeODwiki/文档/药物分类/阿片类药物.html)*) - 这种组合会增强彼此引起的 [肌肉松弛](/FreeODwiki/药效/肌肉松弛.html)、[失忆](/FreeODwiki/药效/失忆.html)、[镇静](/FreeODwiki/药效/镇静.html) 和 [呼吸抑制](/FreeODwiki/药效/呼吸抑制.html)。在较高剂量下，它可能导致突然、意外的意识丧失以及危险程度的呼吸抑制。在失去知觉时被呕吐物窒息的风险也会增加。如果在失去知觉之前出现 [恶心](/FreeODwiki/药效/恶心.html) 或呕吐，使用者应尝试以 [恢复体位](/FreeODwiki/文档/恢复体位.html) 入睡或让朋友将其移入该体位。
*   **解离剂** - 这种组合可能会不可预测地增强彼此可能引起的 [失忆](/FreeODwiki/药效/失忆.html)、[镇静](/FreeODwiki/药效/镇静.html)、[运动控制丧失](/FreeODwiki/药效/运动控制丧失.html) 和 [妄想](/FreeODwiki/药效/妄想.html)。它还可能导致突然的意识丧失并伴有危险程度的 [呼吸抑制](/FreeODwiki/药效/呼吸抑制.html)。如果在失去意识之前出现 [恶心](/FreeODwiki/药效/恶心.html) 或呕吐，使用者应尝试以 [恢复体位](/FreeODwiki/文档/恢复体位.html) 入睡或让朋友将其移入该体位。
*   **兴奋剂** - 兴奋剂会掩盖抑制剂的 [镇静](/FreeODwiki/药效/镇静.html) 效应，这是大多数人用来衡量其中毒程度的主要因素。一旦兴奋剂效应消退，抑制剂的效应将显着增加，导致加剧的 [去抑制](/FreeODwiki/药效/去抑制.html)、[运动控制丧失](/FreeODwiki/药效/运动控制丧失.html) 和危险的 [断片状态](/FreeODwiki/药效/失忆.html)。如果液体的摄入量没有受到密切监控，这种组合还可能导致严重的脱水。如果选择混合使用这些物质，应严格限制自己每小时只服用一定量，直到达到最大阈值。
