# 益智药

| [![](https://psychonautwiki.org/w/thumb.php?f=Nootropic_Survey_2016_Factor_Analysis.png&width=300)](/FreeODwiki/文件/Nootropic_Survey_2016_Factor_Analysis.png) |
| :---: |
| [放大](/FreeODwiki/文件/Nootropic_Survey_2016_Factor_Analysis.png) |
| 来自 [2016年调查](http://slatestarcodex.com/2016/03/01/2016-nootropics-survey-results/) 的益智药评分的 [因子分析](https://en.wikipedia.org/wiki/Factor_analysis) |

**益智药**（Nootropics，也叫 **聪明药**、**神经增强剂** 和 **认知增强剂**）是一类据称可以改善健康个体认知功能（如记忆力、动力、注意力和专注力）的物质哦。[\[1\]](#cite_note-1)[\[2\]](#cite_note-2) 人们认为益智药通过改变大脑神经化学物质（即神经递质、酶和激素）的供应、改善大脑的氧气供应或刺激神经生长来起作用呢。\[*[来源请求](/FreeODwiki/文档/来源请求.html)*\]

*Nootropic* 这个词是1972年由罗马尼亚心理学家和化学家 Corneliu E. Giurgea 创造的，[\[3\]](#cite_note-isbn0-06-088473-8-3)[\[4\]](#cite_note-pmid4541214-4) 源自 [希腊语](/FreeODwiki/文档/古希腊语.html) 单词 νοῦς (*nous*)，即“头脑”，和 τρέπειν (*trepein*)，意思是弯曲或转动喵。[\[5\]](#cite_note-5) Giurgea 博士提出了以下标准来确定一种物质是否符合“益智药”的描述：

*   增强学习和记忆功能
*   当条件被设定为破坏它们时（例如使用健忘剂），改善习得的行为
*   必须具有神经保护作用
*   毒性必须极低，几乎没有副作用

然而，这个术语后来获得了一种通俗的含义，并不要求满足所有这些标准哦。

有几类益智药，以及不符合任何特定分类的化合物呢。最著名和使用最广泛的一类是拉西坦类物质，其原型是 [吡拉西坦](/FreeODwiki/药物/吡拉西坦.html)，它是1964年第一个被认可的益智药。拉西坦类物质都共享一个 2-吡咯烷酮环。其他拉西坦类益智药包括阿尼西坦、奥拉西坦、苯基吡拉西坦和考拉西坦哦。

第二类益智化合物是合成肽，最常见的例子是 Noopept。虽然不是拉西坦，但由于作用模式相似，Noopept 通常与拉西坦类物质归为一类喵。

由于益智药科学尚处于起步阶段，还有一些新合成的化合物，如 IDRA-21、PRL-8-53、unifiram、sunifiram 等，它们并不完全符合某种结构类别。此外，对于其中一些化合物，副作用、剂量、危险的相互作用等几乎都是未知的呢，要小心哦。
