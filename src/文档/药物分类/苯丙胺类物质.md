# 苯丙胺类物质

来自 PsychonautWiki

[![](/FreeODwiki/文件/Substituted_amphetamine.png)](/FreeODwiki/文件/Substituted_amphetamine.png)

[ ](/FreeODwiki/文件/Substituted_amphetamine.png "Enlarge")

一个*苯丙胺*分子的取代结构。

**苯丙胺类物质**是基于[苯丙胺](/FreeODwiki/药物/苯丙胺.html)化学结构的有机化合物[化学分类](/FreeODwiki/文档/化学分类.html)。该类成员涵盖多种药理学亚类，最显著的是[兴奋剂](/FreeODwiki/文档/药物分类/兴奋剂.html)和[共情剂](/FreeODwiki/文档/药物分类/home.html)，但也包括一些[迷幻剂](/FreeODwiki/文档/药物分类/迷幻剂.html)。苯丙胺类物质的例子包括[苯丙胺](/FreeODwiki/药物/苯丙胺.html)（本身）、[甲基苯丙胺](/FreeODwiki/药物/甲基苯丙胺.html)、[卡西酮](/FreeODwiki/药物/卡西酮.html)、[吡咯戊酮](/FreeODwiki/药物/吡咯戊酮.html)、[MDMA](/FreeODwiki/药物/MDMA.html)和[DOx](/FreeODwiki/文档/药物分类/DOx.html)。

## 化学

[![](/FreeODwiki/文件/Substituted_phenethylamine.png)](/FreeODwiki/文件/Substituted_phenethylamine.png)

[ ](/FreeODwiki/文件/Substituted_phenethylamine.png "Enlarge")

一个*苯乙胺*分子的取代结构。

苯丙胺类物质是一类结构基于[苯丙胺](/FreeODwiki/药物/苯丙胺.html)的化合物，在不同位置有不同的取代基。苯丙胺分子具有[苯乙胺](/FreeODwiki/文档/药物分类/苯乙胺类物质.html)分子的结构，并在α-碳上有一个额外的甲基。因此，苯丙胺除了是苯丙胺类物质的未取代成员外，也被视为一种[取代苯乙胺](/FreeODwiki/文档/药物分类/苯乙胺类物质.html)。

## 药理学

苯丙胺类物质的精神活性通常通过[多巴胺](/FreeODwiki/文档/多巴胺.html)、[血清素](/FreeODwiki/文档/血清素.html)和[肾上腺素](/FreeODwiki/文档/肾上腺素.html)通路产生效应。迷幻效应可归因于对[5-HT2A受体](/FreeODwiki/文档/血清素.html#The_5-HT_system)的作用。\[*[来源请求](/FreeODwiki/文档/来源请求.html)*\] 兴奋和共情效应归因于它们作为[多巴胺](/FreeODwiki/文档/多巴胺.html)、[血清素](/FreeODwiki/文档/血清素.html)、[肾上腺素](/FreeODwiki/文档/肾上腺素.html)和[去甲肾上腺素](/FreeODwiki/文档/去甲肾上腺素.html)的[释放剂](/FreeODwiki/文档/神经递质释放剂.html)或作为上述神经递质受体的[激动剂](/FreeODwiki/文档/受体激动剂)的作用。\[*[来源请求](/FreeODwiki/文档/来源请求.html)*\] 这组受体的激动导致突触后神经元放电率增加，从而引发使用者的[认知](/FreeODwiki/药效/认知效应.html)和[身体刺激](/FreeODwiki/药效/刺激.html)。\[*[来源请求](/FreeODwiki/文档/来源请求.html)*\]
