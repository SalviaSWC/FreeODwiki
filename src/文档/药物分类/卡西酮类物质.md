# 取代卡西酮类物质

[![](/FreeODwiki/文件/Substituted_cathinone.png)](/FreeODwiki/文件/Substituted_cathinone.png)

[ ](/FreeODwiki/文件/Substituted_cathinone.png "放大")

卡西酮分子的取代结构。

**取代卡西酮类物质**（也称为**卡西酮类物质**或**合成卡西酮**）是一类基于[卡西酮](/FreeODwiki/药物/卡西酮.html)结构的具有精神活性的化合物。该类别由卡西酮的所有衍生物组成，这些衍生物是通过用取代基替换卡西酮核心结构中的一个或多个氢原子而形成的。例子包括 [MDPV](/FreeODwiki/药物/MDPV.html)、[4-MMC](/FreeODwiki/药物/4-MMC.html)（喵喵）、[Methylone](/FreeODwiki/药物/Methylone.html) 和 [α-PVP](/FreeODwiki/药物/α-PVP.html)。

卡西酮是恰特草（*Catha edullis*）的活性成分，这是一种为了兴奋效果而被咀嚼叶子的植物。卡西酮在结构上与[苯丙胺](/FreeODwiki/药物/苯丙胺.html)密切相关，唯一的区别在于β碳上的酮基。

## 化学

取代卡西酮类物质是基于[卡西酮](/FreeODwiki/药物/卡西酮.html)分子的化学类别。卡西酮类物质是[苯丙胺类物质](/FreeODwiki/文档/药物分类/苯丙胺类物质.html)的一个子类。它们共享核心的苯丙胺结构，但与[取代苯丙胺类物质](/FreeODwiki/文档/药物分类/苯丙胺类物质.html)的区别在于苯丙胺结构的β碳上有酮基取代。因此，一些卡西酮类物质，如 [βk-MDMA](/FreeODwiki/药物/Methylone.html)，与流行的取代苯丙胺类物质仅差一个β-酮基取代，因此会采用取代苯丙胺的名称，并加上“beta-keto”/“βk-”前缀。

值得注意的是，虽然有一类精神活性化合物因缺乏[卡西酮](/FreeODwiki/药物/卡西酮.html)中存在的α碳甲基而在技术上不属于取代卡西酮类物质，但它们与各种[取代苯乙胺类物质](/FreeODwiki/文档/药物分类/苯乙胺类物质.html)的区别仅在于β酮基取代；最著名的是 [βk-2C-B](/FreeODwiki/药物/bk-2C-B.html)。

## 药理学

这一类[共情剂](/FreeODwiki/文档/药物分类/共情剂.html)和[兴奋剂](/FreeODwiki/文档/药物分类/兴奋剂.html)药物通过作为[多巴胺](/FreeODwiki/文档/多巴胺.html)[神经递质再摄取抑制剂](/FreeODwiki/文档/神经递质再摄取抑制剂.html)发挥作用。对这组受体的[激动](/FreeODwiki/文档/受体激动剂.html)导致突触后神经元的放电率增加，从而引发使用者的[思维加速](/FreeODwiki/药效/思维加速.html)和[身体刺激](/FreeODwiki/药效/刺激.html)。
