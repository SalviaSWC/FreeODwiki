# 负责任的用药

[◀返回](./home.html)

对于危险物质，请参阅 [不建议使用的药物](/FreeODwiki/文档/不建议使用的药物.html) 哦。

**负责任的用药** 是一套多样化的理念和实践，基于这样一个前提：娱乐性物质可以在知情的成年人手中以健康和负责任的方式使用呢。它与 **[伤害减少措施](/FreeODwiki/文档/减量戒断法.html)** 的概念密切相关，后者旨在通过教育和培训，最大限度地减少精神活性物质的风险和危害，同时最大限度地发挥其益处。

根据伤害减少的方法，娱乐性物质的使用首先应被理解为一种高风险活动，本质上涉及成瘾、严重身体伤害和死亡的风险。因此，个人可以采取的最务实的策略——除了完全戒断（这可能并不总是现实或令人向往的）——就是仔细研究每种物质，并采取切实步骤来减少与其使用相关的风险和危害，直到益处超过成本为止喵。

娱乐性药物的使用可以被看作与其他既有风险又能丰富生活的活动类似，特别是像帆船、滑雪、跳伞、冲浪和登山这样的极限运动。说得普通一点，也可以把它比作开车、骑摩托车或坐飞机呢。

虽然这些活动都带有巨大的风险（包括死亡），但大家普遍认为，通过适当的教育和培训，这些风险可以被降至可接受的水平。根据文化背景的不同，参与这些活动的能力可能被视为个人自由、自决权和尊严不可剥夺的表现哦。

负责任用药背后的哲学可以被描述为相对激进的，因为它将 *绝对责任* 放在个人身上，要求进行适当的研究并采取必要的安全预防措施。这伴随着一种理解，即 *没有真正“安全”的使用，只有更安全的使用*，个人最终要对自己的选择结果负责，无论是健康相关的、财务的还是其他方面的呢。

负责任用药的倡导者指出，许多著名的艺术家和知识分子都曾使用过药物（实验性地或其他方式），而对他们的生活几乎没有产生不利影响。批评者则认为药物是逃避现实的、危险的、不可预测的，并且往往会上瘾；因此，负责任的用药是一种幻觉。

**一般伤害减少建议的例子包括：**

* 学习正在使用的物质的[药效](/FreeODwiki/药效/home.html)和合法性
* 测量准确的剂量并采取其他预防措施以减少[药物过量](/FreeODwiki/文档/药物过量.html)的风险
* 花时间对所有要使用的物质进行化学测试，以确定纯度和强度
* 不要游泳哦。
* 不要在神志不清时[驾驶](https://en.wikipedia.org/wiki/Drug%E2%80%93impaired_driving)、操作重型机械，或以其他方式直接或间接对另一个人的安全或护理负责
* 避免娱乐性的醉酒拨号、发邮件等，这可能会导致社交上的负面后果呢。
* 在服用不熟悉的物质时，要有一位[旅程保姆](/FreeODwiki/文档/旅程保姆.html)
* 不要试图欺骗或劝说任何人使用他们不愿意使用的物质
* 不要让物质使用掩盖生活的其他方面或责任
* 在道德上要意识到物质的来源
* [避免某些精神活性物质](/FreeODwiki/文档/不建议使用的药物.html)

本页面致力于提供有关在试验或定期使用[精神活性物质](/FreeODwiki/文档/药物分类/药物全索引.html)时应考虑的各种因素的信息。第一部分涵盖所有物质类别的一般伤害减少实践，而后面部分是专门针对[致幻剂](/FreeODwiki/文档/药物分类/致幻剂.html)的哦。

* #### [选择剂量](/FreeODwiki/文档/给药剂量.html)
    | *[主条目：选择剂量](/FreeODwiki/文档/给药剂量.html)* |
    | --- |
    | 在选择剂量时，用户应避免使用自己不舒服或不熟悉的剂量。缺乏经验的用户应始终从剂量范围的低端开始，目标是以小幅度逐步增加，直到熟悉该物质的行为。<br><br>剂量应始终以微小的增量向上调整（例如，之前剂量的1/4到1/2）。应严格避免大幅、不稳定的增加（例如，之前剂量的2倍或3倍）。重要的是要注意，许多物质没有线性的剂量-反应曲线，这意味着 *剂量加倍会导致超过两倍的反应增强*（并迅速导致压倒性的、不愉快的和潜在危险的体验）。遵循这一规则将大大降低负面体验的风险哦。<br><br>至关重要的是要记住，每个人对每种物质的反应都不同，取决于个人耐受性、体重、新陈代谢和个人敏感度等因素。另一个需要考虑的因素是物质纯度，不同供应商和批次的产品纯度可能不同。<br><br>有关任何物质的适当剂量信息，应结合使用 [PsychonautWiki](/FreeODwiki/文档/药物分类/药物全索引.html)、*[Erowid](http://erowid.org)*、*[Tripsit](https://tripsit.me)*、[Wikipedia](http://en.Wikipedia.org)、[DuckDuckGo](http://duckduckgo.com) 和 [Google](http://Google.com) 等来源进行研究呢。 |

* #### [过敏测试](/FreeODwiki/文档/给药剂量.html)
    | *[主条目：过敏测试](/FreeODwiki/文档/给药剂量.html)* |
    | --- |
    | 有些人，特别是那些有健康问题的人，可能会对物质表现出不良反应，形式包括过敏反应、不舒服的身体或认知效应，或超敏反应。[\[1\]](#cite_note-erowid1-1) 这种风险可以通过进行“过敏测试”来降低，这仅仅是服用微量的物质（例如常规剂量的1/10到1/4），并等待几个小时以验证自己没有表现出异常或特异性反应喵。 |

* #### [目测剂量](/FreeODwiki/文档/药物剂量量取.html)
    | *[主条目：目测剂量](/FreeODwiki/文档/药物剂量量取.html)* |
    | --- |
    | [![Microscoops](/FreeODwiki/文件/Microscoops.jpg)](/FreeODwiki/文件/Microscoops.jpg)<br>这张图片展示了为什么使用微型勺进行目测不是一种准确可靠的测量形式。<br><br>**不要目测剂量哦。** 目测是一种非常不准确且不可靠的测量物质的方法，它涉及看着物质并粗略猜测其数量。由于10毫克和30毫克在视觉上差别不大，很容易导致服用过多的物质，这可能会产生致命的后果呢。[\[2\]](#cite_note-erowid4-2)<br><br>有许多形式的目测方法，包括使用微型勺和方格纸法，这些都会导致剂量不准确。<br><br>用户不应相信供应商关于其产品重量的说法，因为供应商意外或故意给出错误数量的产品并非闻所未闻，这导致了一些用户的药物过量。[\[3\]](#cite_note-erowid2-3)<br><br>目测剂量极低的物质（10毫克以下）尤其危险。 |

* #### [毫克秤](/FreeODwiki/文档/药物剂量量取.html)
    | *[主条目：毫克秤](/FreeODwiki/文档/药物剂量量取.html)* |
    | --- |
    | [![GEMINI-20](/FreeODwiki/文件/71gBXwhrvDL._SL1500_.jpg)](/FreeODwiki/文件/71gBXwhrvDL._SL1500_.jpg)<br>上图所示的GEMINI-20便携式毫克秤是[Tripsit](https://wiki.tripsit.me/wiki/Scales)推荐的一款流行秤。这些可以在在线零售商处购买。<br><br>当购买或使用粉末状化学品时，强烈建议投资购买一台可靠且准确的数字毫克秤，以确保摄入安全和已知的剂量。<br><br>为了获得最准确的测量结果，秤只能在完全平坦的表面上使用，远离振动、风和气流。电池电量不足也会影响秤的准确性，所以应该定期更换电池。秤通常带有校准砝码，可用于检测准确性是否下降。[\[4\]](#cite_note-gemini-4)<br><br>各种网站上有大量不同的秤可供选择。要选择高质量的秤，用户应在购买前阅读评论并做出自己的决定。用户理想情况下应选择具有0.001克（1毫克）可读性的秤。当使用具有0.005克可读性（5毫克）的秤时，重量会在任一方向上偏差5毫克，这意味着如果测量20毫克的物质，结果将在15 - 25毫克之间。<br><br>**1000美元以下的毫克秤无法准确称量低于至少50毫克的剂量，并且在10 - 15毫克以下非常不准确。对于强效物质，建议使用[液体容量给药法](/FreeODwiki/文档/液体容量给药法.html)技术。**<br><br>大多数毫克秤在较高范围（5 - 15克）比低范围更准确。因此，最好在校准砝码在秤上时称量化学品。 |

### [给药途径](/FreeODwiki/文档/给药途径.html)

* *[主条目：给药途径](/FreeODwiki/文档/给药途径.html)*
    + 口腔：对于在胃中具有生物利用度的物质，避免舌下、颊面和唇下（嘴唇下方）给药是避免混淆的好方法。例如，[25I-NBOMe](/FreeODwiki/药物/25I-NBOMe.html) 在胃中会分解，但 25I-NBOMe 曾被当作 LSD 出售，这已导致数人死亡[\[7\]](#cite_note-Erowid25I-NBOMe-7)[\[8\]](#cite_note-NY_Daily_news-8)[\[9\]](#cite_note-Ireland_injuries-9)[\[10\]](#cite_note-ACMD_Report2-10)，并且可能常被卖家和用户误认为是 [LSD](/FreeODwiki/药物/LSD.html)。[\[11\]](#cite_note-ACMD_Report-11)
    + 呼吸道（吸入、汽化、抽吸）：通过呼吸道给药更难，因为由于技术不当，呼出的空气中通常含有大量残留物。可以通过以下方式提高生物利用度：1) 尽可能长时间地屏住呼吸，通常为一分钟 2) 如果注意到呼出的雾气（汽化）或烟雾，那么可以将所需剂量分成较小的剂量重复使用。
    + 注射：注射是最危险的给药途径，由于过量、成瘾和其他健康并发症的风险增加，强烈不建议使用。[共享注射用材料](/FreeODwiki/文档/共享注射用材料.html) 可能因血液传播疾病和感染而致命。如果一定要注射，请参阅 [较安全的注射指南](/FreeODwiki/文档/较安全的注射指南.html) 以获取有关减少与注射药物使用相关的一些风险的信息。
    + 鼻吸：频繁鼻吸物质会损害粘膜。根据一项研究，共用鼻吸设备（吸管、纸币、子弹壶等）与丙型肝炎的传播有关，研究人员警告说其他血液传播疾病也可能被传播哦。[\[12\]](#cite_note-12)

| * 虽然许多物质单独使用相当安全，但有许多组合可能极其危险甚至致命。当组合多种精神活性物质时，应始终考虑到这一点，并且如果希望避免伤害、住院或死亡，必须进行充分的独立研究以确定组合的安全性。<br><br>许多处方药与常用的娱乐性物质有负面相互作用。物质相关死亡的最常见原因是[抑制剂](/FreeODwiki/文档/药物分类/抑制剂.html)（如[阿片类药物](/FreeODwiki/文档/药物分类/阿片类药物.html)、[苯二氮卓类物质](/FreeODwiki/文档/药物分类/苯二氮卓类物质.html)或[酒精](/FreeODwiki/药物/酒精.html)）与其他抑制剂的联用。<br><br>[![Dangerous_combination_chart](/FreeODwiki/文件/TripSit%27s_Dangerous_combination_chart.png)](/FreeODwiki/文件/TripSit%27s_Dangerous_combination_chart.png)<br>上图是一个有用的图表，可以用作确定常见组合安全性的快速参考工具。欲了解更多信息，请访问其原创者 [TripSit.me](http://wiki.tripsit.me/wiki/Drug_combinations)。 |

### 成瘾

| ![Line-chart](/FreeODwiki/文件/Line-chart.png) |
| --- |
| * 由于成瘾是使用物质的一个严重且众所周知的风险，用户应仔细监控任何物质使用的频率和强度，以确保其不会滑向滥用和成瘾。特别是，许多[兴奋剂](/FreeODwiki/文档/药物分类/兴奋剂.html)、[阿片类药物](/FreeODwiki/文档/药物分类/阿片类药物.html)和[抑制剂](/FreeODwiki/文档/药物分类/抑制剂.html)已知具有高度成瘾性。应仔细考虑和研究任何这些物质（以及任何一般物质）的定期使用。有些智能手机应用程序可以帮助跟踪物质使用情况。<br><br>虽然经典[迷幻剂](/FreeODwiki/文档/药物分类/迷幻剂.html)被认为在身体上无毒且不成瘾，但这并不适用于所有[致幻剂](/FreeODwiki/文档/药物分类/致幻剂.html)。[解离剂](/FreeODwiki/文档/药物分类/解离剂.html)、[谵妄剂](/FreeODwiki/文档/药物分类/谵妄剂.html)、[共情剂](/FreeODwiki/文档/药物分类/共情剂.html)和某些迷幻剂（例如 [DOx](/FreeODwiki/文档/药物分类/2,5-二甲氧基苯丙胺类物质.html)、[25x-NBOMe](/FreeODwiki/文档/药物分类/N-苄基苯乙胺类物质.html)、[2C-T-x](/FreeODwiki/文档/药物分类/4-硫基-2,5-二甲氧基苯乙胺类物质.html)）可能带来潜在的不良健康影响，需要在用前单独研究。<br><br>如果过度使用，任何致幻剂都可能导致人的心理健康问题。请记住，一旦负面后果开始超过积极方面，并且干扰了用户的工作和人际关系，但他们仍然继续使用，物质使用就变成了成瘾喵。 |

## 致幻剂

以下信息专门针对[致幻剂](/FreeODwiki/文档/药物分类/致幻剂.html)的使用和实验，如[迷幻剂](/FreeODwiki/文档/药物分类/迷幻剂.html)、[解离剂](/FreeODwiki/文档/药物分类/解离剂.html)和[谵妄剂](/FreeODwiki/文档/药物分类/谵妄剂.html)。

### 情景

| ![Home](/FreeODwiki/文件/Home.png) |
| --- |
| * 选择一个合适的地方来体验致幻剂的效应极其重要，并在决定体验结果方面起着主要作用。对于缺乏经验的用户来说，理想的地方是一个熟悉、安全、室内环境，他们对其有完全的控制权，并且没有可能消极影响精神状态的因素。为了为致幻剂准备适当的情景，建议采取以下步骤：<br>* + **确保** 在体验期间完全没有责任，理想情况下第二天也是如此。这是因为即使是最简单的任务，在致幻剂的影响下也可能变得极其困难且可能充满压力。用户应准备好完全放松，不执行家务或日常事务。这包括驾驶和操作重型机械。<br>* + **避免** 不直接参与体验的人。这包括可能睡在同一个房子里的亲戚，以及除了非常值得信赖、理解并了解致幻剂效应的朋友以外的任何人。仅仅是不知道的人在附近就可能引发[焦虑](/FreeODwiki/药效/焦虑.html)和[偏执](/FreeODwiki/药效/偏执.html)，并阻碍完全沉浸在体验中。<br>* + **避免** 不熟悉、嘈杂、杂乱和/或公共的环境。用户应选择一个他们有相当大控制权的环境。这可以简单到能够调节空调设置或自由进出洗手间。在整个体验期间，人们应该能够随意坐下、躺下和走动。选定的环境理想情况下应配备隐私、轻松的音乐、舒适的座位以及现成的食物和水。这种环境的例子包括家里或朋友家里的一个安全、舒适的房间。<br>* + **避免** 任何可能产生“坏气氛”的来源。用户不应让自己暴露在不愉快或令人不安的刺激中，如恐怖电影或黑暗的音乐。如果坏气氛包围了体验，可以通过快速改变用户所在的直接环境来逃脱。例如，如果正关着灯坐着，站起来开灯，换音乐，或者搬到房子里的另一个房间。<br>* + 一旦用户对他们选择的物质变得非常熟悉，是否在控制较少的环境中（如户外自然界、社交聚会、派对、狂欢派对等）进行旅程就是个人的选择了。然而，应该注意，在这些环境中进行旅程会带来更多的身体和法律风险哦。 |

### 心境 (精神状态)

| ![User](/FreeODwiki/文件/User.png) |
| --- |
| * 用户的精神状态在决定旅程结果方面起着主要作用。致幻剂会放大当前的心理状态、情绪和前景：积极的心态可能会变得更积极，消极的心态会变得更消极。因此，一般应避免在生活极度紧张或消极的时期使用致幻剂。用户应充分意识到致幻剂，特别是迷幻剂，能够迫使人面对他们当时可能在心理上还没准备好处理的内部问题。<br><br>那些有既往精神状况的人（特别是患有精神分裂症等精神病性疾病的人）应避免使用致幻剂，因为它们可以强烈放大潜在的精神和情绪状态，并促进[妄想](/FreeODwiki/药效/妄想.html)和[幻觉](/FreeODwiki/药效/外部幻觉.html)。那些希望在有此类状况下服用致幻剂的人应寻求合格医疗从业者的建议。<br><br>旅程中的一个常见建议是“放手”，让物质的效应掌管一切。应该处于隐喻的乘客座位上，放弃试图控制或压制体验的任何部分。至关重要的是，用户只需放松并接受发生的一切，因为任何抵抗只会放大试图避免的东西。<br><br>此外，用户必须理解，旅程的体验往往是不可捉摸的，这意味着应该接受无法理解或表达体验期间发生的全部范围。用户应接受这样一个事实，即他们的思维过程虽然在某些方面可能更清晰，但将不可避免地受损，连同精细运动控制、会话技能和情境意识。用户应确保经常提醒自己，这些效应是正常的，最重要的是，是暂时的呢。 |

### 身体状态

| ![Child](/FreeODwiki/文件/Child.png) |
| --- |
| * 用户的当前身体状况与进入旅程时的情绪和心态同样重要。如果感到疲倦、生病或受伤，这些感觉会表现为相同状况的放大版本，当与可能的身体负担结合时，可能很容易减损或破坏体验。与其在压力、疲倦、生病或受伤时进行旅程，不如等待更合适的机会。这将大大降低拥有负面或不充实体验的机会哦。 |

### [旅程保姆](/FreeODwiki/文档/旅程保姆.html)

| ![Users](/FreeODwiki/文件/Users.png) |
| --- |
| * **[主条目：旅程保姆](/FreeODwiki/文档/旅程保姆.html)**<br>当使用[致幻剂](/FreeODwiki/文档/药物分类/致幻剂.html)时，强烈建议有一位[旅程保姆](/FreeODwiki/文档/旅程保姆.html)在场，特别是如果对该物质缺乏经验。旅程保姆的责任是通过保持冷静和脚踏实地的心态来协助个人或团体。这可以通过简单地照看旅程者并在他们经历任何焦虑或压力时冷静地安抚他们来实现，同时防止他们受到任何伤害。“旅程保姆(trip sitter)”和“保姆(baby sitter)”这两个名字之间有明显的联系：这是因为照看旅程者通常感觉像照看婴儿，这是一项必须同样认真对待的责任呢。<br><br>一个好的旅程保姆必须满足许多要求。除了是一个通常负责任的成年人外，他们理想情况下应该是清醒的，并且能够从个人经验或研究知识中理解小组成员的情况。旅程保姆应理解，当一个人在旅程中时，他们可能无法像平时那样交流或互动。此外，他们的平衡和空间判断可能会受损，因此在执行诸如保持水分或在区域内导航等任务时提供帮助可以大大减少焦虑和困惑。旅程保姆可以参与对话，但也应记得给旅程者空间去探索体验，不要有太多的外部影响。<br><br>一旦用户熟悉了一种物质，是否觉得足够舒适在没有保姆的情况下进行旅程就成了个人选择。在服用高剂量或以前从未服用过的剂量时，也建议使用旅程保姆。应该记住，在旅程中有朋友在身边是避免潜在心理、医疗或法律后果的最佳方式。 |

### 锚点

| ![Anchor](/FreeODwiki/文件/Anchor.png) |
| --- |
| * [![Sign](/FreeODwiki/文件/Photo_2016-03-31_23-01-11.jpg)](/FreeODwiki/文件/Photo_2016-03-31_23-01-11.jpg)<br>*“你在用 2C-P 旅程中”* - 这个标志是一个锚点的例子，可以用来在[自我死亡](/FreeODwiki/药效/记忆抑制.html)期间保持脚踏实地。 |

在[致幻剂](/FreeODwiki/文档/药物分类/致幻剂.html)使用的背景下，锚点是一种活动或物理对象，在人的[时间](/FreeODwiki/药效/时间扭曲.html)、空间、[记忆](/FreeODwiki/药效/记忆抑制.html)和自我感严重扭曲期间，保持用户的脚踏实地。在较高剂量下，这可能导致极度迷失方向和困惑。锚点通常用于抵消这一点，并保持一个人对当前现实情况的概念。

锚点的例子包括：
 + 熟悉和令人振奋的音乐。这方面的例子包括我们社区的[好气氛](/FreeODwiki/文档/情景与心境.html)门户。然而，鼓励用户创建自己的播放列表，由个人认为快乐和放松的音乐组成。
  + 一个极其个人化和根深蒂固的图像或物体。
 + 作为一个咒语不断重复一个有意义的词或座右铭。
  + 在一张大纸上写下一个容易阅读的提醒，并在整个体验过程中将其放在视野范围内。常见的提醒包括物质的名称及其剂量，以及诸如“你在用 LSD 旅程中”之类的短语。同样的原则也可用于在手或其他可见身体部位写下提醒。
   + 一件只在旅程期间穿着并因此与旅程行为相关的衣服或配饰。

### 中止旅程

| ![Thumbs-down](/FreeODwiki/文件/Thumbs-down.png) |
| --- |
| * [致幻剂](/FreeODwiki/文档/药物分类/致幻剂.html)有可能变得压倒性，并将用户推入偏执或可怕的情绪中，特别是如果他们缺乏经验或处于不适当的[情景与心境](/FreeODwiki/文档/情景与心境.html)中。<br><br>如果决定终止旅程，[苯二氮卓类物质](/FreeODwiki/文档/药物分类/苯二氮卓类物质.html)和其他[镇静剂](/FreeODwiki/文档/药物分类/镇静剂.html)如一些[抗精神病药](/FreeODwiki/文档/抗精神病药.html)可以被视为正在走下坡路或过长的旅程的“弹出按钮”。这些物质往往是预防[惊恐发作](/FreeODwiki/药效/惊恐发作.html)、[偏执](/FreeODwiki/药效/偏执.html)和可能的创伤体验的非常有效的工具。然而，有经验的用户通常建议如果可能的话，尽量等待旅程中困难的部分过去。挑战性的时刻往往是暂时的，并且可能成为最伟大的学习经历的催化剂呢。 |
