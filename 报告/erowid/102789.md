# 益智药的奇妙魔力——Noopept&菲尼布特

[◀返回](index.md)

原文网址：<https://www.erowid.org/experiences/exp.php?ID=102789>

|剂量： 剂量: |    | 口服 | Noopept  | (每日) |
|   |    | 口服 | 菲尼布特 |   |

|体重:| 体重 185 磅|
|---|---|   

我发现自己很幸运地遇到了Noopept。它比我熟悉的另一种益智药吡拉西坦强得多。吡拉西坦以前对我影响不大，只是让药物退效期感觉舒服一些。至于之前的用药经历，凡是你能说出来的我都试过，而且还不止这些。我的研究重点主要放在……迷幻类的研究化学品上。

开始服用前五天，我刚结束人生中一段相当糟糕的时期，那段时间我对处方阿得拉和阿普唑仑成瘾了。我当时太胡来了，只能强行停药，但戒断反应把我搞得头昏脑胀。这时来了这个装满合法好东西的小包裹。这里的剂量是推测的，但相当准确。它附带一个8-12毫克的勺子。推荐每天剂量是10-30毫克。下面就是接下来一周的经历。

第1天：收到包裹后，我舀了两勺，大概20毫克左右。首先要说的是，我之前喝的那大杯咖啡完全没让我崩溃。相反我饿得厉害，大约一小时后才吃了那天第一顿饭，大概下午四五点。今天我心情特别愉快，不过这可能也归功于其他几件事：我重新开始跑步，度过了苯二氮卓类药物戒断的身体阶段，而且终于摆脱了学校作业和工作，这已经好久没有过了。晚上9点前我又补了大约20毫克。我感觉头脑清晰，但很难入睡和保持睡眠。不过做了非常生动的梦；不能说全是积极的，但通常就是这样。

第2天：早上醒来服了大约20毫克，中午左右又服了20毫克。在课堂上保持清醒轻松极了。喝了一杯咖啡，其实不是特别需要，但很享受它的味道。

那天晚上我服用了菲尼布特：晚上6点半服了2克。两小时后没什么感觉。又吃了2克，两小时后还是没感觉。不过我和朋友们一起玩，完全没有焦虑，这在冷火鸡戒掉阿普唑仑才一周后已经很了不起了。大约凌晨12点半我站起来准备离开。哦，我这时候明显感觉到它了。超级放松。我回家睡了一觉，醒来感觉像中了百万大奖。我很久没这么兴奋过了。

第3天：剂量和效果都一样。另外还补充了卵磷脂（提供胆碱）和鱼油。

我又服用了菲尼布特。先是3克，2.5小时后又2克。耶，兄弟。两小时后我感觉有点迷糊，但非常放松。只睡了5-6小时就醒了，精神十足，感觉棒极了，只是稍微有点迷糊。这种化学品会带来一种有趣的恶心感，但对我来说很容易忽略。我后来还是回去补觉了，因为我通常要睡8小时以上。

第4天：剂量和效果都一样。今晚没服菲尼布特，因为耐受性建立很快，而且有成瘾风险。

第5天：剂量从每天两次20毫克改成一次30毫克。我白天清醒状态更稳定了；记忆力和专注力确实提升了。我不会说它有娱乐性，但这种认知增强绝对让人愉悦。

第6天：醒来后服了30毫克。最奇怪的事情发生了。自从开始服药以来，我每天早上都比闹钟早5到10分钟醒来。虽然还是要花大约一小时才能把我这懒家伙从床上拽起来，但能在需要起床前就醒来还是挺酷的。

第7天：依然醒得早。我现在比一周前状态好多了。那些阴霾、焦虑、抑郁……似乎全都消失了。而且我感觉一天的时间变得好多，这鼓励我去加入社团、扩展社交生活。此外，我的耐力翻了一倍。从以前跑30分钟变成跑一小时。从做1.5小时瑜伽变成3小时。

我不能说这一切都归功于我服用的这些补充剂，也不能说别人会有相似体验。这只是我的主观经历，我发现这种化学品里蕴藏着一种被严重低估的微妙力量。
