# 格式

记录包括扩展语法在内的格式用法，以便提高文档可读性以及吸引力。

优先考虑在 Material for MkDocs 呈现，同时尽可能保证兼容 [Github 预览](https://docs.github.com/zh/get-started/writing-on-github/getting-started-with-writing-and-formatting-on-github/basic-writing-and-formatting-syntax)。

## 基本文本样式

|        | 源码                | 预览              |
| ------ | ------------------- | ----------------- |
| 粗体   | `**粗体**`          | **粗体**          |
| 斜体   | `_斜体_`            | _斜体_            |
| 粗斜体 | `***粗斜体***`      | **_粗斜体_**      |
| 删除线 | `~~删除线~~`        | ~~删除线~~        |
| 高亮   | `<mark>高亮</mark>` | <mark>高亮</mark> |
| 下划线 | `<ins>下划线</ins>` | <ins>下划线</ins> |
| 上标   | `上<sup>标</sup>`   | 上<sup>标</sup>   |
| 下标   | `下<sub>标</sub>`   | 下<sub>标</sub>   |

## 引用、提示、警告等

### 原生 Markdown 引用格式

> 北越部队包围了城市，MACV 不得已只好销毁证据，那个俄国叛逃者必死无疑。

### 告示

详见 [Admonitions - Material for MkDocs](https://squidfunk.github.io/mkdocs-material/reference/admonitions)，Github 预览不支持。

!!! quote 引用

    北越部队包围了城市，MACV 不得已只好销毁证据，那个俄国叛逃者必死无疑。

!!! tip 提示

    北越部队包围了城市，MACV 不得已只好销毁证据，那个俄国叛逃者必死无疑。
