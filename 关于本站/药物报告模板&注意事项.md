# 药物报告模板&注意事项

[◀返回](../index.md)

大家在投稿药物报告之前，一定要仔细阅读哦~

## 注意事项

同一般文档的规范一样，药物报告（以及报告中可能会涉及的图片）的文件名必须没有空格。

同一般文档的规范一样，药物报告也请注意空格！

## 模板 

`报告/<你报告的标题>.md`
```markdown


# <你报告的标题>

[◀返回](index.md)

<*建议*：报告人>  
<*建议*：你报告的药物&剂量>  

<报告内容>......


```
以下文本为你的报告在报告区添加索引，强烈建议

`报告/index.md`
```
<前面的内容>......不要留空，直接跟着
- [你报告的标题](./你报告的标题.md)
```
